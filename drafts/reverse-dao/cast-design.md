# Cast Design ——《废徒倒丹》

> 上游契约（全部 locked）：[[premise-card]] / [[controlling-idea]] / [[genre-contract]] / [[spine]] / 五个角色文件（沈砺 / 玄琢真人 / 玄漪 / 沈砚（已故）/ 司徒明璋）
> 模式：**audit**（既有 cast 五人全锁，校验对抗力量场是否平衡，无新增角色提案）
> 工作原则：cast as antagonism field ——五个人物各自对沈砺施加的压力必须**形态互异、不可替代**，且四个维度全覆盖；否则一人冗、整本书塌（[[principle-of-antagonism]]，Ch. 14）。
> 下游交接：scene-architect（cast 锁定后进入具体场景设计）

---

## 0. 一段定调

本作品是 ≤10 万字中短篇 archplot 单弧，cast 已饱和。需校验的不是"是否补人"，是**这五个人物对主角施加的压力是否各自不可替代**——任何同质压力的存在都会让 cast 显得拥挤、让承认轴四级递降的承载分布失焦、让 [[principle-of-antagonism|对抗原则]]（Ch. 14）的"对抗力量总和压住主角"的硬要求出现冗余。

本次审计的核心结论先行：**五个人物各自的施压形态互不重叠，cast 锁定通过；但有三处咬合点需要在 scene-architect 阶段做精度细化，不需要新增或合并任何人物**。

## 3. Redundancy Diagnosis（同质压力检验）

按用户指定的三项关键审计 + cast-balancer 自身的全行对比，共审计四组潜在的同质压力风险：

### 3.1 师傅 vs 师弟（已故）：对主角"反丹道天赋"的施压

**用户问**：他们对主角反丹道天赋施加的压力是否同质？

**审计**：**不同质——形态精确互补，密度差异巨大**。

- **师傅**：对反丹道天赋的施压形态是**直接对抗 + 镜像同体**（[[characters/master]] §2 真实性格 / §3 维度 2 / §8.1 关系函数）——他出手压主角的反丹道是为了不让"反丹道必反噬"的避险费白付；他越压主角，主角越确认自己对了；他越压，他自己一辈子押的赌注越白付。**师傅对天赋的施压是『活体对抗+ 镜像同体』**：他不是在阻挡天赋，他是在被天赋逼出自己一辈子的反讽。

- **师弟（沈砚）**：对反丹道天赋的施压形态是**主体性解构 + 起源前推**（[[characters/junior-ghost]] §2 真实性格 / §3 维度 3 / §8.1 关系函数）——他不在场，但他通过字迹同形 / 仪式同形 / 银砂死法让主角逐渐意识到"反丹道不是我发现的，是它通过我活着的形状再次出现"。**师弟对天赋的施压是『不在场地解构主体唯一性』**：他不是在对抗天赋，他是在把天赋的"是我自己的"这件事撕掉。

- **形态的精确互补**：师傅施压的是天赋的**社会合法性**（"反丹道不该立"），沈砚施压的是天赋的**主体唯一性**（"反丹道不是你独创的"）。**两种压力的物理学完全不同**——前者是"对面有一个人在对抗你"，后者是"你身上的某些东西不属于你"（[[characters/junior-ghost]] §8.6）。
- **判定**：**不冗余**。两人对天赋施加的是 D5 维度（"反丹道最强执行者 vs 不可能炼正方丹"）的两个不同子轴：师傅施压的是"反丹道作为道是否合法"，沈砚施压的是"反丹道是否属于你自己"。互不替代。

### 3.2 师傅 vs 司徒明璋：对主角"被承认"潜意识欲望的施压

**用户问**：他们对主角"被承认"潜意识欲望施加的压力是否同质？

**审计**：**不同质——但需要在 scene-architect 阶段严守 [[controlling-idea]] §7 违规列表 #9 的「替代承认者禁令」**。

- **师傅**：对"被承认"潜意识欲望的施压形态是**唯一可能给那一字的人**（[[characters/master]] §8.1 关系函数 / §10 意象矩阵的"那一字"执行者）——他是 D4 维度（"想反到底 vs 想被师傅再认一次"）的承认对象本人。**他的不点头 / 他的雷音盖过的一字 / 他的化银砂——是承认轴四级递降的物质源头**。

- **司徒明璋**：对"被承认"潜意识欲望的施压形态是**反向投影 + 镜像投射**（[[characters/situ-mingzhang]] §0 速描 + §4.4 同侧分叉 + 维度 3）——他作为"如果当年没反、没被废、安稳走正方路线的镜像沈砺"让主角第一次看见**如果我没反，我会变成什么**。**司徒明璋对承认欲望的施压不是「替代承认者」，是「反向投影承认欲望的另一种走法」**——他让主角看见"借来信仰"的人是怎么活的；他承担的是主角 D4 潜意识欲望的**反向镜像**（"如果我安稳走正方路线，我会得到那个点头吗？"——读者读出的答案是"会，但代价是变成借来信仰的人"）。

- **形态的精确互补**：师傅是 D4 的**承认对象本人**（他不点头 = 欲望永远饥饿）；司徒明璋是 D4 的**反向投影**（他承担"另一条路上的我"——揭示就算得到那个点头也只是借来的位置）。**两种施压在 D4 同一维度上分占不同功能槽**：师傅是"我等的那个人本身"，司徒明璋是"我等的那个东西在另一种命运里的廉价仿制品"。

- **关键保险（[[controlling-idea]] §7 违规列表 #9 "替代承认者禁令"）**：司徒明璋**绝对不能**在任何场景中对主角说"师弟你对了"或任何形态的"师兄认了你"——这一禁令在 [[characters/situ-mingzhang]] §2 真实性格 / 维度 3 引爆机制中已严守。**Seq 3.1 同门反目场司徒明璋的"师傅那夜下令时手是抖的"是他能给的全部——他承认师傅那夜的脆弱，但他不承认主角对**。**Seq 3.3 第九转他低头退半步是他维度 1+维度 2 的物理引爆，不是「替代承认」**——他放弃持戒尺姿态、戒尺所有权交接给主角，但他不开口说话、不解释、不公开承认。这正是同侧对照（same-axis foil）与替代承认者的精确区别。

- **判定**：**不冗余**。两人在 D4 维度上是承认对象本人 vs 反向投影的关系，不是两个承认者。**但 scene-architect 必须严守司徒明璋的「永不说出口」硬约束**——任何形态的明确认证（哪怕是隐性的"师弟我懂"式眼神交流）都让司徒明璋滑入替代承认者，[[controlling-idea]] §1 锁定句即时塌方。

### 3.3 师妹 vs 师弟（已故）：对主角"被无条件爱过"的施压

**用户问**：她们对主角"被无条件爱过"是否同质？

**审计**：**不同质——形态精确互补，向量方向相反**。

- **师妹（玄漪）**：对"被无条件爱过"的施压形态是**唯一证据 + 主动悬空**（[[characters/sister]] §0 速描 + §2 真实性格 + 维度 3 / 维度 4）——她是主角十二年来唯一在"乾元首徒"以外被一个人看见的瞬间（12 岁那年偷偷塞窝头）；她活着这件事是主角"我害死了她"自我叙事的物质反例；她意识反损叫不出"师兄"让"她活着+她爱过我"这件事在 Resolution 那一帧又变成不完全的——**她是「无条件被爱过」这件事的活体证据，同时是这条证据被永久结构性悬空的承担者**。

- **师弟（沈砚，已故）**：对"被无条件爱过"的施压形态是**反向解构 + 形态预占**（[[characters/junior-ghost]] §8.1 关系函数 + §0 速描"主体唯一性的解构者"）——他不在场，但他通过字迹同形 / 仪式同形 / 灰青旧道袍 / 起炉手势让主角第一次面对**"我活着的形状本身就不只是我"**这一冷酷事实。**他施加的不是「爱」的反例，是「被爱」的主体性的反例**：主角无父无母无家世，他原以为自己的反向天赋、灰青旧道袍、起炉手势都是他自己的；沈砚的存在让他第一次明白这些"被爱的物质载体"在他自己身上之前已经在另一个人身上存在过。**他施加的是「连『被爱』的载体都不只是属于你」的孤独——比『被爱』本身的缺位更深一层**。

- **形态的精确互补**：师妹施加的是"被爱过这件事的**证据**"（曾经被爱过），沈砚施加的是"被爱过这件事的**载体的主体性**"（你被爱过的形态不只是你自己的）。**两种施压的向量方向完全相反**：师妹是"她在你身上留下了爱的痕迹"，沈砚是"你身上的爱的痕迹不全是为你留下的"。**两人在「被无条件爱过」这一价值轴上承担的是『证据』和『载体主体性』两个完全不同的子轴**。

- **判定**：**不冗余**。两人施压的向量相反、维度子轴不同、物理形态完全不重叠。

### 3.4 主角自己的承担 vs 师傅的施压（次要审计）

cast-balancer 自检题：主角自己作为承认轴四级递降的承载体（[[characters/protagonist]] §9）会否与师傅作为承认轴四级承担的执行者（[[characters/master]] §9）产生冗余？

**审计**：**不冗余——承载者与执行者是同一个递降的两端，不是同一种施压**。

- **主角是承载者**：四级递降发生在他身上（他从基线被推到否定之否定）。
- **师傅是执行者**：四级递降的物质源头由师傅制造（基线"嗯"的给予者→戴斗笠不在场→说"你这是在杀我"→说雷音盖过的一字+化银砂）。
- **两者是**承受 vs 施加的关系**，不是冗余**。

## 4. 司徒明璋的加入必要性审计

**问题**：司徒明璋被 act-designer 浮出（不在 premise-card §6 原班四人之列）。他对主角施加的压力（"没走那条路的镜像沈砺"）是不可替代的吗？还是和师傅的压力部分重叠？

**审计**：**必要——且其压力形态不可被其他四位 cast 成员承担**。

### 4.1 他承担了哪些原班四人无法承担的压力

#### 4.1.1 戒尺所有权的物理交接（[[key-image]] §1 + [[characters/situ-mingzhang]] §5 弹药 #5）

[[key-image]] 锁定了三个 image system 核心意象之一——**戒尺（断半）**。23 年前废徒大典上**亲手敲断主角丹田的是司徒明璋，不是师傅**——师傅下令、他动手；戒尺断成两截后另一截在主角手里养着三年。Seq 3.3 大典反炼第九转他低头退半步**是戒尺所有权从他向主角的物理交接**。

**这一压力没有人能替代**：
- **师傅**不能替代——师傅是下令者，不是亲手执行者；师傅持的是"完整戒尺"（另一柄宗门正典戒尺，未被敲断的那一柄），不是被敲断的那一柄。如果让师傅承担戒尺所有权交接，戒尺意象的"敲断"动作就失去了承担者；同时师傅作为反面镜像（[[characters/master]] §0）的位置被弱化——他不再是承认对象本人，变成"承认对象本人 + 执法动手者"，角色函数过载。
- **师妹**不能替代——她在戒尺意象中是"目击者 / 不在场者"（[[characters/sister]] §10），不是执法者。让她承担戒尺所有权交接会撕掉她作为 foil-of-recognition 的同侧反向位置。
- **沈砚**不能替代——他 50 年前已死，没有 23 年前敲断戒尺这件事的物理基础；如果让他通过遗物/回忆"替代"戒尺执法者，他作为 ghost-engine 的"不在场地施压"姿势被打破。

**结论**：戒尺意象在 [[controlling-idea]] §3 锁定为承认轴矛盾→对立级的物理标记物——它的"敲断者"和"放弃持典姿态者"必须是同一个独立人物。**司徒明璋承担的是这一物理位置——他是戒尺意象的物理执法者，戒尺意象塌则承认轴矩阵裂**。

#### 4.1.2 "没走那条路的镜像沈砺"的反向投影（[[characters/situ-mingzhang]] §4.4 + 维度 3）

主角的 D4 维度（"想反到底 vs 想被师傅再认一次"）的潜意识反例需要一个**活体对照**：如果主角当年没反、没被废、安稳走正方路线，他会变成什么？**没有司徒明璋，这个反例只能存在于读者的脑补里——故事中无人承担**。

- 让师傅承担这个反例（让师傅"如果当年我没押反丹必反噬，我会变成怎样"）会撕掉师傅 §3 维度 2（赌一辈子+三百条人命的避险者）——师傅一辈子押了赌注是他角色的核，让他承担"未押的反例"等于让他失去角色函数。
- 让师妹承担会撕掉她 §4 自觉欲望（不让父亲与师兄相撞）——她追的是延后那一天，不是承担"未反的另一种路"。
- 让沈砚承担——沈砚是"反到底"的反例（他做了），不是"安稳走正方路线"的反例（他没安稳）。

**只有司徒明璋承担这一反例**：他是"如果当年沈砺没被废、没反、安稳走正方路线"的活体镜像。**没有他，主角 D4 维度的反向投影无承担者；主角的"我等的那个点头到底值不值得"这一问就只是脑补，不是故事中的物理对位**。

#### 4.1.3 借来信仰 vs 押了一辈子的对照（[[characters/situ-mingzhang]] §0 速描 + §4.4 四人 gap 对位）

师傅押了一辈子（300 条人命的避险费 + 22 岁挖三个月尸首的雪夜 + 右膝冻坏的旧伤）；司徒明璋借了一辈子（13 年师傅给他突破元婴 + 5 年宗主令任期 + 完整戒尺的备份地位）。**两人都站在"正方"信仰里，但一个是真心相信的押注者，一个是借来信仰的红利接收者**——这一对照让师傅的"反丹必反噬"不被读者读成"老顽固固执"，而是被读成"一辈子赌注的真实成本"。**没有司徒明璋作为「借来信仰」的对照**，师傅作为信仰押注者的真实成本会被读者错位读为"师傅就是个独裁老头"——师傅维度 2"看见徒弟反向天赋时第一反应是恐惧而非愤怒"的物理引爆会失去对照面，他的"押了一辈子"的份量被读者麻木化。

**这一压力位置只有司徒明璋能承担**——他是师傅赌注的红利接收者；他踩了 20 年祠堂第七块青砖不知道砖下是地宫入口（[[characters/situ-mingzhang]] §1.1 微动作 3）这一物理形态是"借来信仰"的最精确具象化。

#### 4.1.4 类型契约的同门反目戏承担（[[genre-contract]] §2 缺口 C 补场 + §6 反陈词滥调 #14 锁定）

[[genre-contract]] §2 缺口 C 锁定了"同门反目独立场"作为师徒情/复仇剧次类型的硬性必备场（[[genre-contract]] §2 表中"同门反目（缺口 C）—— 必须补"）。反陈词滥调 #14 锁定"现任宗主司徒明璋必须有让读者隐隐心疼的一面"。**没有司徒明璋，缺口 C 无承担者，类型契约违约**。

让师傅承担同门反目戏会让师傅的稀缺品定价崩（[[genre-contract]] §8 硬约束 5：师傅前 9 万字戏份合计 ≤ 1.2 万字）；让师妹/沈砚承担在物理上不可能（一个被认为已死、一个 50 年前已死）。

**结论**：类型契约硬要求一个独立的同门角色承担"反目戏"——只有司徒明璋符合所有约束。

### 4.2 他是否与师傅的压力部分重叠（用户具体担忧）

**没有重叠**——见 §3.2 师傅 vs 司徒明璋审计：

- 师傅是 D4 的**承认对象本人**（他不点头 = 欲望永远饥饿）；
- 司徒明璋是 D4 的**反向投影**（他承担"另一条路上的我"——揭示就算得到那个点头也只是借来的位置）。

两人在 D4 同一维度上分占两个完全不同的功能槽——承认对象本人 vs 反向投影。**这是同一维度的两个互补子轴，不是同一种施压的两个执行者**。

### 4.3 司徒明璋是否需要被合并 / 缩减 / 删除

**不需要**。他的施压不可替代（§4.1）、与师傅不冗余（§4.2）、且承担类型契约硬必备场（缺口 C）。

**但 scene-architect 需要严守以下三条**：
1. **永不说出口的"师弟你对了"**（[[controlling-idea]] §7 违规列表 #9 严守）。
2. **他在 Climax 没有动作**（[[characters/situ-mingzhang]] §4.4 锁定的"四人 gap 同帧合上时他唯一没做选择"）——他作为同侧对照的最深一种形态是"借了一辈子的人在最大选择面前没有任何东西可付出"。
3. **戒尺所有权交接的非台词化**（Seq 3.3 第九转他的低头退半步必须是纯物理动作，不允许任何内心独白或对白补充）。

## 5. 师弟（沈砚）作为 Ghost-Engine 的兑现度审计

**问题**：他不直接出场，但通过遗物+闪回控场。他对主角施加的压力如何被读出？这一压力是否在 spine 上被实现？

**审计**：**兑现度完整——五种载体在 spine 上各自有具体的引爆场景，符合 [[exposition-as-ammunition|铺陈即弹药]]（Ch. 15）硬要求**。

### 5.1 五种载体在 spine 上的具体兑现位置

按 [[characters/junior-ghost]] §0 速描 / §5 弹药 / §7 压力点 / §8.5 关系函数 + [[spine]] §4 复杂化节点 + [[act-design]] sequence：

| 2 | **沈砚最后一炉的旧炉底金石片**（沉鸠贴身保存） | Seq 2.1.c / 2.2 后期 | 主角解读金石片刻字"以人为核" | 反丹道第五变量的物质前体——主角终极反转 B 反核机制揭露的暗预告 |

| 5 | **师傅化银砂死法对位 50 年前师弟屠村**（Climax 拍 5）| Climax 拍 5 | 主角在场目睹师傅化银砂 | 两代人承认轴在同一帧上同时落于 negation of the negation——主角余生不知那一字是什么；沈砚 50 年前死时也不知道师兄会不会有一天来看他那一炉 |

### 5.2 五种载体的弹药学（[[exposition-as-ammunition]] 兑现）

按 [[genre-contract]] §6 反陈词滥调 #10（关键剧情靠回忆/闪回硬塞被禁止 + [[controlling-idea]] §7 违规列表 #11 内心独白禁令）的硬约束：

- **载体 1（手札残页）**：物理对象，主角在 Seq 2.1 物理夺得——**通过动作发射**，不依赖回忆。
- **载体 2（金石片）**：物理对象，主角在 Seq 2.2 后期物理解读——**通过动作发射**。
- **载体 3（师傅回忆闪回）**：师傅的微动作（听"反"字时右手食指一抖 / 决定难做的事之前左手按右膝 / 进出祠堂正殿绕开第七块青砖）——**通过物理动作发射**，不通过任何独立闪回章节。
- **载体 4（井壁残页字迹）**：物理对象，主角在 Seq 3.4 物理看见——**通过画面发射**，主角不需要任何内心独白。
- **载体 5（银砂死法对位）**：Climax 拍 5 物理同形——**通过死法对位发射**，主角不能"回忆"师叔屠村画面（[[characters/junior-ghost]] §12 #4 严守的"死法对位由读者完成认知"）。

**五种载体全部通过物理动作 / 物质对象发射**——零独立闪回章节，零内心独白依赖。**[[exposition-as-ammunition]] 与反陈词滥调 #10 双重硬约束兑现**。

### 5.3 沈砚作为 Ghost-Engine 的关键测试问题

**测试**：如果把沈砚从 cast 中删除，会破坏哪些链条？

按 [[characters/junior-ghost]] §8.5 关系函数去除塌方测试 + 本审计的交叉检验：

1. **反丹道的物理来源消失**——主角变成"独孤异端天才"的网文俗套（[[setting-survey]] §3.5 硬规则：主角不是金手指、是规则的执行者——这条硬规则失去先例证据）。
2. **师傅一辈子押"反丹必反噬"的避险费本金消失**——师傅维度 2 失去物理根。
3. **主角主体唯一性的解构无源**——D5 的"反丹道最强执行者"身份失去反向解构的物质载体。
4. **Climax 拍 5 师傅化银砂的死法对位失去对位对象**——反讽极性塌一层。
5. **玄漪体内胎毒的可能源头消失**——师傅"怀疑亡妻之死与师弟有关"的双重根失一层。
6. **承认轴的正面级反向源头消失**——师傅按主角肩说"嗯"那一刻"心里在叫『砚』"这一暗能源失去对象，承认轴四级递降的基线失去深度。

**结论**：沈砚作为 ghost-engine 不可被删除——他不在场，但他控场到所有关键人物。**兑现度通过**。

### 5.4 沈砚兑现度的一处需要 scene-architect 严守的精度

[[characters/junior-ghost]] §12 #4-#5 锁定的关键硬约束：

- Climax 拍 5 主角**不能**通过回忆师叔屠村画面完成死法对位认知；
- Climax 七拍内主角**不能**有任何完整的内心活动（[[controlling-idea]] §7 违规 #11）。

**所有的回忆 / 连结 / 认知都由读者在 Resolution 之后自己完成**。这是 ghost-engine 兑现度的最严苛硬约束——任何让主角"明白沈砚的意义"的内心独白都把 ghost-engine 从"不在场地施压"降级为"被主角心理化的角色"，反讽极性塌方。

**判定**：scene-architect 在 Climax 七拍设计阶段必须严守这一硬约束——本次审计标记为**高风险点**，需要 scene-architect 在场景设计阶段反复回看 [[characters/junior-ghost]] §12 #4-#5。

按 cast-balancer 系统提示 §5 第 2 条："Every protagonist dimension is pressured by at least one character。"

### 6.1 主角五个维度的施压承担者

| 维度 | X vs Y | 主要施压者 | 次要施压者 | 覆盖度 |

### 6.2 维度 D2 的覆盖警示

**唯一施压者是师妹**——其他四位 cast 成员对 D2（慈悲 vs 收割）维度均不施加独特压力。这是否构成 coverage 缺口？

**审计**：**不构成缺口，但需要 scene-architect 注意一处精度**。

D2 的引爆场景在 [[characters/protagonist]] §3 锁定为 "场 4（反丹副作用首现）：他在循环二中段亲眼看见自己救过的人七窍流银砂——他没有立刻停手，他在那个尸体边上又救了下一个人"。**这一场的施压源是反丹道副作用本身**——是物理后果，不是某个具体角色的施压。师妹的施压是把 D2 中"慈悲到自虐"的反例（"她活着 = 我三年的赎罪是建立在虚假的悔上"）钉死。

**判定**：D2 的覆盖结构是"角色（师妹）+ 物理后果（反丹副作用）+ 主角自身的选择"三方合担——师妹是唯一的角色施压者，但 D2 整体的对抗力量足够（反丹副作用作为物理对抗力量是 [[forces-of-antagonism]] 中的超个人层力量之一，已在 [[premise-card]] §5 / [[genre-contract]] §5 锁定）。**不需要补人**。

**但 scene-architect 需要注意**：Seq 2.5（反丹副作用首现，[[act-design]] Act 2 Sequence 2.5）必须把"他没立刻停手 + 师妹三年前对他说『我快死了求你救我』的回声"作为对位场景——D2 的物理后果与师妹的施压必须在同一场咬合，否则 D2 的"敢拿别人命赌一炉"的份量被读者读为"主角冷漠"而非"主角对反丹道立宗的执念"。

### 6.3 主角自身（内在层）的承担

按 [[characters/protagonist]] §2 真实性格 + §3 维度表，主角自身的内在层承担"我害死了她"自我叙事 + "师傅毕竟是师傅"旧自我 + 对"被承认"的结构性需要——**这是 [[forces-of-antagonism]] 三层结构中"内在"层的核心承担**（[[premise-card]] §5 内在层锁定），与四位 cast 成员的内在层施压形成五方共担。

**主角自身的内在层施压不需要被外部角色"承担"**——主角对自身的对抗是 spine 双层撕裂的根（[[spine]] §0 / §2 / §6），是 cast-balancer 不审计的部分（不是 cast 设计问题，是主角设计问题）。

## 7. 类型契约 foil 覆盖检验

按 [[genre-contract]] §1-2 锁定的主类型/次类型混合，校验 cast 是否承担所有类型必备 foil：

### 7.1 玄幻爽文 · 升级流的必备 foil（同辈天才）

**类型预期**：升级流契约要求至少一个**同辈天才**作为主角能力跃迁的对位（[[genre-contract]] §1.主类型五项之一"对抗梯度递进"）。

**审计**：**司徒明璋作为同辈但更年长的"师兄"承担了部分对位**——他是元婴初期、二品丹师下乘，主角是元婴预备 / 二品丹师天花板（被废前是三品金丹）。但**司徒明璋不是「天才」——他是借来信仰的人**。

**这是缺位还是反陈词滥调？**

按 [[genre-contract]] §6 反陈词滥调 #5（反派宗门长老脸谱化）+ #14（让读者隐隐心疼的一面），本作品**主动反**了"同辈天才"的网文俗套——主角不需要一个天赋同等的同辈对手来证明自己；他的对位是师傅（不对称权力关系的反面镜像）+ 沈砚（不在场的主体性解构者）。**升级流契约的"对抗梯度递进"由师傅 + 联盟 + 天道债 + 反丹副作用四方承担**——不依赖"同辈天才"这一陈词滥调。

**判定**：**缺位但被替代——属于「明确替代」非「未知缺口」**。
- **替代物**：师傅作为反面镜像（不对称权力关系）+ 沈砚作为不在场的主体性解构者（两代人对位）+ 天道债 + 反丹副作用。
- **替代的代价**：升级流读者预期的"同辈天才对决"被取消——读者得不到"主角战胜同辈天才"的爽感。但本作品的类型混合契约里这一爽感被替换为"主角战胜师傅一辈子的赌注"（金场 1 大典反炼）+ "主角面对自己活在另一个人形状里"（Seq 3.4 入井）的两种更深爽感。**代价已被 [[genre-contract]] §1-3 显性签署**。

### 7.2 师徒情 / 复仇剧的必备 foil（替代师傅角色 / 代理父亲）

**类型预期**：师徒情/复仇剧契约要求至少一个**替代师傅角色 / 代理父亲**作为主角情感支撑——一个可以"反向投影师傅"的角色，让主角的师徒情有第二个承载体。

**审计**：**师傅 + 司徒明璋两人共担**——
- **师傅**：父亲式承认者本人（D4 的承认对象），承担师徒情/复仇剧的"亲师 / 仇师"双向情感的全部主轴。
- **司徒明璋**：作为亲手敲断者 + 反向投影的同侧对照，承担"如果当年没反，安稳走正方路线的代理父亲式师兄会是什么样"的反向投影——他不是"代理父亲"（他没有承担父亲角色的施压），他是"师徒情中关于『另一种师徒关系』的活体对照"。

**这两位是否足够覆盖类型预期？**

按 [[genre-contract]] §2 表格 + §6 反陈词滥调 #14，本作品**主动反**了"代理父亲"这一师徒情陈词滥调——主角不需要一个"温暖的代理师傅"来给他情感支撑。他的情感支撑来源是**师妹 + 沈砚的不在场关系**（师妹是童年唯一被爱者；沈砚是无意中长成的另一个形状）。**师徒情/复仇剧的"代理父亲" foil 被反向消解**——主角余生没有任何"代理父亲"来填补师傅这一位置的空缺；他活在一个**没有父亲的世界**里。这是反讽极性（[[controlling-idea]] §1）在 cast 覆盖上的精确兑现。

**判定**：**覆盖——通过反向消解兑现类型契约**。师傅 + 司徒明璋两人组成的"双层师徒"结构（生父 + 兄长式师兄）已足够承担师徒情/复仇剧的全部情感重量。**不需要再加"代理父亲"型角色**。

### 7.3 命运反转剧的必备 foil（命运承担者）

**类型预期**：命运反转剧要求一个**命运承担者**——通常是被命运碾压但仍然选择行动的人物，是主角面对终极反转时的情感对位。

**审计**：**师妹 + 沈砚（已故）两人共担**——
- **师妹**：被父亲一辈子立的"正"包成"反"的形态续命三年，是反丹道在父亲身上的活体证据——她是**命运被反向操弄但主动选择如何被操弄的人**（[[characters/sister]] §2 真实性格 / 维度 4）。
- **沈砚（已故）**：被父亲死时无丹可救逼上反丹道、立反丹道立成之炉时炸死、死后被宗门除名丹经焚毁——他是**命运用「替代品」从未能给的代价压死的人**。

**判定**：**双重承担，密度足够**。师妹 + 沈砚两人组成的"命运承担者"双层（活体 + 死后回响）兑现命运反转剧的全部情感重量。**师妹自愿入炉那一刻的「师兄这次你别照方子来」+ 沈砚 50 年前临死前的「我怕的是有一天我做对了，但已经太迟」——两句话同帧合上承担命运反转剧的最高情感密度**（金场 2 Climax 拍 4-5 + Seq 4.2 师妹现身）。

### 7.4 类型契约 foil 覆盖小结

| 类型 | 必备 foil | 本 cast 承担方式 | 判定 |
|---|---|---|---|
| 玄幻爽文升级流 | 同辈天才 | 师傅（不对称权力）+ 沈砚（主体性解构）+ 天道债+反丹副作用——**替代**| 明确替代，代价已在 [[genre-contract]] 显性签署 |
| 师徒情 / 复仇剧 | 替代师傅 / 代理父亲 | 师傅 + 司徒明璋（双层师徒）——**反向消解**| 通过反向消解兑现反讽极性 |
| 命运反转剧 | 命运承担者 | 师妹 + 沈砚（活体+死后回响双层）——**双重承担**| 密度足够 |

**全类型 foil 覆盖通过——无缺位 / 无未知替代**。

## 8. Merge / Cut / Promote 建议

### 8.1 总建议

**无修改建议**。Cast 五人各自承担不可替代的施压形态，全维度覆盖、承认轴四级递降全程承担、三层冲突全程承担、类型契约 foil 全程承担（含明确替代与反向消解）。

### 8.2 不需要新增人物的硬性确认

本作品是 ≤10 万字中短篇 archplot 单弧，cast 已饱和。按 [[spine]] §1 "archplot 在中短篇里的代价"：

> archplot 一般匹配长篇（200K+）。压在 105K 内的代价是——所有支线、群像、世界观奇观必须为主线让路。

新增任何独立 cast 成员（包括"同辈天才"或"代理父亲"）会触发以下违约：

1. **[[spine]] §1 单一主人公因果链** 被破坏——次要角色喧宾夺主；
2. **[[genre-contract]] §8 硬约束 5**（师傅前 9 万字戏份合计 ≤ 1.2 万字）——新增人物会挤占师傅的稀缺戏份；
3. **[[characters/sister]] §13**（师妹主要戏份 ≥ 1.5 万字 + [[characters/situ-mingzhang]] §13 司徒明璋戏份 5K-6K）——新增人物会让现有四位非主角 cast 的戏份分布失衡。

**结论**：cast 不允许新增。

### 8.3 三个需要 scene-architect 严守的咬合精度

虽然 cast 锁定无修改建议，但本审计标记三个高风险咬合点供 scene-architect 在场景设计阶段反复回看：

#### 8.3.1 司徒明璋的"永不说出口"硬约束

- **位置**：Seq 3.1（同门反目场）+ Seq 3.3（第九转低头退半步）+ Climax 拍 1（站立不动）+ Resolution（握完整戒尺不举）。
- **硬约束**：**绝对禁止**司徒明璋在任何场景中说出"师弟你对了"或任何形态的"师兄认了你"。
- **风险来源**：[[controlling-idea]] §7 违规列表 #9 替代承认者禁令——任何形态的明确认证（哪怕是隐性的眼神交流）都让司徒明璋滑入替代承认者，[[controlling-idea]] §1 锁定句即时塌方。
- **scene-architect 措施**：每场涉及司徒明璋的对白必须过审 [[controlling-idea]] §7 违规列表 #9 的硬规则。

#### 8.3.2 沈砚的"零内心独白"硬约束

- **位置**：Seq 2.1 / Seq 2.2 / Seq 2.4 / Seq 3.4 / Climax 拍 5。
- **硬约束**：主角面对沈砚遗物 / 字迹 / 死法对位时，**绝对禁止**任何"主角心里想起师叔"的内心独白；所有的回忆 / 连结 / 认知都由读者在 Resolution 之后自己完成。
- **风险来源**：[[characters/junior-ghost]] §12 #4-#5 锁定 + [[controlling-idea]] §7 违规 #11——任何让主角"明白沈砚的意义"的内心独白都把 ghost-engine 从"不在场地施压"降级为"被主角心理化的角色"。
- **scene-architect 措施**：Climax 七拍设计阶段必须反复回看 [[characters/junior-ghost]] §12 #4-#5；任何沈砚相关场景的主角内心活动必须删除或改为物理动作。

#### 8.3.3 师妹的"姿态对位"咬合（D4 同帧反例的物理化）

- **位置**：Seq 3.4 地宫第七层石棺缝那一帧 + Seq 4.2 师妹现身 + Climax 拍 5 师妹意识反损 + Resolution 张嘴叫不出。
- **硬约束**：师妹与主角的关系必须严守"不是爱情对象，是被爱过的唯一证据"（[[genre-contract]] §6 反陈词滥调 #1）——**绝对禁止**任何眼神交流式的暧昧场、互相称呼变化、触碰戏（除入炉场）。
- **风险来源**：D4 维度（主角自觉欲望"反到底" vs 潜意识欲望"被师傅再认一次"）的同帧反例是 spine 双层撕裂的核——如果师妹滑入爱情对象，D4 维度的反例就被恋爱戏吃掉，spine 的承认轴四级递降在主角的情感账本上失去对位。
- **scene-architect 措施**：师妹相关场景的对白和动作必须过审 [[genre-contract]] §6 反陈词滥调 #1 的硬规则。

## 9. 七点 Cast 审计

按 cast-balancer 系统提示 §5 六点 cast 审计 + 本作品的反讽极性硬约束扩展为七点：

- [x] **5. 类型契约 foil 全部承担（含替代与反向消解）**：§7 类型契约 foil 覆盖检验——三类型 foil 全覆盖，缺位由明确替代或反向消解承担，代价已在 [[genre-contract]] 显性签署。
- [x] **6. 每位角色赢得场景时间**：
  - 主角：主线全程；
  - 师傅：前 9 万字 ≤ 1.2 万字 + Climax 拍 1-5（约 2.5K）= 合计约 1.5 万字；
  - 师妹：Seq 3.4 + Seq 4.2 + Climax 拍 4-5 + Resolution = 合计约 1.5-2 万字；
  - 沈砚：五种载体跨循环二全程 + Climax 拍 5 = 合计约 0.6-0.8 万字（[[characters/junior-ghost]] §13 锁定）；
  - 司徒明璋：Seq 3.1（3K）+ Seq 3.3 第八-九转（0.3K 折叠）+ Climax 拍 1（约 200 字物理动作）+ Resolution 末（约 200 字物理动作）= 合计约 5-6K。
  
  **每位角色都赢得足够场景时间承担其施压函数；无任何角色被声称为主要施压者但戏份不足**。
- [x] **7. 反讽极性硬约束兑现（本作品扩展项）**：四位非主角 cast 成员的每一种施压形态都在反讽极性（[[controlling-idea]] §1 锁定的 negation of the negation）框架下被精确分配——师傅说一字但被雷音盖、师妹叫不出"师兄"、沈砚 50 年前已死、司徒明璋永不说出口；**所有四方都在「说出但不可被验证」或「不说出」的姿势里完成施压**。这是承认轴四级递降在 cast 设计上的最深一种保险——cast 中没有任何角色能让承认轴在最后一帧被验证；[[controlling-idea]] §1 锁定句的物理兑现由 cast 设计本身承担。

**七点全过——cast 锁定**。

## 10. 给作者 / 下游的开放问题（≤5）
