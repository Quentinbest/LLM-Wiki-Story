#### 1. Frame

- **Where/when**：帝京祭天台前空地（大典次日清晨 / Day 1 卯时三刻）；主角刚刚走出宗门祠堂广场往帝京方向走了三里
- **Who's in it**：主角沈砺（POV）；远处人群（市井甲乙丙——疫情爆发后聚集听钟的百姓约百余人，不具名）；天空（作为对抗力量物理化的承载者）
- **Spine position**：终极反转 A（[[spine]] §4 锁定，Day 1）

#### 2. Objectives & antagonism

- **POV 场景目标**（active verb the actor can play）：**走到帝京最近的疫情爆发处——他想用昨日反炼九转还魂的炉气余韵给街市疫民做一炉应急小丹**。他还在 False Ending 的伪希望里——他以为反炼九转还魂的胜利可以延续，他以为公共承认到来后可以让他凭着昨日的炉气把疫情先压住、然后等师傅找他说话。**他想用一炉应急丹证明 False Ending 的伪希望可以兑现**。
- **Forces of antagonism**（[[levels-of-conflict]] 三层）：
  - **超个人**：天空——雷云倒卷、天色由清明转暗赤；这是天道债账本已抵 144 道阈值前夜的物理预警（[[setting-survey]] §6 硬规则 3 锁定）。天空作为对抗力量物理化于本场达到最高峰（评分 5）
  - **个人**：无人在场（人群是背景，不是对抗者；评分 1，**用户已接受**）
  - **内在**：False Ending 伪希望与脚下街市疫情死人物理事实的撕裂——他想压住疫情但脚下三具尸体已经摆在街口（评分 3）
- **其他角色秘密目标**：远处人群秘密目标=听钟（瘟疫钟三短一长已响了两日；他们不知道今日会响第三短一长——天道之炉降临之钟）

#### 3. Value

- **Named value**：**伪希望 / 真相**（false hope / unmasked truth）—— 在承认轴的 Act 4 入口阶段，伪希望（"师徒和解可能"+"反炼胜利可延续"）尚未被翻面；真相（天道之炉降临=反丹副作用引爆十省疫情的因果链回收）即将砸下
- **Opening charge**：**−**（伪希望悬空，但已经在 Act 3 末摇晃）
- **Closing charge**：**−**（**charge 相同但 magnitude 加深**——伪希望被物理逐出，真相显形；承认轴从 −2.5 进 −2.6；**McKee Ch. 11**：scene 的 turn 不一定从 + 到 − 或反向，可以是 magnitude 在同极性内深化——本场是后者）

#### 4. Beats（5 beats）

1. **Action**：主角走出宗门祠堂广场，往帝京方向走三里。**Expected**：街市该有早起的肉摊和卖晨粥的妇人。**Actual**：街市空了，只有瘟疫钟"咚——咚咚"在响（三短一长第二日的姿势）。**Gap**：他预期街市的烟火气 / 实际是死寂——他第一次在物理层面看见反丹副作用扩散到帝京。

2. **Action**：他在街口停下，看见三具尸体并排摆在石板上，七窍流出暗银色——**这是反丹副作用的物理形态**（[[setting-survey]] §3.3 反丹副作用钉死）。**Expected**：他可以炼一炉应急小丹压住。**Actual**：他俯身摸尸体胸口，指尖触到尸体——尸体的反向气**没有**逃逸（按反丹理论尸体应该向他指尖逃逸反向气，但这三具尸体气路已被某种更高一级的力量"截下"）。**Gap**：他预期能用反丹道处理 / 实际反丹道在这三具尸体上失效——他第一次（在身体上）感觉到"有一个比反丹道更高的力在场"。

3. **Action**：他抬头看远处帝京祭天台方向——往日的祭天台只在大典开幕日开放，今日不该开。**Expected**：祭天台关闭。**Actual**：祭天台台顶已有一层暗赤色光晕——这层光晕不是火光，是天道之炉降临前夜的"先导云"。**Gap**：他预期祭天台关闭 / 实际祭天台已被某种力量"激活"——他还不能命名这是什么，但他知道这与昨日反炼九转还魂的炉气余韵无关。

4. **Action**：他往祭天台方向走（脚步开始加快——右脚轻度跛感被他自己压住）。**Expected**：他可以走到祭天台外圈观察一会儿。**Actual**：走到祭天台外圈第二十步——**天色突变**：清晨的天光被一层暗赤色帷幕从上往下抹掉；雷云**倒卷**（不是聚拢，是从天心向外卷开 / 这是天道之炉降临的标志性物理形态，[[setting-survey]] §7 锁定）。市井甲乙丙抬头看天 / 有人开始跪 / 瘟疫钟突然停了。**Gap**：他预期能观察 / 实际天道之炉的降临姿势已经物理铺开——他作为目击者被钉在原地。

5. **Action**（**Turning Point**）：祭天台中央——**"苍生之炉"从云中砸落**。炉身九丈高、铜色泛青；炉口直径三丈；炉壁外侧浮现一行十二个篆字"**正方五气朝元·济苍生疫·限七日**"（[[setting-survey]] §3.4 + §7 锁定）。台基崩裂三尺；远处宗门祠堂方向**承天梁倒数第六震齐响**（"咯"一声木裂——[[scenes/03-act-3-scenes]] §7.A 承天梁微震第五次的延续）。**Expected**：他刚才以为这是某种预警的最大形态。**Actual**：这是天道之炉降临的兑现——不是预警，是判决书的物理送达。**Gap 最大**：False Ending 的"反炼胜利可延续 + 师徒和解可能"在这一秒被天道之炉的物理事实**一秒逐出**——他还没回神，但他的身体已经知道伪希望死了。

#### 5. Turning Point

> 祭天台中央"苍生之炉"从云中砸落、台基崩裂三尺、他身后承天梁倒数第六震齐响"咯"一声木裂——False Ending 的伪希望被天道之炉的物理降临一秒逐出（承认轴 −2.5 → −2.6）。

#### 6. Subtext

主角看着祭天台中央那座九丈高的炉，他的手不自觉地按了一下左袖下断戒尺的位置——这是他三年来每次起炉前的潜意识仪式。**但这一次他按断戒尺的姿势不是"准备敲炉"，是"确认它还在"**——他第一次发现自己面对一座炉的姿势不是主动起炉的丹师，是被一座炉看见的对象。**他还没说出"两难"两个字，但他的身体已经知道这一炉不属于他**。

- [x] Turns（伪希望 → 真相 / 承认轴 magnitude 加深 −2.5 → −2.6）

- [x] Gap is present（每拍 expected ≠ actual，最大 gap 在 Beat 5）
- [x] Minimum, conservative action（主角全程只走、看、按断戒尺——没有超出他在 Act 3 末的物理状态）

#### 2. Objectives & antagonism

- **POV 场景目标**：**看懂这座炉要他做什么**——他从 Beat 5 苍生之炉砸落后呆立到现在，他要在这一刻钟内把眼前的局面分类清楚（这是认知动作，可演——他扫炉壁、读篆字、看天色、按胸口印记位置、按断戒尺位置——一连串具体的"看懂"动作）

#### 4. Beats（5 beats）

1. **Action**：他走近祭天台外圈第三十步，抬头读炉壁十二个篆字"正方五气朝元·济苍生疫·限七日"。**Expected**：他读懂"五气朝元"是神品级正方丹（[[setting-survey]] §3.1 锁定）。**Actual**：他**读懂了，但同时读出第二层**——"济苍生疫"四个字意味着这炉是用来止反丹副作用引发的疫情的（[[setting-survey]] §3.3 + §6 硬规则锁定的反丹副作用与天道债的因果链回收）。**Gap**：他预期看见的是一个挑战 / 实际看见的是一个回收他自己的反丹副作用账目的物理装置。

2. **Action**：他低头看自己胸口（衣襟内侧）——印记位置。**Expected**：印记仍是暗赤一格深化。**Actual**：印记**金黑流动开始**——这是 [[setting-survey]] §4.1 第 4 阶段的物理形态；他三年来第一次看见印记的颜色不再是赤红的某一格，而是**金黑相互流动**（这意味着账本即将爆出，144 道阈值临界）。**Gap**：他预期印记仍按"每炉累积"的旧节奏 / 实际印记已脱离他个人节奏被天道债账本接管。

3. **Action**（**账本爆出**）：他抬头看天——**雷云倒卷的中心位置**，一道金黑色光柱降下，光柱内浮现**一份巨大的账本物理形态**（[[setting-survey]] §6 硬规则 3 锁定 / 144 道阈值兑现）。账本物理形态：每一道劫是一行字，每一行字对应主角三年来反丹道每一炉的具体时间地点丹方代价；账本最后一页"144"赫然在目；账本悬在他头顶约九丈高（与苍生之炉同高）。**Expected**：他以为账本会逐项累计到他眼前。**Actual**：账本**一次性物理化**——他三年来 144 道劫一次性被天道公开（虽然在场百姓看不见，但账本物理存在）。**Gap**：他预期渐进的清算 / 实际是一次性的物理判决——他的内核空洞被天道用具体的 144 行字填满。

4. **Action**：他举起右手要碰账本（账本悬在他够不到的高度），他**没有跳**——他在举手到一半时收手；他改用左手按住胸口印记位置（这是他三年来每次"看懂自己处境"的潜意识动作）。**Expected**：他可以触碰账本验证账本上的字是否每一行都对。**Actual**：账本**自动翻开**到最后一页——最后一行字写着"**正方五气朝元·济苍生疫·限七日**"，与苍生之炉壁上的篆字完全同步（这是天道用账本与炉同时下达指令的物理形态）。**Gap**：他预期账本是过去的清单 / 实际账本同时是未来的判决书——账本闭合了过去与未来的因果环。

5. **Action**（**Turning Point**）：他在祭天台外圈站着，**在心里把两条路物理地排开**：
   - **路 1**：起炉用正方"五气朝元"济苍生疫。但他丹田已自封反向（[[setting-survey]] §2.3 钉死）—— 强行起正方 = 反噬 = 当场炸炉 = **必死**。
   - **路 2**：拒炉。但账本满 144 道阈值 + 苍生之炉降临 = 反丹副作用扩散十省疫情会持续爆发死十万 + 主角作为账本主体本人**永世为天道债奴**（[[setting-survey]] §6 硬规则 3 锁定）。
   - **第三条路**：**没有**。他在心里物理地排查了五种可能（找师傅 / 找司徒明璋 / 找师妹 / 拖延 / 用反丹道反炼苍生之炉自身）—— 五种全部被物理钉死（师傅未发一言转身离场 / 司徒明璋低头退半步不公开承认 / 师妹在地宫第七层石棺缝不可移动 / 七日限期不可拖 / 反炼苍生之炉=反向核需要被反向气养满三年的活物=唯一符合条件的活物是师妹）。**Expected**：他能找到第三条路（这是他三年来反丹道每次都找到第三条路的肌肉记忆）。**Actual**：第三条路**不存在**——他第一次在反丹道立宗的胜利之上面对"我的胜利使我抵达了不能胜利的位置"（[[characters/protagonist-arc]] §4 维度 5 引爆）。**Gap 最大**：他三年来反丹道每次都能找到第三条路 / 这一次他找不到——他第一次**看清两难**。

#### 5. Turning Point

> 他在祭天台外圈站着，在心里物理地排开五种可能后承认第三条路不存在——他看清了两难（起炉自死 / 拒炉成永奴），他的反丹道立宗的胜利使他抵达了不能胜利的位置。承认轴 −2.6 → 走向 Crisis 入口前夜（仍 −2.6 但已无回路）。

#### 6. Subtext

主角在第五拍里的"看清"不是哭、不是颤抖、不是按胸口印记—— 他的物理姿势是**站直**。**他站直的姿势是这个故事三年里第一次他的身体没有按任何"求承认"的潜意识仪式做动作**：他没有扫人群最外圈找玄色斗笠（场景 1 的姿势 / [[controlling-idea]] §8 obligatory #1）、没有按断戒尺敲炉沿（场景 2 的姿势 / obligatory #2）、没有抬头找师傅的方向（Pin 系列的姿势）。**他第一次只站着——身体不为任何人留位置**。**这是 character revelation 的次次终形态的物理前奏——他在 Crisis 之前已经完成了"内核空洞被填满"的姿势**（虽然他自己还没意识到这是 character revelation 的前奏）。

#### 4. Beats（4 beats）

1. **Action**：主角从祭天台外圈走到祭天台中央苍生之炉前（这一动作的物理过程跨 Day 5 至 Day 7 / 本拍只呈现 Day 7 亥时三刻他走到炉前的最后三步）。他**主动布阵**——用反向丹道符文在炉口外圈布反向起气阵（[[setting-survey]] §3.2 反向核机制的火门倒拨形态）；他**主动起气**——反向丹气从他丹田自封反向的位置流出，沿地脉返流上升（这是他三年来反丹道每一炉起气的物理姿势 / 但本场是最后一次）；他**主动按炉沿**——左手按苍生之炉的炉口边缘（炉壁烫但他的手不抖 / [[characters/protagonist-arc]] §6 Pin 10 锁定）。**Expected**：他以为自己布阵、起气、按炉沿的动作可以顺利完成而不被任何人打断（他不知道师傅在身后）。**Actual**：他**顺利完成**所有动作—— 布阵、起气、按炉沿，没有任何外部打断；他甚至能听见自己心跳与炉壁印记金黑流动的节奏完全同步。**Gap**：他预期会有外部打断 / 实际没有任何外部打断——他的内核空洞与他的主动动作完全合一；这是他三年来反丹道每一炉中**第一次**他的心跳与炉壁完全同步（之前每一炉都因师傅或司徒明璋或长老的物理介入而有节奏偏移）。

2. **Action**：他的左手按在炉沿上**不动**；他用右手从左袖下抽出**断戒尺**——**戒尺第六次入画**（[[key-image]] §2.1 Item 9 / 戒尺累积重量 +15.5 → +18 / 这是戒尺力学反讽顶点的下一级峰值）。断戒尺在亥时三刻的暗赤天色下**反射出三种光**：（1）木柄的旧光（戒尺断裂前的师傅形态延续）；（2）断口的金光（断口处累积三年反丹道反炼的丹气余韵）；（3）戒尺通体的金黑色（与炉壁印记金黑流动同色——戒尺与印记本质上是同一物的两种物理形态 / [[key-image]] §2.1 锁定）。**Expected**：他以为举断戒尺的姿势会像 Seq 2.5 / Seq 3.3 那样有"敲炉沿"的预期 readiness。**Actual**：他举断戒尺的姿势**比之前任何一次都重**——戒尺累积重量 +18 让他的右手在举起到三尺高的位置时**手腕微抖**（这是他三年来反丹道每一炉举戒尺第一次手抖 / 不是身体疲惫的抖 / 是戒尺自身累积重量物理化的抖 / [[key-image]] §2.1 锁定的"戒尺累积重量在 Item 9 兑现时达全系统峰值 +18 左右" 的物理兑现）。**Gap**：他预期举戒尺的姿势如常 / 实际戒尺自身重量物理化让他第一次感觉到这柄断戒尺的"全部历史重量"—— 三年来六次入画的累积让戒尺成为他承认仪式的物理化身。

3. **Action**：他举断戒尺**到最高位**（举过头顶 / 戒尺与他的右肘成 180 度直角 / 这是他三年来每次起炉前敲炉沿"叮"那一声前的最高位姿势）。他的内心动作（**这一拍是 Crisis 决定的物理顶点 / 必须让读者清晰看见他已选了**）：他在举到最高位的姿势中**最后一次**在心里物理地排查了三层对位：起炉自死 → 师妹仍要入炉作核（他不知道师妹已经在炉中——这一信息差是 Crisis 决定的悲剧密度的最重要承载）+ 师傅仍要面对女儿入炉 / 拒炉成永奴 → 师妹仍要被天道债逼到炉口 + 师傅仍要面对女儿入炉 + 自己永世为天道债奴 → **没有第三条路**。他在举到最高位的姿势中**做出决定**：**起炉自死**。**Expected**：他以为决定会带来"知道该做什么"的确定感（三年来反丹道每次决定都带来这种确定感）。**Actual**：决定带来的不是确定感——决定带来的是**空**。他第一次在反丹道立宗的胜利的最高位姿势下面对"我做出最大主动选择但我不知道这选择会兑现成什么"的空——这是 [[characters/protagonist-arc]] §6 锁定的"已经选了但选的内容被对方先一步选走的人"的内在动作前奏（他还不知道下一秒师傅会走过）。**Gap 最大**：他预期决定带来确定 / 实际决定带来空——他第一次理解"主动选择不等于知道结局"。

4. **Action**（**Turning Point**）：他在举断戒尺到最高位的姿势中——**他喊**：
   > **"师……"**
   
   **这一字是他做出决定之后的最后一声叫**——他喊"师"字的对象在他自己心里也不明确：可以是师傅（他这辈子最后一次叫师傅的姿势）/ 可以是师妹（他三年来每次反炼时心里叫师妹的姿势）/ 可以是炉沿（他三年来每次敲炉前的潜意识叫名）/ 可以是他自己（他作为反丹道宗师的师承自我）。**这一字未完**—— 戒尺**未落** / 雷音**未起** / 师傅**未走过**（虽然师傅在他身后九步外 / 师傅即将在下一秒走过）。**Expected**：他以为喊出"师……"会让戒尺落下、会让"叮"一声响起、会让起炉自死的动作链开始物理执行。**Actual**：**这一字未完**——本场**停在这一秒**。戒尺举着；"师"字未完；雷音未起；师傅未走过；师妹在炉中（他不知道）。**Gap 最大**：他预期"师……"会接着"叮"声 / 实际"师……"未完 / 戒尺未落 / 雷音未起——他作为反丹道宗师的全部主动选择**停在喊出『师』字的那一秒**；他的所有动作链等待下一秒被夺走（虽然他还不知道下一秒会被夺走）。

#### 5. Turning Point

### Scene 4.4.2 — 拍 3 师傅张嘴说一字 + 雷音同时（**obligatory revelation scene**）
