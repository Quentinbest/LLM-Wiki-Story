## §1 Act 3 概览（全 4 sequences）

### 1.1 一段定调

Act 3 是承认轴从"对立级深化外向兑现"（Act 2 末师傅在远处目睹后什么都没说就走了）跌到"对立级峰值 → 否定之否定过渡帧"（Act 3 末师妹活着 / False Ending）的递降第三段。本幕的设计原则：**主角每赢一次公开胜利，他离那一句师傅的"对"或"错"就远一寸**——他在 Seq 3.1 第一次面对"师傅废我可能不是惩罚"的怀疑但还没法处理它；他在 Seq 3.2-3.3 大典反炼九转还魂赢得三万人雷动的公共承认，但师傅未发一言转身离场；他在 Seq 3.4 入井看见师妹活着——三年自我叙事被釜底抽薪 + 公共承认本身被司徒明璋密折截下未生效 + 帝京疫情爆发钟声同帧合上。

Act 3 同时是 **司徒明璋作为 same-axis foil 首次登场（Seq 3.1）+ 戒尺所有权物理交接（Seq 3.3 第九转）的一幕** —— 司徒明璋在 Seq 3.1 以"亮宗主令牌发出联盟正式拘押令——他自己的手在抖"的姿势登场（[[antagonism-test]] §3 Prescription #1 修订），承担主角动摇"师傅废我不是惩罚"的物理引爆者，但 **绝不说"师弟你对了"**（[[controlling-idea]] §7 违规 #9 / [[cast-design]] §3.2 永不说出口硬约束严守）。

## §2 Sequence 3.1 同门反目 + 联盟拘押令（C 折叠前半 / P#1 antagonism 修订）

> **字数预算**：4-4.5K · **场数**：3 mini-arcs（3.1.1 / 3.1.2 / 3.1.3）
> **主导问题**：沈砺能否在地宫入口与司徒明璋的拦截戏中刺穿司徒明璋一辈子合法性，且在司徒明璋亮宗主令牌发出联盟正式拘押令前不退？
> **答案**：yes-but——他刺穿了对方的合法性（"师兄，你这一辈子的位置是师傅废我那一夜替我空出来的"）；司徒明璋回一句让他愣神的话（"师傅那夜下令的时候，手是抖的。"）+ **亮宗主令牌发出联盟正式拘押令——他自己的手在抖**；主角第一次意识到师傅废他可能不是惩罚，且面对的不只是借来信仰的师兄，是一个手在抖的现任宗主——他比之前更危险，因为他更脆弱
> **承认轴推进**：对立级深化外向兑现（Act 2 末）→ 对立级清晰深化（主角第一次怀疑师傅废他不是惩罚 = 内在层动摇第三波）
> **三层冲突**（[[antagonism-test]] §1.1 C7 修订后总和 10-11）：内在（4——动摇"师傅废我是为惩罚"的核叙事）+ 个人（4——司徒明璋以同侧 foil 形态在场 + 手抖的现任宗主 + 反破都成立的对抗）+ 超个人（3——联盟正式拘押令颁布 / 体制屏障升级）

### Scene 3.1.1 ——地宫入口拦截 + 主角刺穿合法性

- **位置**：Sequence 3.1 内第 1 场（3.1.a 司徒登场 + 主角刺穿）
- **字数预算**：~1.3-1.5K

**进入价值**：承认轴对立级深化外向兑现（Act 2 末状态——师傅"什么都没说就走了"）；情绪轴：沉默承担（他用沉默承担师傅的沉默）。**戒尺**：系腰间（断戒尺隐于左袖下）；**印记**：赤红蔓延一寸（约 55 道劫）；**天道债**：55 道。

**目标（Objective）**：穿过宗门祠堂地宫入口前的广场——他要在大典前三日入地宫第六层确认逆经井的物理位置（为大典反炼后的下井做最后的勘点），他不预期遇见任何人，他不预期遇见司徒明璋。

**冲突（Conflict）**：
- **个人层（主导）**：他穿过广场走到地宫入口前——**司徒明璋站在那里**。这是司徒明璋作为 same-axis foil 的 first appearance（[[characters/situ-mingzhang]] §0 / Act 2 全程不出现的硬约束在此打破）。司徒明璋穿乾元紫道袍、戴九转莲冠、腰间挂乾元印 + **一柄完整的三百年宗门戒尺**（[[characters/situ-mingzhang]] §1.1）——主角第一次在物理空间里看见**完整戒尺的另一柄**（与他左袖下断戒尺形成对位）。司徒明璋的姿态——他没有出手、没有动剑、没有调动联盟戒卫——他只是站在那里，挡在主角与地宫入口之间。
- **超个人层**：地宫入口外侧是宗门祠堂正殿——按宗门律，魔路余孽不可入祠堂百步内；主角已越界。司徒明璋作为现任宗主有权当场拿下他。但司徒明璋没有出手。
- **内在层**：主角看见司徒明璋——他**先扫了一眼司徒明璋的腰间**（找戒尺）；他**第二眼扫司徒明璋的右脚**（找祠堂正殿第七块青砖——[[characters/situ-mingzhang]] §1 微动作 3 / §5 弹药 #3）。**司徒明璋此刻正好踩在第七块青砖上**——他不知道砖下藏着师傅一辈子最大的秘密；主角知道。这一帧主角第一次（在身体上）感觉到**他与司徒明璋是同侧但分叉的两个人**——他读出对方的盲。

**Beats**（5 beats，每一拍 action → reaction，gap 命名）：

1. **Action**：主角走到地宫入口前十步停下。**Expected**：司徒明璋发难（出手 / 调兵 / 宣读联盟律）。**Actual**：司徒明璋只是站着，目光从主角左袖下的鼓起（断戒尺的形状）扫过。*（gap：司徒明璋的"看见"先于他的"动作"——他的姿态不是宗主而是知情者）*
2. **Action**：主角"师兄。"——他以最普通的称呼开口（不是"司徒明璋"也不是"宗主"）。**Expected**：司徒明璋以"魔路余孽"回敬。**Actual**：司徒明璋"沈砺。"——他叫主角的名字，不叫"师弟"也不叫"魔路余孽"。*（gap：两人都拒绝使用宗门律的语言；这一对称是同侧对照的物理形态）*
3. **Action**：主角向前走一步。**Expected**：司徒明璋后退或亮兵刃。**Actual**：司徒明璋不动，但右手食指在拇指内侧轻擦一下（[[characters/situ-mingzhang]] §1 微动作 2——他焦虑期的肌肉记忆暴露）。*（gap：他动了"说宗门律之前的小动作"，但他没有说宗门律——他在准备一句他自己还没决定说不说的话）*
4. **Action**：主角"师兄，你这一辈子的位置，是师傅废我那一夜替我空出来的。"——他用最普通的陈述句说出这句话，没有讥讽、没有控诉、没有情绪。**Expected**：司徒明璋立刻否认 / 以宗门律驳斥 / 当场出手。**Actual**：司徒明璋**沉默三秒**——他没有动作、没有否认、没有出手。他的胸口起伏了一次（[[characters/master]] §3 师傅 Act 2 末同形微动作的回声——同门同动作）。*（gap：他没有否认这件事——他不能否认，因为这是事实；他的沉默就是承认）*
5. **Action**：主角不再开口，等司徒明璋的回应。**Expected**：司徒明璋亮令牌 / 调戒卫 / 用宗门律语言回敬。**Actual**：司徒明璋开口——但他说的不是宗门律的语言，他说的是另一件事（详 Scene 3.1.2 第一拍）。*（gap：主角刺穿了对方的合法性；但对方的回应不是辩护，是另一刀；本场未结束）*

**转折点（Turning Point）**：第 4 拍——**主角说出"师兄，你这一辈子的位置，是师傅废我那一夜替我空出来的"的同时，司徒明璋胸口起伏一次**。这一帧是承认轴对立级清晰深化的物理引爆：司徒明璋的合法性根基（[[characters/situ-mingzhang]] §1.2 / §7 压力点 2 / §6.2 #8）在主角的陈述句下被撕开一道缝——他没有否认，因为事实无法否认；他的沉默承担了主角的刺穿。**但他没有承认主角对了**（[[controlling-idea]] §7 违规 #9 严守 / [[cast-design]] §3.2 永不说出口硬约束）——他用沉默承担，不是承认。

**退出价值**：承认轴对立级深化外向兑现 → 对立级清晰（主角第一次在物理空间里赢得"对方无法否认"的姿势——这一帧是 [[characters/protagonist-arc]] §1.2 主角 want 在 Act 3 开场的物理化）；情绪轴：沉默承担 → 警觉（他知道对方接下来要说一句让他付代价的话）。

**Key Image 关联**：**断戒尺隐于左袖下**——司徒明璋目光扫过左袖下鼓起的形状但**不开口问**（[[key-image]] §2.1 Item 6 → 7 的过渡前奏：完整戒尺与断戒尺第一次在同一物理空间共存，但都未入画作前景）。**累积重量保持**——本场戒尺不出现作前景，但两柄戒尺的物理共存被两人各自读出。

**承认轴 progression 推进**：对立级深化外向兑现 → 对立级清晰（主角首次以陈述句姿势赢得对方无法否认的姿势）。

**印记状态**：赤红蔓延一寸（约 55 道劫）；本场无反炼，账本不变。但**胸口左侧锁骨下三寸的烫疤跳了一格**（[[setting-survey]] §4.1）——这是司徒明璋这个具体人物在场的物理共振：当年戒尺先点的就是这个位置，断戒尺断的就是这一击，烫疤认得司徒明璋的气息。

**潜文本一句**：他说"师兄，你这一辈子的位置，是师傅废我那一夜替我空出来的"那一秒，他实际上在做的事是——他在不让自己说"师兄，你那夜手是抖的"；他知道一旦说出口，他就要承担"我作为承担者听到的是抖的戒尺"这件事；他用陈述合法性的姿势换取自己作为"反丹道执行者"的身份还有几句话的稳定。

### Scene 3.1.2 ——司徒回那一句 + 主角愣神 + 印记疼痛

- **位置**：Sequence 3.1 内第 2 场（3.1.b 司徒回那一句 + 让主角愣神 + 印记疼痛）
- **字数预算**：~1-1.2K

**进入价值**：承认轴对立级清晰（Scene 3.1.1 末状态）；情绪轴：警觉。**戒尺**：仍隐于左袖下；**印记**：赤红蔓延（约 55-56 道劫——本场司徒明璋的话引发胸口烫疤再跳一格）。

**目标（Objective）**：等司徒明璋的回应——他已经刺穿了对方的合法性，他要等对方说一句让他付代价的话，然后他绕过对方进入地宫。

**冲突（Conflict）**：
- **个人层（主导）**：司徒明璋开口——他说"**师傅那夜下令的时候，手是抖的。**"（[[characters/situ-mingzhang]] §7 压力点 2 锁定句）。这句话不是辩护、不是宗门律、不是反驳——这是 **司徒明璋第一次（对外人）承认那一夜不是简单的"师门律法分明"**（[[characters/situ-mingzhang]] §3 维度 1 物理引爆）。他不否认自己执了戒尺（这件事无法否认），但他用师傅那夜手抖这件事撕开了主角三年来对"师傅废我是为惩罚"叙事的根基。
- **内在层**：主角**愣神**——他这辈子第一次（在物理上）面对"师傅废我可能不是惩罚"这个念头。三年来他对师傅废他这件事的全部叙事是：师傅当众判他"走火入魔"=师傅相信他错了=师傅废他是为惩罚。这个叙事是他三年来反丹道动机的承重墙。如果师傅那夜下令的时候手是抖的——那这件事不是惩罚，是别的什么。他还不能命名"别的什么"是什么。他只能愣神。
- **超个人层**：印记反向回应——**胸口烫疤跳一格**（[[setting-survey]] §4.1 / 印记与心理冲击同步——师傅可见印记的物理基础在反方向被司徒明璋的话再点一次）；主角胸口的物理疼痛与他心理上的动摇同帧。

**Beats**（4 beats）：

1. **Action**：司徒明璋开口"师傅那夜下令的时候，手是抖的。"**Expected**：他出手 / 调戒卫 / 宣读联盟律。**Actual**：他说了一件别的事——一件他这辈子从未对任何人说过的事（[[characters/situ-mingzhang]] §6.2 #8 / §7 压力点 2）。*（gap：他付出的代价是他自己一辈子合法性根基的撕开——他用一句话换取主角的愣神，但这句话同时让他自己的根基出血）*
2. **Action**：主角愣神——他的身体停在原地，呼吸停了半秒，他左手不自觉地按到了胸口烫疤位置。**Expected**：主角立刻反驳 / 立刻问"什么意思" / 立刻继续走向地宫入口。**Actual**：主角没动、没问、没继续走——他**承担**这句话；他的肉身比他的脑子先反应。*（gap：他还没意识到这句话改变了什么，但他的身体已经知道了）*
3. **Action**：主角胸口烫疤跳一格——一阵刺痛从胸口左侧锁骨下三寸传到颈侧。**Expected**：他低头看烫疤 / 露出痛苦表情 / 后退。**Actual**：他没有低头、没有表情、没有后退——他**让烫疤跳完那一格**；他知道司徒明璋看不见印记（[[setting-survey]] §4.3 司徒明璋对反向气无视感），他不让自己暴露任何让司徒明璋读出的反应。*（gap：他承担两份疼痛——心理的动摇 + 身体的烫疤跳格——但他不让对方看见任何一份）*
4. **Action**：主角缓缓抬眼看司徒明璋。**Expected**：他回敬一句"师兄，你那夜手是抖的"（这是 [[characters/situ-mingzhang]] §5 弹药 #2 锁定的反向引爆——主角作为承担者的记忆）。**Actual**：他**没有说出这句话**——他咽下去了；他选择让这句话留在心里。*（gap：他的克制比说出来更深——他让那句话在两人之间悬空但不点破；这是同侧对照最深一种相互不承认的预演）*

**转折点（Turning Point）**：第 2 拍——**司徒明璋说出"师傅那夜下令的时候，手是抖的"那一秒，主角的左手按到了胸口烫疤位置**。这一帧是承认轴对立级清晰深化的物理引爆：主角三年来"师傅废我是为惩罚"的核叙事第一次被撕开一道缝，但他还不能处理它——他能做的只有让烫疤跳完那一格，让肉身承担这一句话的重量。**这是 [[characters/protagonist-arc]] 锁定的"对立级深化在内在层第三波动摇"的物理化**。

**退出价值**：对立级清晰 → 对立级深化（内在层动摇——主角第一次怀疑师傅废他不是惩罚但还不能命名替代叙事）；情绪轴：警觉 → 内核动摇（他的承重墙第一次裂开，但他还没倒）。

**Key Image 关联**：**断戒尺在主角左袖下随主角抬手按胸口的动作微微贴住前胸**——这是 [[key-image]] §2.1 Item 6 闪回与现实对位的延伸：当年司徒明璋执的戒尺断那一截贴着主角胸口，**现在另一截在他自己怀里贴着同一位置**——戒尺的两截在同一帧通过两人的物理空间共振。**累积重量微增**（仪式动作的内在承接）。

**承认轴 progression 推进**：对立级清晰 → 对立级深化（内在层"师傅废我是为惩罚"叙事首次被撕开）。

**印记状态**：赤红蔓延（约 56 道劫，+1 道由司徒明璋的话引发的胸口烫疤跳格——这一道是"司徒明璋这个具体人物在场对主角的物理代价"的物质化）；本场无反炼，但印记因心理冲击 +1 道。

**潜文本一句**：他咽下"师兄，你那夜手是抖的"那一句的同时，他实际上在做的事是——他在保护司徒明璋（让对方维持自己"那夜手是稳的"的记忆），但他自己以为只是不愿意把疼痛分给对方；他还不知道这是他作为反丹道执行者第一次对同侧对照的人产生不忍——而不忍这件事本身是承认轴递降的另一个隐秘出口。

### Scene 3.1.3 ——司徒亮宗主令牌 + 联盟拘押令 + 手抖 + 主角不退

- **位置**：Sequence 3.1 内第 3 场（3.1.c 司徒亮令牌发出拘押令 + 手抖 + 主角不退 / Seq 3.1 末 TP）
- **字数预算**：~1.5-1.8K

**进入价值**：承认轴对立级深化（Scene 3.1.2 末状态——内在层动摇第三波）；情绪轴：内核动摇。**戒尺**：仍隐于左袖下（贴胸口）；**印记**：赤红蔓延（约 56 道）。

**目标（Objective）**：绕过司徒明璋进入地宫第六层——他要在大典前三日完成最后的勘点；他知道司徒明璋已经说了那句话，他知道两人都已经各自承担了对方的脆弱；他要继续走他要走的路。

**冲突（Conflict）**（[[antagonism-test]] §3 Prescription #1 修订核心动作）：
- **超个人层（主导首段）**：司徒明璋从腰间解下**乾元印**（联盟首座令牌——[[characters/situ-mingzhang]] §1.1 锁定的物件）——他把令牌举起到自己胸口位置。**他的手在抖**（[[antagonism-test]] §3 Prescription #1 / 维度 1 物理引爆延伸）。读者第一次看见司徒明璋的手抖——这是 [[characters/situ-mingzhang]] §0 "他怕的不是反丹道，他怕的是自己的手抖"（沉鸠那一句）的物理兑现。司徒明璋以令牌发出**联盟正式拘押令**——按宗门律宣告："沈砺以魔路余孽身分入祠堂百步内，联盟七大宗门联议在此发出正式拘押令——三日内自缚于联盟戒坛，否则大典之日联盟戒卫将以斩立决之姿处置。"
- **个人层（主导后段）**：司徒明璋亮令牌时**他的手在抖**——读者立刻读出"手抖"这件事不只是师傅那夜的事，也不只是主角作为承担者的记忆里的事；司徒明璋自己也在抖。这是他维度 1（公开持典 vs 私下回避那一夜）的物理引爆的最深一帧（[[characters/situ-mingzhang]] §3 维度 1 锁定）。主角面对的不只是一个"借来信仰的师兄"，是一个"在他面前手在抖的现任宗主"——他比之前更危险，因为他更脆弱。
- **内在层**：主角看见司徒明璋的手在抖——他**第二次愣神**（Scene 3.1.2 是因为"师傅那夜手是抖的"，本场是因为"司徒明璋自己的手也在抖"）。两次愣神之间他做的全部理解是：**手抖这件事不属于一个人；那一夜手抖的可能不只是师傅**。他还不能命名这是什么，但他第一次（在身体上）感觉到——他三年来的反丹道动机所对抗的对象，**比他想象的更脆弱**。

**Beats**（5 beats）：

1. **Action**：司徒明璋从腰间解下乾元印。**Expected**：他用印牌作威吓 / 调戒卫 / 武力拘押。**Actual**：他举起令牌的姿势是宗门律宣读的标准姿势（双手平举到胸口高度）——但他的手在抖。*（gap：他做的是"现任宗主的标准动作"，但他的手暴露了他自己不能承担这个动作；动作与肉身分裂）*
2. **Action**：司徒明璋宣读拘押令——他用宗门律的语言（[[characters/situ-mingzhang]] §1 嗓音特征——他说话像念条文）。**Expected**：他的声音稳。**Actual**：他的声音稳，但他的右手食指在拇指内侧第二次轻擦（[[characters/situ-mingzhang]] §1 微动作 2 的第二次出现——焦虑期的肌肉记忆在最坚硬的公开姿势下再次暴露）。*（gap：他用最稳的声音说最稳的话，但他的手知道他在说什么——他在签发一份他自己未必完全相信的拘押令）*
3. **Action**：主角看见司徒明璋的手抖——他第二次愣神，但这次他没有让肉身停下；他**继续向前走一步**。**Expected**：司徒明璋后退 / 出手 / 调戒卫执行拘押。**Actual**：司徒明璋**不退**——他举着令牌，让令牌挡在主角与地宫入口之间。两人都没有退。*（gap：两人各自坚守对方无法理解的位置——主角坚守"我要继续走"，司徒明璋坚守"我必须签发这令牌"；两人都没有退，但两人都知道对方为什么不退）*
4. **Action**：主角"师兄。"——他开口，但没有反驳拘押令，没有回敬"你那夜手是抖的"，没有任何对抗性的回应。**Expected**：他争辩 / 出手 / 嘲讽。**Actual**：他只说"师兄"两个字，然后**绕过司徒明璋**——他不抢令牌，不破令牌，他绕过；他用绕过这件事承担拘押令的物理事实（他不破联盟律，但他不停下）。*（gap：他用绕过完成对联盟拘押令的物理抗命——这一动作让超个人对抗从"被拘押"变为"在公开姿势下不退"；他没有给联盟一个可以斩立决的反击姿势，但他也没有承认拘押令）*
5. **Action**：主角走过司徒明璋身侧，走向地宫入口——他经过司徒明璋时，他的左袖下断戒尺与司徒明璋腰间完整戒尺擦过约半寸（两柄戒尺在物理空间中第一次靠到这么近）。**Expected**：司徒明璋出手拦截。**Actual**：司徒明璋**不动**——他举着令牌站在原地，他的手仍在抖；他**让主角过去**。*（gap：他签发了拘押令，但他没有亲手执行——这是 [[characters/situ-mingzhang]] §0 "他怕的是自己的手抖" 在物理空间的最深兑现：他能签字，但他不能动手；他借的赌注让他能宣读，但他借的赌注无法承担动手）*

**转折点（Turning Point）**：第 5 拍——**主角走过司徒明璋身侧，两柄戒尺在物理空间中擦过约半寸，司徒明璋的手仍在抖但他不动手**。这一帧是 Seq 3.1 末 TP 的物理兑现：**联盟正式拘押令颁布（超个人对抗 1 → 3） + 司徒明璋的手抖（个人对抗 3 → 4） + 两柄戒尺第一次物理接近（戒尺所有权交接的前奏）**——三层对抗同帧合上（[[antagonism-test]] §3 Prescription #1 完整兑现）。**但司徒明璋没有动手** —— 这一帧承担"借来信仰的人在最大公开姿势下无法承担动手"的物理形态，与主角"绕过不破"形成同侧对照的镜像：两人都没有退，两人都没有动手，两人都把动作让给对方。

**退出价值**：对立级深化 → 对立级深化加深（外向层：联盟拘押令颁布；内在层：主角面对一个手在抖的现任宗主+一个动摇的"师傅废我是为惩罚"叙事+一道升级的体制屏障——三层同帧加压但他不退）；情绪轴：内核动摇 → 决意（他还没处理那两次愣神，但他选择把处理推到大典之后；他用绕过这一动作钉死自己不退的姿势）。

**关闭的退路**（按 spine §4 C7 锁定）：
- 不能再把司徒明璋当脸谱化反派——他是一个手在抖的具体人。
- 不能再相信"师傅废我是为惩罚"的单一叙事——这一叙事已被司徒明璋的话撕开一道缝。
- 不能再以"私下绕过"完成大典反炼——拘押令颁布让大典反炼成为对联盟律的当众违反。

**下一幕的压力增加**：他必须在三日内完成大典反炼前的全部准备 + 大典当日面对联盟戒卫斩立决之姿（这是 spine 因果链 C7 → C8 的体制屏障升级）。

**Key Image 关联**：**断戒尺与完整戒尺第一次在物理空间擦过约半寸**——这是 [[key-image]] §2.1 Item 6 → 7 的过渡帧（Item 7 是 Seq 3.3 大典反炼第七至第九转的两柄戒尺同时举起未落）。**累积重量微增**（两柄戒尺的物理接近）。**玄色斗笠在 Act 3 不出现的硬约束保持**——司徒明璋戴的是九转莲冠，师傅本场不在场。

**承认轴 progression 推进**：对立级深化 → 对立级深化加深（拘押令颁布 = 公共承认在体制层提前给出了第一道障碍，预演 Seq 3.4 公共承认本身被截下未生效）。

**印记状态**：赤红蔓延（约 56 道劫；本场无新增反炼，但司徒明璋拘押令的物理压力让胸口烫疤再跳一格——印记累积形态保持赤红，未升级到暗黑流动）。

**天道债账本**：约 56 道（无反炼新增）。

**潜文本一句**：他绕过司徒明璋走向地宫入口那一秒，他实际上在做的事是——他在不让自己感谢司徒明璋让他过去；他知道一旦让"感谢"这一念头浮起，他就要承担"对方借的信仰也是一种活着"这件事；他用绕过这一动作换取自己作为反丹道执行者的身份还能继续直到大典；但他左袖下的断戒尺贴住胸口的位置比任何时候都更烫——**他听见自己心跳里有一个比 Act 2 末更深的频率：他不只是在敲自己，他在听一个手在抖的人不让自己倒下的声音**。

### Scene 3.2.1 ——庶民席走出 + 主坛拱手

- **位置**：Sequence 3.2 内第 1 场（3.2.a 庶民席走出 + 拱手 / 金场 1 第 1 小幕）
- **字数预算**：~1.5K

**进入价值**：承认轴对立级深化加深（Seq 3.1 末状态——拘押令颁布 + 内在层动摇）；情绪轴：决意。**戒尺**：系腰间（断戒尺隐于左袖下）；**印记**：赤红蔓延（约 56 道）；**天道债**：56 道。

**目标（Objective）**：从庶民席走出，向主坛拱手——他要在三万人面前完成位置反转的物理仪式：以"白身"（无资格上坛）的身份在公开仪式的开场之刻**主动认领反炼九转还魂的位置**（[[setting-survey]] §5.3 锁定的循环三大典开场——"他从庶民席中走出，在三万人面前向主坛拱手"）。

**冲突（Conflict）**：
- **超个人层（主导）**：祭天台位于帝京东郊乾元宗山主峰祠堂前的九转祭坛，三万人观礼席环绕祭坛三周；主坛中央立着师傅（玄琢真人——退到联盟首席之前他执宗主令，现在他作为前任宗主+反炼九转还魂的立宗人，在主坛中央持典）；主坛侧立司徒明璋（现任宗主，腰间挂乾元印 + 完整戒尺）；七大宗门长老分立祭坛四角；联盟戒卫四十八人持戟环绕祭坛外圈（联盟拘押令的公开执行准备）。这是 [[setting-survey]] §5.3 锁定的丹道大典最大公开仪式。
- **个人层**：主角穿着庶民布衣（无任何宗门服色），他坐在庶民席最后一排（按 [[setting-survey]] §5.3 锁定，白身无资格上坛，只能坐庶民席）。他与师傅之间隔三万人 + 主坛 + 七大宗门长老 + 四十八联盟戒卫——这是物理距离的最大形态。但他知道师傅在台上看着他。
- **内在层**：他**必须不看师傅**——他知道一旦看师傅一眼，他三年来反丹道动机所建立的"我必须不依赖师傅承认"的姿势会崩。他要赢的不是师傅的承认，他要赢的是"我可以在不看师傅的姿势下完成反炼"。但他知道师傅在台上看着他——这是他这辈子最难的不看（[[characters/protagonist-arc]] §3 Pin 8 锁定的"主角扫师傅一眼"将在 Seq 3.3 第七转兑现；本场是 Pin 8 之前的最长不看）。

**Beats**（5 beats）：

1. **Action**：开场大典钟响（[[setting-survey]] §5.3 锁定的丹道大典开场钟）。**Expected**：第一位宗门长老登坛持典宣告大典开始。**Actual**：钟响第三声末——**主角从庶民席最后一排起身**。庶民席的人没注意（他穿庶民布衣无任何特征），但他起身的动作引起前几排人的回头。*（gap：他用最普通的起身完成最非典的动作——白身在大典开场之时主动起身向主坛走）*
2. **Action**：主角从庶民席最后一排起身，走向主坛——他穿过庶民席中央的过道。**Expected**：联盟戒卫立刻拦截（按拘押令斩立决之姿）。**Actual**：联盟戒卫**犹豫了三秒**——他们看见主角起身但他没有出手没有亮兵刃，他只是在走；他们看向司徒明璋等指令。*（gap：联盟戒卫的犹豫暴露了拘押令在大典开场之时执行的体制摩擦——主角不出手让斩立决的姿势无法立刻启动）*
3. **Action**：主角走到祭坛前三百步处——他**在三万人面前停下**。庶民席的人完全静了；观礼席的修真界三万人完全静了。**Expected**：他继续走或回头。**Actual**：他**向主坛拱手**——他双手抱拳，向主坛中央的师傅深揖一礼。这是 [[setting-survey]] §5.3 锁定的"这一拱手是他对师傅这辈子的最后一礼"——但他**没有看师傅的脸**（他的目光始终落在主坛中央地砖上，不抬起）。*（gap：他用最古礼的姿势向师傅行礼，但他不看师傅；这是他这辈子最矛盾的一礼——他承认师傅在那里，但他不让自己见师傅）*
4. **Action**：主角拱手完毕——他**直起腰**，不等回应（他不预期师傅回礼，他不预期任何回应；他要的是这一拱手本身，不是回应）。**Expected**：司徒明璋立刻喝令逐出（按拘押令）。**Actual**：司徒明璋**举起乾元印**——他向主坛中央示意"以宗主之名宣告逐出"。他的手还在抖（[[characters/situ-mingzhang]] §1 微动作 / Seq 3.1.3 的手抖延续）。*（gap：司徒明璋按宗门律必须当场逐出，他举令牌，但他的手抖暴露他自己也在挣扎——逐出令将在第 5 拍兑现或被止）*
5. **Action**：司徒明璋将开口宣读逐出令。**Expected**：师傅沉默 / 师傅默许 / 师傅亲自下令逐出。**Actual**：师傅**开口**——他说三个字（详 Scene 3.2.2 第一拍）。*（gap：师傅这辈子第二次开口说与主角直接相关的话——第一次是 23 年前那夜判主角"走火入魔"下令废徒；本场是第二次，但说的是相反方向的话；本场未结束）*

**转折点（Turning Point）**：第 3 拍——**主角向主坛拱手但不看师傅的脸**。这一帧是 Seq 3.2.1 末 TP 的物理兑现：**位置反转**（白身走出庶民席向主坛行礼 = 物理空间的逆转，他作为"无资格上坛者"主动认领反炼九转还魂的位置）+ **不看的姿势**（他承认师傅在那里但不让自己见师傅 = 内在层"我必须不依赖师傅承认"的姿势的物理化）—— 两层同帧合上。

**退出价值**：对立级深化加深 → 对立级深化（公共仪式开场 / 主角主动认领位置 / 但不看师傅 = 他赢得的是"我可以不看师傅的姿势下完成反炼"的开场资格）；情绪轴：决意 → 决意深化（他用拱手钉死自己接下来的全部行动；他不能再回头）。

**Key Image 关联**：**断戒尺仍隐于左袖下**——他向主坛拱手时左袖随抱拳动作贴住胸口烫疤位置；断戒尺与烫疤同帧贴合（[[key-image]] §2.1 Item 7 大典反炼前奏：断戒尺第一次以"贴住烫疤位置"的姿势在公开仪式中入画）。**累积重量保持**（戒尺尚未敲炉沿；戒尺敲炉沿将在 Scene 3.2.3 反炼第一转开炉时兑现）。**玄色斗笠仍不出现**——师傅在主坛中央但戴的是宗主玄色衣袂+九转莲冠（与 Seq 2.3 自封丹田反向场同款），不是出门便装的斗笠形态。

**承认轴 progression 推进**：对立级深化加深 → 对立级深化（公共仪式中位置反转 / 主角主动认领位置但不看师傅）。

**印记状态**：赤红蔓延（约 56 道；本场无反炼，账本不变）。但**胸口烫疤在他拱手深揖时跳了半格**——这是师傅作为可见印记的观察者第一次在公开空间看见徒弟胸口印记的物理共振（[[setting-survey]] §4.3 锁定的师傅可见印记 + Seq 2.5.3 末的延续——师傅在 Act 2 末已经看见过赤红一格，本场他第一次（在公开空间）再次看见徒弟印记的状态）。

**潜文本一句**：他向主坛拱手不看师傅那一秒，他实际上在做的事是——他在不让自己抬眼；他知道一旦抬眼，他就会发现师傅的眉毛比 Seq 2.5.3 那次又灰了一寸；他用不看换取自己作为反丹道执行者的姿势还能继续直到第九转；但他左袖下的断戒尺贴住胸口烫疤的位置比任何时候都更烫——**他听见自己心跳里有一个新的频率：他在敲一个不允许自己听见的声音**。

### Scene 3.2.2 ——师傅以"让他炼"止司徒明璋逐出令 + 反向丹炉悬挂梁柱

- **位置**：Sequence 3.2 内第 2 场（3.2.b 师傅止逐出令 + 反向丹炉悬挂祭坛上方梁柱 / 金场 1 第 2 小幕）
- **字数预算**：~1.8K

**进入价值**：承认轴对立级深化（Scene 3.2.1 末状态——拱手但不看师傅）；情绪轴：决意深化。**戒尺**：仍隐于左袖下；**印记**：赤红蔓延（约 56 道）。

**目标（Objective）**：在司徒明璋逐出令被止之后，完成反向丹炉的悬挂——他要把自带的反向丹炉以铁链悬挂祭坛上方梁柱（[[setting-survey]] §3.2 反方丹标准结构 / 丹炉倒挂 + 不用宗门祭炉的物理形态）；这是反丹道第一次在公开仪式中以视觉形态出现在大典舞台中央。

**冲突（Conflict）**：
- **个人层（主导）**：师傅在主坛中央 —— 他**开口**说三个字："让他炼。"（[[setting-survey]] §5.3 锁定句）。这是师傅这辈子第二次开口说与主角直接相关的话；第一次是 23 年前下令废徒；本场是相反方向。司徒明璋僵在原地——他举着的乾元印停在半空，他的手抖比之前更明显。**师傅没有看主角**（与主角不看师傅形成镜像对位——两人都在不看对方）；师傅的目光落在主坛地砖上某一处（[[characters/master]] §1 师傅的眼睛微动作锁定）。
- **超个人层**：师傅的"让他炼"作为前任宗主+反炼九转还魂的立宗人的姿势，**直接撤销了司徒明璋作为现任宗主的拘押令**（按宗门律，前任宗主可对当任宗主的体制行为进行"宗师驳令"——但此举几乎不被使用，因为这意味着前任宗主公开撕开当任宗主的合法性）。这是 [[antagonism-test]] §4.1 锁定的师傅作为 antagonist-as-mirror 在 Act 3 开场付出的代价：**他用"让他炼"三个字撕开了自己亲手扶上位的司徒明璋的合法性**——他亲手撤销自己的赌注的红利接收者的体制位置。
- **内在层**：主角**听见师傅说"让他炼"**——他**没有抬头**（仍不看师傅）。但他的左手不自觉地按到胸口烫疤位置（与 Seq 3.1.2 的同形动作）。他三年来反丹道动机的全部支撑是"师傅不会再认我"——师傅的"让他炼"不是认，是默许；但默许比认更难承担——他不知道这是 [[characters/master]] §1 师傅的 Want vs Need 推进（[[antagonism-test]] §4.3 Want 出现裂缝 / Need 浮出的物理化）；他只知道他必须不让自己读懂师傅这三个字。

**Beats**（6 beats）：

1. **Action**：师傅开口："让他炼。"**Expected**：师傅沉默 / 师傅默许 / 师傅亲自下令逐出。**Actual**：师傅说三个字，撤销司徒明璋逐出令。*（gap：师傅这辈子第二次开口说与主角直接相关的话，且方向与第一次完全相反；师傅自己的赌注的红利接收者的合法性被他亲手撕开）*
2. **Action**：司徒明璋僵在原地，他举着乾元印的右手抖得比之前更明显。**Expected**：司徒明璋反驳 / 坚持逐出令 / 撤回令牌。**Actual**：司徒明璋**缓缓放下乾元印**——他不撤回但他不执行；他承担师傅"让他炼"的撤销但他保留令牌悬而未决（[[characters/situ-mingzhang]] §3 维度 1 / §0 借来信仰的人在最大公开姿势下无法承担动手的延续）。*（gap：他用放下令牌完成对师傅撤销的承担，但他不撤回令牌——这是他作为现任宗主对师傅前任宗主姿势的最大让步）*
3. **Action**：主角听见师傅的三个字——他左手按到胸口烫疤位置。**Expected**：主角抬头看师傅 / 谢师傅 / 任何回应师傅的动作。**Actual**：主角**没有抬头、没有谢、没有任何回应**——他**继续完成既定动作**：他抬手向身后挥示意；他的两名沉鸠系弟子（[[setting-survey]] §3.5 锁定的沈砚隔代弟子群体——这是反丹道传承的物理基础，非主角金手指；他们在大典前已秘密入帝京作主角后勤）从庶民席外搬来主角自带的反向丹炉（黑铜质、炉口朝下、炉腹有反向气流的细纹刻痕——[[setting-survey]] §3.2 反方丹炉标准形态）。*（gap：他用不回应师傅完成对师傅"让他炼"的最大尊重——他不让自己读懂这三个字，他只让自己继续动作；但他的左手按胸口暴露他听见了）*
4. **Action**：两名沉鸠系弟子搬来反向丹炉，主角抬手指祭坛上方梁柱（祭坛上方有四根三百年宗门正殿梁柱，按宗门律是"承天梁"——丹炉应正立于祭坛中央与承天梁同向）。**Expected**：他正立丹炉于祭坛中央。**Actual**：他指示弟子**以铁链将反向丹炉悬挂祭坛上方梁柱**——丹炉倒挂、炉口朝下、铁链由承天梁吊下。这是 [[setting-survey]] §3.2 反方丹炉标准形态在三万人公开仪式中的首次视觉化（[[setting-survey]] §6 硬规则锁定）。*（gap：反丹道的物理形态第一次以"祭坛上方倒挂"的姿势在三万人面前出现；承天梁本是承"正方"丹炉之气的物理基础，现在被反过来用作反方丹炉的悬挂支点；反方对正方的物理颠倒在视觉上首次完成）*
5. **Action**：反向丹炉悬挂完毕，铁链微晃。**Expected**：联盟戒卫立刻执行拘押令（拘押令未撤回，按宗门律执行仍合法）。**Actual**：联盟戒卫**看向司徒明璋**等指令——司徒明璋仍举着乾元印于半空（虽然他刚才将令牌缓缓放下，但他没有撤回令牌）；他**不发指令**。戒卫的戟尖朝下，没有上前。*（gap：拘押令在体制上仍生效，但司徒明璋作为签发者不发指令——拘押令成为悬而未决的体制屏障；主角赢得的不是拘押令被撤销，是拘押令在大典反炼期间被悬空）*
6. **Action**：主角站在祭坛中央，反向丹炉在他头顶倒挂——他**抬手摸炉沿**（这是反炼起炉前的反方仪式动作；[[setting-survey]] §3.2 反方丹炉标准结构的火门倒拨准备）。**Expected**：他直接起炉。**Actual**：他抬手摸炉沿之后**停一秒**——他的左手仍按在胸口烫疤位置。他在等什么？他在听师傅会不会再开口。师傅没有再开口。一秒之后，他**放下左手**，**右手按炉沿火门倒拨**。*（gap：他用一秒的停顿暴露了他在等师傅再说一句话——但师傅没有；他放下左手钉死自己接下来的全部行动；他在不让自己看师傅的姿势下放弃了"师傅会再说一句"的最后一丝期待）*

**转折点（Turning Point）**：第 1 拍 + 第 6 拍 **合并**——**师傅说"让他炼"那一秒 + 主角放下左手按炉沿那一秒**。前者是师傅这辈子第二次为主角说话的物理化（[[characters/master]] §1 Want vs Need 推进 / [[antagonism-test]] §4.3 Want 撤回到最低 / Need 浮出的物理化）；后者是主角不让自己读懂师傅的姿势的物理化（他放下左手意味着他不再期待师傅再开口；他承担师傅的三个字本身就是全部）。两拍合上构成对立级深化的最深一帧：师傅第一次为主角说话，主角第一次主动放弃期待师傅再说话——两人都让动作让给对方。

**退出价值**：对立级深化 → 对立级深化加深（公共仪式中反丹道物理形态首次视觉化 + 师傅第一次为主角说话但主角不让自己读懂 = 公共承认到来的前奏 / 私密承认未到的进一步预演）；情绪轴：决意深化 → 决意彻底（他放下左手钉死自己不再期待师傅再开口）。

**Key Image 关联**：**反向丹炉以铁链悬挂祭坛上方梁柱**——这是反方丹炉在三万人公开仪式中首次视觉化，构成反丹道 image system 的最大公开形态（[[key-image]] §1.2 反丹炉的倒挂姿势作为反丹道 image system 主轴 / 但不替代 Key Image 戒尺）。**断戒尺仍隐于左袖下未入画**——主角抬手按炉沿火门倒拨的动作之前，他的左手按胸口烫疤位置；这是 [[key-image]] §2.1 Item 7 戒尺力学反讽顶点（Seq 3.3 大典反炼第七至第九转）的最近一帧前奏。**累积重量保持**（戒尺尚未敲炉沿）。

**承认轴 progression 推进**：对立级深化 → 对立级深化加深（师傅第一次为主角说话 / 主角放弃期待 / 反丹道物理形态首次视觉化）。

**印记状态**：赤红蔓延（约 56 道；本场尚未开炉反炼，账本不变）。但**胸口烫疤在师傅说"让他炼"那一秒跳了半格**——这是师傅这三个字对主角的物理共振；与主角不让自己读懂师傅这三个字的姿势形成内外反向（外在压抑+内在共振）。

**潜文本一句**：他放下左手按炉沿那一秒，他实际上在做的事是——他在让师傅那三个字成为他余生唯一可以反复回响的私密物质；他知道一旦让自己读懂"让他炼"，他三年来"师傅不会再认我"的核叙事会立刻塌方；他用放下左手换取自己继续完成反炼第一转的姿势，但他左袖下的断戒尺贴住胸口烫疤的位置告诉他——**师傅说"让他炼"那三个字已经成为他余生不可能再问的问题的第一份本金**。

### Scene 3.2.3 ——反炼第一至第六转 / 六转丹气逆走 + 长老开始动摇

- **位置**：Sequence 3.2 内第 3 场（3.2.c 反炼第一至第六转 / 金场 1 第 3 小幕）
- **字数预算**：~1.7K

**进入价值**：承认轴对立级深化加深（Scene 3.2.2 末状态——师傅"让他炼" + 主角放下左手）；情绪轴：决意彻底。**戒尺**：仍隐于左袖下；**印记**：赤红蔓延（约 56 道）。

**目标（Objective）**：完成反炼九转还魂的前六转——他要在三万人观礼席的注视下、师傅在主坛中央的注视下（他不看师傅）、司徒明璋仍举令牌的注视下、联盟戒卫四十八人持戟环绕的注视下，完成前六转的丹气逆走（[[setting-survey]] §3.2 反方丹标准结构的四变量同步操作 + 二品反炼级的丹气流向）；这是反丹道在公开仪式中的物理兑现的前 6/9。

**冲突（Conflict）**（[[antagonism-test]] §1.1 C8 = 15 的前半承担）：
- **超个人层（主导）**：三万人观礼席 + 七大宗门联议 + 联盟戒卫四十八人——这是 Act 3 超个人对抗的物理峰值的前半（[[antagonism-test]] §1.1 C8 = 15 / 超个人 5 的物理基础）。前六转丹气逆走每一转都让承天梁微震一次——这是承天梁三百年来第一次承受倒挂丹炉的反向丹气压力（[[setting-survey]] §3.2 + §3.3 反丹副作用首现的公开物理化）；承天梁微震传到祭坛地砖，传到七大宗门长老的脚下；**长老开始动摇**（[[act-design]] §1 Seq 3.2 锁定句"六转丹气逆走、长老开始动摇"）。
- **个人层**：师傅在主坛中央——他**看着主角的炉**（他没有看主角的脸，他看的是炉的丹气流向）；这是 [[characters/master]] §1 师傅的眼睛微动作的延伸（他眼里在算评估炉力与丹气流向的兼容性）。司徒明璋仍举令牌（虽然他放下过一次但他没撤回）—— 但他的手抖比之前更明显；他的眼睛**第一次看见反向丹气的颜色**（虽然他作为正方丹师对反向气有视感局限 [[setting-survey]] §4.3，但二品反炼级的丹气浓度让他即使无视感也能感觉到压力的方向）。
- **内在层**：主角在反炼过程中**必须不看师傅**——但他知道师傅在看他的炉。这是他这辈子最难的不看的第二段（第一段是 Scene 3.2.1 的拱手不看；第二段是反炼六转过程的不看，连续约半个时辰的"知道对方在看自己但自己不看对方"）。每一转的丹气逆走都让他胸口烫疤跳一格——印记累积从约 56 道 涨到约 80 道（六转 × 约 4 道/转 = +24 道，[[setting-survey]] §4.2 二品反炼级的前 6/9）。

**Beats**（6 beats）：

1. **Action**：主角按炉沿火门倒拨——反向丹炉的火门反向逆时针拨动一格；炉腹开始有反向气流的细纹浮起（[[setting-survey]] §3.2）。**第一转开炉**。**Expected**：丹气立刻逆走 / 立刻成形。**Actual**：第一转的丹气**缓慢成形**——主角用反向相克序倒着下第一组药材（[[setting-survey]] §3.2 反方下锅顺序），气流离心扩散；炉腹微动。第一转完成约耗时一刻钟。*（gap：反炼的物理过程比正方炼丹慢一倍——反向相克序需要药材在炉内反向自相消解才能进入下一阶段；这一慢是反丹道的物理代价的视觉化）*
2. **Action**：第二转 / 第三转——主角连续完成两转；炉腹的反向气流由细纹变粗纹；承天梁第一次微震（铁链微晃约半寸）。**Expected**：观礼席静默。**Actual**：观礼席**第一次有声**——前几排有人低声议论；中段几排有人开始记录（修真界三万人中有约三分之一是丹师或丹道相关，他们看出了反向气流的物理形态）。*（gap：观礼席的低声议论从沉默变为窃语 = 反丹道第一次在公开仪式中获得修真界部分人的"物理认可" = 但这一认可仍是体制层的悬空认可，不是承认轴的私密认可）*
3. **Action**：第四转——主角的胸口烫疤跳到约 70 道；他的左手第一次不自觉地按到胸口（与 Seq 3.1.2 / Seq 3.2.2 同形动作的第三次出现）。**Expected**：他停手 / 失力 / 任何不稳的迹象。**Actual**：他**没有停手**——他用右手单手继续操作炉沿，左手按胸口；他的额角第一次见汗。*（gap：他的身体开始承担反炼的物理代价；但他不让代价让自己停下；这是 [[characters/protagonist]] §3 维度 2 "慈悲到自虐 vs 敢拿别人命赌一炉"的进一步引爆，但本场是引爆 "敢拿自己命赌一炉" 一侧）*
4. **Action**：第五转——承天梁微震第二次（铁链微晃约一寸）；**七大宗门长老中的一位**（不具名 / 老一辈正方丹师 / 用户已拒绝 antagonism-test §3 Prescription #3 具名新增角色）**第一次微微低头**——他在祭坛侧位看反向气流的物理形态时低头检视脚下祭坛地砖的微震（保留"长老动摇"的体制层物理化作为公共承认前奏，但不命名 / 不构成新独立角色）。**Expected**：长老们维持宗门律的静默姿势。**Actual**：那位长老低头检视；其余六位长老的眼睛余光跟随他的低头。*（gap：长老开始动摇 = 七大宗门联议中的第一位长老的物理姿势第一次偏离宗门律 = 反丹道在体制层赢得第一帧的视觉化承认，但这一承认仍是私下的微动作不是公开宣告）*
5. **Action**：第六转——主角的胸口烫疤涨到约 80 道（赤红蔓延深化但未升级到暗黑流动）；他完成第六转的丹气逆走。**Expected**：他立刻进入第七转。**Actual**：他**停一秒**——他的右手按炉沿火门，第六转的丹气在炉腹中稳定旋转（反向）；他**听见祭坛侧位有人脚步声**。这一脚步声是主坛中央——**师傅持戒尺登台**走向祭炉旁三步（[[premise-card]] §7 Q3 + [[setting-survey]] §5.3 + [[key-image]] §2.1 Item 7 锁定）；但师傅的脚步声还在祭炉外约三步——师傅尚未走到祭炉旁三步的位置；师傅的登台动作刚刚开始。*（gap：第六转完成后主角第一次（在身体上）感觉到师傅在动——但师傅还没走到祭炉旁；这一帧是 Seq 3.3 开场的前奏，本场未结束师傅持戒尺登台的全过程）*
6. **Action**：主角听见师傅脚步声——他**没有抬头**。他**继续准备第七转**——但他的右手停在火门上没有立刻拨动；他在等。**Expected**：他立刻拨动火门进入第七转。**Actual**：他**等师傅走到祭炉旁三步**（这是 [[setting-survey]] §5.3 锁定的师傅持戒尺立于祭炉旁三步的物理位置）——他要让师傅持戒尺登台与第七转同帧开始。*（gap：主角主动让出第七转开炉的瞬间，以让师傅持戒尺登台与第七转同帧合上；他要把"承认轴对立级峰值"的两个物理元素同时叠加；这是他作为反丹道执行者对仪式美学的最深主动控制；本场结束于师傅脚步声靠近祭炉旁三步——第七转尚未开始）*

**转折点（Turning Point）**：第 4 拍 + 第 5 拍**合并**——**一位长老（不具名）第一次微微低头 + 主角完成第六转停一秒等师傅脚步声**。前者是七大宗门长老开始动摇的物理化（体制层的反丹道认可第一次微动作出现）；后者是主角主动让出第七转开炉瞬间以等师傅持戒尺登台（仪式美学的主动控制）。两拍合上构成 Seq 3.2 末 TP 的物理兑现：**公共承认到来的前奏（长老动摇）+ 仪式美学的主动等待（主角让出节拍）**——反丹道在公开仪式中赢得物理认可的同时，主角让出节拍以让师傅持戒尺登台与第七转同帧合上。

**退出价值**：对立级深化加深 → 对立级深化（公共承认到来的前奏 / 长老开始动摇 / 主角主动让出节拍 = 反丹道物理兑现的前 6/9 + 师傅持戒尺登台的前奏）；情绪轴：决意彻底 → 等待（他主动让出第七转开炉的瞬间以等师傅持戒尺登台）。

**Key Image 关联**：**第七转的开始与师傅持戒尺登台合一**（Seq 3.3 开场兑现 / [[key-image]] §2.1 Item 7）——本场结束于师傅脚步声靠近祭炉旁三步；师傅持戒尺登台立于祭炉旁三步将在 Seq 3.3 开场兑现（[[premise-card]] §7 Q3）。**断戒尺仍隐于左袖下**——主角的左手按胸口烫疤位置（第三次同形动作）；断戒尺将在 Seq 3.3 第七转敲炉沿（Seq 3.3 戒尺敲炉沿第四次入画，[[key-image]] §2.1 戒尺累积重量 +3 / 戒尺力学反讽顶点）。**累积重量微增**（仪式动作的延续）。

**承认轴 progression 推进**：对立级深化加深 → 对立级深化（公共承认到来的前奏 / 长老开始动摇 / 主角主动让出节拍以与师傅持戒尺登台合一）。

**印记状态**：赤红蔓延一寸（约 80 道劫——六转反炼累积 +约 24 道，[[setting-survey]] §4.2 二品反炼级的前 6/9）；本场反炼前六转完成，账本从 56 道涨到 80 道。胸口烫疤仍是赤红一格但已蔓延到锁骨下三寸的两倍范围（[[setting-survey]] §4.1 第 2-3 阶段的过渡带）。

**天道债账本**：约 80 道（前六转反炼累积）。

**潜文本一句**：他完成第六转停一秒等师傅脚步声那一秒，他实际上在做的事是——他在让师傅持戒尺登台与第七转同帧合上；他知道师傅持戒尺登台是承认轴对立级峰值的物理元素，他要把这两个元素同时叠加以让反炼九转还魂成为"师傅在台上看着他但他必须不看师傅"的最大公开兑现；他用让出节拍换取自己作为反丹道执行者对仪式美学的主动控制，但他左袖下的断戒尺贴住胸口烫疤的位置告诉他——**师傅每走一步靠近祭炉旁，他三年来反丹道动机所建立的"我可以不看师傅"的姿势就薄一寸；师傅持戒尺登台立于祭炉旁三步那一帧，他可能不得不抬头看师傅一眼**——但这一帧的兑现在 Seq 3.3，本场不让它发生。

### Scene 3.3.1 ——第七转开炉 + 师傅持戒尺登台 + 主角扫师傅一眼 + 戒尺举起未落

- **位置**：Sequence 3.3 内第 1 场（3.3.a 金场 1 第 4 小幕 / 戒尺力学反讽顶点）
- **字数预算**：~2K

**进入价值**：承认轴对立级深化（Seq 3.2.3 末状态——师傅脚步声靠近祭炉旁约三步外）；情绪轴：等待（他主动让出第七转开炉瞬间以等师傅持戒尺登台）。**戒尺**：断戒尺仍隐于左袖下；师傅持完整戒尺即将立于祭炉旁三步。**印记**：赤红蔓延（约 80 道）。

**目标（Objective）**：让师傅持戒尺登台与第七转开炉同帧合上——他要把这两个物理元素严格叠加在同一拍内兑现（[[premise-card]] §7 Q3 + [[setting-survey]] §5.3 + [[key-image]] §2.1 Item 7 锁定），然后完成第七转的丹气逆走，**断戒尺敲炉沿一下"叮"**（戒尺第四次入画 / 累积重量 +3）。

**冲突（Conflict）**：
- **个人层（主导）**：师傅持完整戒尺从主坛中央走到祭炉旁三步——这是 [[characters/master]] §1 师傅 Act 3 的物理动作核心；他的脚步声不快不慢，他的眼睛看炉不看主角；他走到祭炉旁三步停下，**举起手中完整戒尺**——但**戒尺举起未落**（[[premise-card]] §7 Q3 / [[key-image]] §2.1 Item 7 戒尺力学反讽顶点）。师傅举戒尺的姿势是宗门律的"裁决位"，他作为前任宗主有权当场打断反炼仪式；但他**没有让戒尺落下**——这是 [[characters/master]] §3 维度 1 + 2 + 3 三层同帧合上的最大物理形态。
- **超个人层**：三万人观礼席静默——所有人都在看戒尺举起未落的姿势；七大宗门长老看着师傅；联盟戒卫四十八人持戟环绕但他们的目光全在祭炉与师傅之间的三步空间上；承天梁微震第三次（铁链微晃约一寸半）。
- **内在层**：主角扫师傅一眼——这是他这辈子最难的不看在 Seq 3.2 持续约半个时辰之后的**首次破戒**（[[characters/protagonist-arc]] §3 Pin 8 锁定的兑现帧）；他的眼睛从炉沿抬起，越过祭炉边缘三寸的火气，落在师傅举戒尺的右手腕上——他**第一次（在身体上）感觉到师傅举戒尺的手没有抖**；与 Scene 3.1.2 司徒明璋说"师傅那夜下令的时候手是抖的"形成内在层最大对位——三年前下令的那只手在抖、今天举戒尺的这只手不抖。他还不能命名"为什么不抖"。

**Beats**（6 beats）：

1. **Action**：师傅持完整戒尺从主坛中央走到祭炉旁三步停下。**Expected**：师傅当场打断反炼仪式（按宗门律此时是最后一次合法打断窗口）。**Actual**：师傅**只是站到了祭炉旁三步**——他没有出手，他没有说话，他只是站到了"裁决位"上（[[setting-survey]] §5.3 锁定的物理位置）。*（gap：他用站位代替动作——他作为前任宗主的最后一次裁决权被他自己的站位姿势悬空保留；他在等什么但他不让人读出他在等什么）*
2. **Action**：师傅举起手中完整戒尺——戒尺由腰间挂位平举至与眉同高，戒尺尖端朝向祭炉沿三步外的虚空。**Expected**：戒尺落下打断仪式。**Actual**：**戒尺举起未落**——师傅的右手腕悬停在与眉同高的位置，戒尺不动；他的手腕**没有抖**（[[characters/master]] §3 维度 1+2+3 同帧合上）。*（gap：戒尺力学反讽顶点的兑现——师傅作为"师徒情"主轴的最大物理姿势就是举起戒尺但不落下；他的不抖与 Seq 3.1.2 司徒明璋所述"师傅那夜手是抖的"形成内在层最大对位）*
3. **Action**：主角抬眼——他从炉沿抬起眼，越过祭炉边缘三寸的火气，**扫师傅一眼**（[[characters/protagonist-arc]] §3 Pin 8 兑现帧）。**Expected**：他看师傅的脸。**Actual**：他**只看师傅举戒尺的右手腕**——他不让自己看师傅的脸；他知道一旦看脸他会读出师傅这三年来在远处看他的姿势的全部重量；他选择只看手腕。他看见**那只手没有抖**。*（gap：他破了 Seq 3.2 持续约半个时辰的不看，但他破得最小——他只看一寸的右手腕；这是 [[characters/protagonist]] §5 弹药 #6 锁定的"主角的看比他的不看更克制"的物理化）*

### Scene 3.3.2 ——第八转地宫共振 + 主角认出师妹活着

- **位置**：Sequence 3.3 内第 2 场（3.3.b 金场 1 第 5-6 小幕 / 同盟揭密前奏）
- **字数预算**：~2K

**进入价值**：承认轴对立级峰值的前奏（Scene 3.3.1 末状态——师傅戒尺举起未落 + 主角断戒尺敲炉沿）；情绪轴：共在。**戒尺**：师傅完整戒尺仍举起未落（位置：祭炉旁三步、与眉同高）；断戒尺已敲炉沿一下"叮"完成（位置：炉沿上方约半寸悬停）；**印记**：约 82 道。

**目标（Objective）**：完成第八转——反向气流从粗丝变粗带（[[setting-survey]] §3.2 二品反炼级第八转的丹气流向），九色丹气在炉腹中形成稳定的反向旋转环；他不预期地宫深处有任何动静，他不预期反向气有"另一个源头"。

**冲突（Conflict）**：
- **超个人层（主导）**：**地宫深处反向气升起**（[[setting-survey]] §3.2 + §3.4 反向核机制的物理前奏）——第八转的反向气共振让地宫第七层的反向核（师妹）开始苏醒；地宫第七层的逆经井底（位置距祭坛地下约九十丈）开始释放一道极细的反向气脉，顺着九层地宫的反向气脉网（[[setting-survey]] §5.2 地宫九层物理勘测）向上传导；这道反向气脉在第八转的丹气逆走完成那一秒到达祭炉下方约三丈的位置，与主角第八转的炉腹反向气流**共振**——共振的物理形态是承天梁的微震第四次（铁链微晃约二寸 / 比前三次明显加大）。
- **个人层**：师傅戒尺仍举起未落（与 Scene 3.3.1 同形姿势，但此时已悬空约一刻钟）；师傅的眼睛**第一次（在身体上）感觉到反向气脉的共振方向**——他的视线从祭炉炉腹微微下移约半寸（[[characters/master]] §1 眼睛微动作 / [[setting-survey]] §4.3 师傅作为前任宗主对反向气有视感的物理基础）；他读出**反向气有第二个源头**，但他不让任何人读出他读出来了。司徒明璋仍举乾元印于半空、手抖比之前更明显；他**不知道**反向气有第二个源头（[[setting-survey]] §4.3 司徒明璋对反向气无视感）。
- **内在层**：主角在炉沿——他第八转的操作姿势中**第一次（在物理上）感觉到丹气流向有"另一个共振点"**——他的右手按炉沿火门、他的左手仍持断戒尺停在炉沿上方约半寸；他的呼吸停半秒；他**第一次（在身体上）感觉到地宫深处有反向气共振**（[[setting-survey]] §3.4 反向核机制）。他还不能命名这是什么，但他知道反向气有"另一个源头"——他这辈子第一次（在物理上）面对反向气有第二个源头这件事。**他还没认出这是师妹**，但他的肉身已经开始读出"这道反向气有人的形状"。

**Beats**（6 beats）：

1. **Action**：主角继续第八转操作——反向气流从粗丝变粗带，九色丹气在炉腹中形成稳定的反向旋转环。**Expected**：第八转顺利完成（如第七转）。**Actual**：第八转的丹气逆走**比第七转更顺**——反向气流稳定旋转，炉腹的反向旋转环明显成形；他用约半刻钟完成第八转的前半。*（gap：物理上的"顺"——反向气流的稳定是好事，但越顺意味着越接近完成；他的肉身比脑子先反应这一接近）*
2. **Action**：第八转的反向气流完成稳定旋转那一秒——**承天梁微震第四次**（铁链微晃约二寸）。**Expected**：微震如前三次（约一寸半）。**Actual**：微震**明显加大**——铁链微晃约二寸；承天梁的木质纤维第一次（在物理上）发出可听见的"咯"一声（[[setting-survey]] §5.3 承天梁三百年首次承受倒挂丹炉反向丹气的物理代价）。三万人观礼席中**前几排有人抬头看承天梁**。*（gap：物理上的"加大"——反向气有第二个源头与第八转炉腹反向气流共振的视觉化第一次发生，但观礼席不知道为什么加大；只有师傅与主角在身体上读出这一加大有"两个方向"）*
3. **Action**：主角的呼吸停半秒——他**第一次（在身体上）感觉到反向气有第二个源头**。他的右手按炉沿火门停半秒；他的左手仍持断戒尺停在炉沿上方约半寸。**Expected**：他立刻判断为反丹道副作用或承天梁的物理代价。**Actual**：他**没有判断**——他知道这不是反丹道副作用（副作用的物理形态是七窍流银砂 / [[setting-survey]] §3.3 / 与共振的形态不同）；他知道这不是承天梁的物理代价（梁的物理代价是单向震动 / 这道气是反向旋转的）；他知道这是**反向气有第二个源头**——他的肉身比脑子先承担这件事。*（gap：他的判断停在"是什么"的前一秒——他的肉身已经知道但脑子还没命名；这是 [[characters/protagonist]] §4 主角的认知滞后于身体反应的物理化）*
4. **Action**：师傅的视线从祭炉炉腹微微下移约半寸——他**读出反向气有第二个源头**。**Expected**：师傅做出任何外向反应（说话 / 戒尺落下 / 离开 / 调用宗门律）。**Actual**：师傅**没有任何外向反应**——他的视线下移半寸然后回到祭炉炉腹位置；他的戒尺仍举起未落；他的脸没有任何表情变化。他**让自己读出这件事但不让任何人读出他读出来了**（[[characters/master]] §3 维度 1+2+3 / 他作为知情人的全部姿势）。*（gap：师傅在第八转那一秒第一次（在身体上）读出师妹的反向核已苏醒；他三年来藏师妹于地宫第七层这件事的物理形态首次在仪式空间中暴露给他自己，但他不让暴露被外人读出；他的戒尺仍举起未落的姿势承担这一帧——他不能让戒尺落下，因为戒尺落下意味着仪式被打断、地宫共振被读出、师妹的位置可能被读出）*

6. **Action**：主角把左手持断戒尺转回炉沿上方约半寸的位置——他**强行让自己继续第八转**。他没有让自己进一步命名"地宫深处那个人是谁"；他用第八转的物理操作姿势继续覆盖这一认出。**Expected**：他离开炉沿、走向地宫入口、当场放弃仪式。**Actual**：他**继续操作**——他完成第八转的后半；反向气流从粗带变粗环（[[setting-survey]] §3.2 二品反炼级第八转的完成形态）；九色丹气在炉腹中形成完整的反向旋转环。第八转完成。*（gap：他选择把"认出地宫深处有人持反向气脉活着"这件事推到第九转之后处理——他用反炼的物理仪式覆盖内在层的最大动摇；他知道一旦让自己命名"那个人是谁"，他三年来反丹道动机的承重墙会立刻塌方；他用第八转的完成换取自己作为反丹道执行者的身份还有一转的稳定）*

**转折点（Turning Point）**：第 5 拍——**主角左手持断戒尺尖端转向地宫深处方向悬停约半秒**。这一帧是承认轴对立级峰值的物理引爆的前半（后半在 Scene 3.3.3）：主角第一次（在身体上）认出"地宫深处有人持反向气脉活着"——他还没命名是师妹，但他的肉身已经知道；这一认出与第八转的物理完成同帧合上但不合一（赢得反炼与认出师妹活着两个动作在 Seq 3.3 内部被严格撕开）。**这是 [[characters/protagonist-arc]] 锁定的"第八转地宫共振主角认出师妹活着"的物理化前奏**——但 Seq 3.3 内部不让主角命名是师妹，这一命名留到 Seq 3.4 入井时兑现。

### Scene 3.4.3 ——断戒尺开石壁 + 入逆经井 + 看见石棺缝里玄漪睁眼

- **位置**：Sequence 3.4 内第 3 场（3.4.c 入井 / 师妹只说一句 / 三年自我叙事釜底抽薪）
- **字数预算**：~2.5K

**进入价值**：承认轴对立级峰值的内在崩塌的扩散（Scene 3.4.2 末状态——反丹副作用扩散到帝京）；情绪轴：内核空洞的扩散。**戒尺**：断戒尺第三次共振完成（即将作为钥匙开石壁）；**印记**：暗赤一格深化（约 115 道）；**天道债**：约 115 道。

**目标（Objective）**：开第七层入口石壁，入逆经井，确认地宫深处那个反向核——他要让自己（在身体上）面对那个反向核是谁；他**仍未让自己命名是师妹**，但他的肉身已经在 Scene 3.3.2 读出"反向气有人的形状"、他三年来对师妹"我用正方丹杀了她"的核记忆开始（在身体上）松动。

**冲突（Conflict）**：
- **超个人层**：第七层入口石壁的物理状态——石壁需以**断戒尺作为钥匙**插入石壁中央的反向符文位置才能开启（[[key-image]] §2.1 Item 8 setup payoff / [[setting-survey]] §5.5 反丹道符文的物理钥匙机制 = 断戒尺三年存留之谜的物理回收：师傅三年前断戒尺一击的物理意义在此处兑现，断戒尺不是惩罚物，是入第七层的钥匙）；石壁开启后是一段约三十丈的下降石阶通向第七层中央——**逆经井**（[[setting-survey]] §5.6 逆经井的物理形态：井深约六十丈、井底是汉白玉石棺、井壁有沈砚手札残页贴在井壁上）。
- **个人层（主导）**：井底**汉白玉石棺**——石棺长约六尺、宽约二尺半、高约二尺；石棺盖与石棺身之间有约一寸宽的缝隙（不是完全封闭的石棺）；主角直起身向井底看时——他看见**石棺缝里有一双眼睛睁着**；眼睛**是师妹玄漪的眼睛**（[[characters/sister]] §1 玄漪的物理识别——主角看了她十几年，他认得她的眼睛）；玄漪**开口说一句话**：**"师兄，原来你回来了。"**（[[spine]] §4 / [[sister-arc]] §3 锁定 / **仅此一句**，用户已拒绝 antagonism-test §3 Prescription #5，师妹后两句话在 Seq 4.2 + Climax 拍 4-5 出现）。
- **内在层**：主角看见玄漪睁眼那一秒——他**三年自我叙事被釜底抽薪**：（1）"我用正方丹杀了她"被推翻——玄漪活着；（2）"师傅废我是为惩罚"被推翻——师傅废他不是为惩罚是为藏她（地宫第七层逆经井是宗门祠堂最深的物理藏匿位置）；（3）"师兄，原来你回来了"是保护主角不让他为三年的离开愧疚的姿势——但**不构成承认的二审通道**（[[controlling-idea]] §7 违规列表 #5 严守——师妹不构成"师傅认我"的二审通道）；玄漪只说一句，她没有说"师傅其实一直认你对"或类似台词（[[controlling-idea]] §7 违规 #4 严守）。

**Beats**（7 beats）：

1. **Action**：主角抬起断戒尺——他**把断戒尺尖端插入第七层入口石壁中央的反向符文位置**（[[key-image]] §2.1 Item 8 兑现 / 戒尺第五次入画 / 累积重量 +2）。**Expected**：石壁立刻开启。**Actual**：石壁**慢慢开启**——断戒尺插入符文位置后，石壁中央的反向符文开始（在物理上）反向旋转；旋转约半刻钟后，石壁从中央向两侧打开约三尺宽的入口；入口内是一段下降石阶通向第七层中央的逆经井位置。*（gap：断戒尺三年存留之谜的物理回收完成——师傅三年前断戒尺一击的物理意义在此处兑现：断戒尺不是惩罚物，是入第七层的钥匙；这是 [[key-image]] §2.1 Item 8 setup payoff 的物理化）*
2. **Action**：主角穿过石壁入口走下石阶——石阶约三十丈、向下倾斜约三十度；他走到石阶底部进入第七层中央——他看见**逆经井**（井深约六十丈、井壁内壁有反丹道符文 + 沈砚手札残页贴在井壁上）。**Expected**：井底空旷（按他三年来的核记忆师妹已死）。**Actual**：井底**不空旷**——井底有一具**汉白玉石棺**（[[setting-survey]] §5.6 锁定）；石棺位置在井底正中央；石棺盖与石棺身之间有约一寸宽的缝隙。*（gap：他第一次（在物理上）看见井底有石棺——他还没让自己读出石棺意味着什么，他的肉身在 Scene 3.3.2 已经读出"反向气有人的形状"，但他的脑子在物理上看见石棺那一秒才开始追上肉身）*
3. **Action**：主角从井口沿绳索下降到井底——他下降约六十丈到达井底地面；他走向汉白玉石棺位置；他**蹲下**——他的视线与石棺缝隙的高度齐平。**Expected**：石棺缝隙里是空的（按他三年来的核记忆师妹已死 / 石棺缝隙里应是空的）。**Actual**：石棺缝隙里**有一双眼睛**——他**先看见眼睛的形状**（不是先看见瞳孔颜色，是先看见眼睛的轮廓形状）；眼睛是**他认识的形状**——他看了那个眼睛十几年（[[characters/sister]] §1 玄漪的物理识别）。他**第四次愣神**（Seq 3.1.2 第一次、Seq 3.3.1 第二次、Seq 3.4.1 第三次、本拍第四次）。*（gap：他第一次（在物理上）认出地宫深处那个反向核的身份是师妹玄漪——他三年来"我用正方丹杀了她"的核记忆在物理上证伪；这一帧的物理形态是 Act 3 末 turning point 的物理兑现的前半）*

### Scene 3.4.4 ——主角爬回井口 + Act 3 末 TP 收口 / False Ending 完整形态

- **位置**：Sequence 3.4 内第 4 场（3.4.d Seq 3.4 末 / Act 3 末 TP / False Ending 完整形态）
- **字数预算**：~1.5-2K

**进入价值**：承认轴对立级末（Scene 3.4.3 末状态——三年自我叙事被釜底抽薪 / False Ending 物理形态出现）；情绪轴：内核空洞的釜底抽薪。**戒尺**：断戒尺仍在主角手中（已开石壁完成）；**印记**：暗赤一格深化（约 125 道）；**天道债**：约 125 道。

**目标（Objective）**：爬回井口、回到地宫第七层入口外、关闭石壁（让石壁恢复原状以保护师妹的位置）、离开地宫——他要在 Act 3 末完成"False Ending 看起来完整"的物理收口；他不预期回程会有任何信息。

**冲突（Conflict）**：
- **超个人层**：地宫第七层的物理状态——按宗门律地宫第七层是宗门祠堂最深的物理藏匿位置，按反丹道符文体系师妹的石棺位置受反向符文保护（[[setting-survey]] §5.5 + §5.6）；主角离开第七层时需关闭石壁以维持藏匿状态。
- **个人层**：师傅藏师妹于第七层这件事——主角爬回井口的过程中他**第一次（在身体上）让自己读出师傅三年来的全部姿势**：师傅三年前断戒尺一击不是惩罚是钥匙（断戒尺第三次共振是对师傅笔迹 / 第四次入画是敲炉沿 / 第五次入画是开石壁）；师傅大典前夜默许或命令司徒明璋截下联议密折是为不让反丹道合法性被体制层公开认证（这一姿势的物理含义主角还没读出，但他知道师傅在做的不是单纯否认）；师傅大典上"未发一言转身离场"是不公开承认反丹道但也不公开否认；师傅持戒尺举起未落是不打断反炼也不公开承认。**四件事合上**——师傅三年来的物理姿势全部指向同一个方向：**他在保护一件事但他不能公开说出他在保护什么**。
- **内在层**：主角爬回井口——他知道他无法立刻处理师傅四件事合上意味着什么，但他**第一次（在身体上）让自己读出"师徒可以和解"这一伪希望**——师妹活着 + 公共胜利 + 师傅亲笔批语的存在 + 师傅未发一言转身离场（不是否认）= 看起来"师徒可以和解"的物理形态首次出现；这是 [[false-ending]] McKee Ch. 9 锁定的 False Ending 物理形态——读者与主角短暂共同读出"师徒和解可能成立"的物理元素全部到位。**但他还不能让自己真正信这一可能**——他用爬回井口的物理动作承担这一伪希望的悬空状态。

**Beats**（5 beats）：

1. **Action**：主角抓住绳索向上爬——约六十丈的物理距离；他的右肩酸痛、右脚跛感持续；他爬到约三十丈的位置时**停一秒**——他**回头**最后一次看石棺位置；石棺缝里玄漪的眼睛仍睁着但已不在他的视线范围内（井底太深 + 角度问题）；他**继续向上爬**。**Expected**：他做出承诺或说告别。**Actual**：他**没有任何承诺或告别**——他用沉默的爬升姿势承担师妹活着这件事的全部物理重量。*（gap：他选择不让自己回头说话——这是 [[characters/protagonist-arc]] §3 锁定的"主角对师妹的克制比对师傅的克制更深"的物理化）*
2. **Action**：主角爬到井口——他从井口翻身到地宫第七层中央地面；他**走向第七层入口石壁**位置；石壁仍是开启状态（约三尺宽的入口）。**Expected**：他立刻关闭石壁。**Actual**：他**先看石壁内侧**——他第一次看见石壁内侧的物理状态：石壁内侧有**师傅亲笔的反丹道符文标注**（[[characters/master]] §1 师傅笔迹的物理识别 / [[setting-survey]] §5.5 反丹道符文的物理标注机制——师傅三年来在地宫第七层入口石壁内侧用反丹道符文标注开启与关闭的物理步骤）；这是师傅三年来**亲自维护师妹藏匿位置**的物理证据。*（gap：他第一次（在物理上）确认师傅三年来不是"什么都没说就走了"——他在做的是亲自维护师妹的藏匿位置；这与师傅"未发一言转身离场"的姿势构成内在层最大对位的进一步深化）*
3. **Action**：主角看见师傅亲笔的反丹道符文标注——他**第五次愣神**（Seq 3.1.2 + Seq 3.3.1 + Seq 3.4.1 + Seq 3.4.3 + 本拍）。他的右手按石壁内侧师傅亲笔标注的位置——他的左袖下断戒尺**共振第四次**（最强一次 / 物理震动约四寸 / 持续约两秒）。**Expected**：他读懂师傅的标注内容。**Actual**：他**没有完全读懂**——他读出标注的物理步骤但他**没有让自己读出标注意图**（标注意图层留白以让 Act 4 Seq 4.1 + 4.2 兑现）；他知道师傅三年来在做的事比他三年来反丹道的方向更复杂，但他还没让自己命名是什么。他用断戒尺第四次共振承担这一帧的全部物理重量。*（gap：师傅亲笔标注是 Act 3 末 turning point 的最后一层物理化——师傅三年来的物理姿势的全部碎片在主角面前合上但合上的意义层留到 Act 4 兑现；这是 [[false-ending]] McKee Ch. 9 锁定的 False Ending 物理形态——主角短暂面对"师徒可以和解"的可能但 Act 4 第一秒将翻面）*
4. **Action**：主角**按反丹道符文步骤关闭石壁**——他用断戒尺尖端**反向插入**石壁内侧的反向符文位置；石壁中央的符文开始（在物理上）正向旋转（与开启时反向）；旋转约半刻钟后，石壁从两侧向中央关闭；石壁完全闭合后表面恢复原始的反向相克符文状态——**没有任何被开启过的物理痕迹**。**Expected**：石壁关闭后留有痕迹（不可能完全恢复）。**Actual**：石壁**完全恢复**——没有任何痕迹，符文位置没有任何动过的迹象；师妹的藏匿位置的物理保护回到完整状态。*（gap：他选择关闭石壁、保持师妹藏匿位置不被外人读出——这是 [[characters/protagonist]] §3 维度 1 "对师傅的忠诚 vs 对反丹道的承担"的物理化的最大一帧；他作为反丹道执行者本应让师妹活着这件事公开（这是他三年来反丹道全程最大的物理"证据"），但他作为师傅弟子选择维持师傅三年来藏师妹的物理姿势——他用关闭石壁承担两种身份的最大物理对位）*
