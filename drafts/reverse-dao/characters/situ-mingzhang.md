# 司徒明璋 ——同门 / 同侧对照（Same-Axis Foil）

> 上游契约：[[premise-card]] §6 §7 / [[controlling-idea]] §7 违规列表 #9（替代承认者禁令）/ [[spine]] §4 C7 §4 C8 / [[act-design]] Seq 3.1 Seq 3.3 / [[key-image]] §2.1 戒尺出现位置表 / [[characters/master]] §1 §6 / [[characters/protagonist]] §1 §8.4 / [[characters/sister]] §8.4 / [[characters/junior-ghost]] §8.4
> 角色函数：**「如果当年沈砺没被废、没反、安稳走正方路线」的镜像**——他是一个没走那条路的沈砺。师傅亲传二弟子；当年废徒大典上**他执的戒尺**（师傅下令、他动手）；现任宗门联盟首座。**他不是嫉妒成性的二师兄，他是被师傅"正方"信仰拯救过又锁死的人**——立场比师傅更僵，因为师傅有屠村记忆作为信仰的真实成本，而他的"正方"信仰是**借来的**。
> 主控思想 value progression 对位：**他承担"矛盾级（被忽视）"——他自己也是被师傅忽视的人**（他选了正方=不需要费力=不被关注）；他在 Seq 3.3 大典反炼第八转**低头退半步**那一帧承担承认轴意象系统的物理交接——戒尺**所有权**从他向主角的物理交接（他放手让主角接管这把戒尺的承担）。

## 0. 一段速描

48 岁，丹道宗门联盟现任首座（任期 5 年，师傅退到首座之前他执宗主令）；师傅亲传二弟子，按辈分是主角"二师兄"。**当年废徒大典上他执的戒尺**——师傅下令、他动手；他这把戒尺断了主角丹田后被主角偷拾一截养在炉边三年（[[key-image]] §1.3 锁定）。**他这一辈子的合法性来自"师傅的正方是对的"**——师傅一辈子押的赌注（"反丹必反噬"+ 三百口避险费）他完全继承，但他没有师傅 22 岁挖三个月尸首的雪夜，他没有右膝冻坏的旧伤，他没有屠村现场的肉身记忆——他的"正方"信仰是**借来的**：他借的是师傅一辈子的代价、师傅一辈子的痛、师傅给他的位置——他只付出了一件事的代价，那就是**当年那一夜亲手敲断师弟丹田**。这一件事是他至今唯一的"真"，但他不能让自己看见它是真。**他是被师傅信仰锁死的人**——立场比师傅更僵，不是因为他更虔诚，是因为他**更没退路**。

## 1. Characterization（人物塑造 / 表面层）

| 职业 / 能力 | 丹道宗门联盟现任首座（任期 5 年——师傅 67 岁那年退到联盟首席之前他执宗主令）；**二品丹师下乘**（师傅是二品丹师天花板——这是司徒明璋一辈子无法超越的天花板，他自己也知道）；元婴初期。他这辈子炼成的二品丹只有一炉：43 岁的"乾元持守丹"（联盟续命丹）——这一炉的方子完全是师傅"乾元守正丹"的细微变奏，连他自己都知道这只是"师傅炉子的临摹"。**他炼丹的天赋远不及师傅，更远不及主角**——他这辈子从未独立创立过任何丹方；他的合法性不是来自能力，是来自师傅给他的位置 |
| 教育 / 谈吐 | 受过完整宗门正统教育（清辉宗 12-30 岁 + 玄琢真人门下 30-48 岁）；说话工整、字斟句酌；他说话的特征是**从不抢话也从不打断**——但他不打断的方式与师傅不同：师傅不打断时眼睛在算（评估你说的话与他一辈子赌注的兼容性），司徒明璋不打断时眼睛是空的（他在等你说完，然后他用宗门律的语言复述一次你的意思）。**他说话像念条文**——这不是冷漠，是他被宗法语言塑造了 18 年；他甚至在和师傅私下说话时也不知道如何不引用宗门律 |
| 衣着 / 取舍 | 一袭乾元紫道袍（联盟首座规制，师傅退位后传给他的同款）；冠"九转莲冠"（与师傅同款——这是宗主令象征）；腰间挂"乾元印"（联盟首座令牌，师傅亲手交给他）和**一柄完整的三百年宗门戒尺**——**这是关键**：宗门正典戒尺有两柄，师傅那一柄三年前敲断在主角丹田处（[[key-image]] §2.1 Item 0），司徒明璋持的是另一柄完整戒尺；**他这把戒尺是宗门正典戒尺的"备份"**——读者在循环三大典反炼那一帧第一次同时看见两把戒尺（师傅持完整戒尺举起未落 / 主角持断戒尺敲炉沿——但司徒明璋作为主坛持典者站在师傅身后，他没拿戒尺；他这场戏里把戒尺交在了师傅手上）|

### 1.2 别人眼里的他（多角度）

- **师傅（玄琢真人）眼里**：他是一个**可以放心把宗门交给的人**——因为他不会想去反，因为他没有反的天赋，因为他没有反的本钱。师傅 60 岁收他入门时心里清楚：这个徒弟不会成为下一个沈砚。**这是师傅对司徒明璋最大的认可，也是最大的低估**——师傅从未把司徒明璋当作一个能继承"立宗"重量的人，他把他当作"守宗"的人。司徒明璋一辈子从未察觉师傅这一层判定，但他每七日入井换药这件事师傅从不告诉他——这是师傅对他最深的不信任的物理形态。
- **主角（沈砺）眼里**：他先是"二师兄"（被废前那 12 年——主角 14-23 岁、司徒明璋 36-48 岁；两人在宗门的辈分关系），后是"那个亲手敲断我丹田的人"（被废后三年——主角 23-26 岁），再是"那个一辈子合法性建立在师傅废我那一夜的人"（循环三 Seq 3.1 同门反目场之后）。**主角对他的恨不是个人的恨**——是结构性的：他在主角眼里**就是主角自己当年没走那条路的镜像**——一个安稳走正方路线的沈砺。主角看他的时候在某种程度上看见的是**反例的自己**——这是师傅看徒弟时看见沈砚的鬼魂的对位（师傅看徒弟看见已死的师弟；主角看二师兄看见未活的自己）。
- **师妹（玄漪）眼里**：他这辈子都不知道师妹还活着——他以为她 23 岁那年死于沈砺手下。在他对师傅的宗法依赖里，"小师叔玄漪"是一个被供奉的过往（[[characters/sister]] §1.2 锁定）。**Seq 4.2 师妹现身那一帧是他一辈子合法性的第二次裂缝**——他第一次明白师傅一辈子立的"正"对他完全保留了一个秘密（藏女儿三年这件事），他对师傅的宗法依赖从这一帧开始崩。但他在 Seq 3.3 大典反炼第八转那一帧已经预演过这种崩——他第一次看见师傅没让戒尺落下时，合法性根开始动摇（[[characters/sister]] §8.4 锁定）。
- **沈砚（已故，通过师傅闪回 + 宗门档案）**：他从未见过沈砚。他对沈砚的认识来自宗门档案的简略记载（"反丹屠村案首犯，自焚于丹炉"）+ 师傅 18 年师徒任期里偶尔提及的"我那位走偏的师弟"。**他对沈砚的情感是完全的隔阂**——他作为现任宗主的合法性建立在"反丹必反噬"上，沈砚是这条信念的物质本金；他不能让自己对沈砚有任何同情，否则他对师傅的宗法依赖就裂（[[characters/junior-ghost]] §1.2 / §8.4 锁定）。
- **联盟其他六大宗门长老眼里**：他是"师傅的位置"——七大宗门长老对他的恭敬是对师傅的恭敬的延续；他自己一辈子都知道这一点。**他在联盟里没有任何"自己的位置"**——他的位置是师傅给他的位置；任何让师傅的位置动摇的事都让他的位置动摇。这是他比师傅更僵的本质：师傅有一辈子立的学说、有屠村避险费的本金、有自己的位置；司徒明璋只有"师傅给的位置"——一旦师傅的判断被怀疑，他失去全部。
- **沉鸠（鬼丹窟瞎眼老丹师）眼里**：他从未见过司徒明璋；他对司徒明璋的判断来自帝京下水道流传的传闻——"那位现任宗主从不下手处理反丹案，他都让七大宗门派代表去做。" 沉鸠对此评价：**"他怕的不是反丹道，他怕的是自己的手抖。"**——这是沉鸠对司徒明璋最准确的一句话（沈砚生前说"我怕的是有一天我做对了，但已经太迟"是沈砚对自己的判断；沉鸠这一句是对司徒明璋的判断）。这句话主角在循环二中段从沉鸠那里听到——但他当时还不懂；Seq 3.1 同门反目场他第一次完全读懂。

## 2. True Character（真实性格 / 压力下显形的选择）

> **在压力到达极限的那一刻——Seq 3.3 大典反炼第九转丹成、宗门联盟当众承认"反方亦道"、师傅持戒尺举起未落转身离场那一帧——司徒明璋会低头退后半步。**

但他真实性格的核**不在于"低头退半步"是觉醒**——这是太多悲剧二师兄写过的烂尾。他真实性格的核在于：**他低头不是因为他认了反丹道、不是因为他对主角说"师弟你对了"、不是因为他对师傅产生了怀疑**——他低头是因为他**在那一帧第一次（在身体上）面对自己作为"借来信仰的人"的事实**：师傅没让戒尺落下=师傅自己也在动摇=他这辈子借的赌注的本人都在动摇=他借的全部都开始失重。他低头不是认主角对，**是他借的东西不再撑得住他**。但他不能公开承认（联盟律），所以他低头同时退后半步——**这是他作为现任宗主对反丹道的物理承认，但是不公开、不解释、不让任何人（包括自己）说出口的承认**。

> **他真实性格的最终命名**：**「借了别人一辈子的信仰，借到他自己的腿都软了的人——但他至死不肯承认那是借的」。**

### 为什么这是他的真实性格而不是别的

- **不是"嫉妒成性的二师兄"**——网文俗套；他从未嫉妒过主角的天赋（他知道自己一辈子都达不到那个高度，他和这件事和解了 18 年）。他对主角的敌意不是嫉妒，是**结构性的**：主角的存在让他借的东西失重。
- **不是"师傅的傀儡"**——他不是没有自己的判断；他 30 年来用宗门律的语言重新组织了自己一辈子的判断，他真心相信"师门律法分明"。但他相信这件事的基础是借来的（师傅一辈子的代价），不是他自己付出的。
- **不是"觉醒了的反派"**——他在 Seq 3.3 第八转那一帧没有觉醒，他只是**第一次（在身体上）感觉到借的东西在塌**。觉醒需要他能命名"我借了一辈子"——他做不到，因为命名等于承认他一辈子的合法性是空的。
- **不是"主角的替代承认者"**（[[controlling-idea]] §7 违规列表 #9 锁定的硬约束严守）——他在 Seq 3.1 同门反目场**绝不会**对主角说"师弟你对了"或类似台词；任何形态的"师兄认了你"都直接撕掉 controlling idea 锁定句"承认被永远逐出可被验证的世界"。**他是同侧对照（同行另一条路的人），不是反向认证者**。
- **是"借了别人一辈子的信仰，借到他自己的腿都软了的人——但他至死不肯承认那是借的"**——这一维度在 Seq 3.3 第九转那一帧的"低头退半步"完整兑现：他低头是身体在塌（借的东西撑不住了），他退半步是退出戒尺裁决位（让出他借的位置）；但他**不公开承认、不解释、不说话**。这是同侧对照（same-axis foil）作为反面镜像的同侧对位——他不是镜像（他不是主角的反面），他是**同侧但没走到那一炉的人**。主角走到了那一炉，他没走；主角的胜利不是对他的击败，是对他借的东西的釜底抽薪。

### 维度间的咬合检查（character-forger §8 自检）

- 维度 1（公开持典 vs 私下回避那一夜）和维度 2（持戒尺 vs 无丹）共用"持戒尺"这一物理形态——但分布不同：维度 1 是**伦理层**（他执的戒尺断了师弟的丹田），维度 2 是**身份层**（他持的戒尺是他作为丹师的全部合法性——但他没有独立创立的丹作为支撑）。两个不重复——前者是"我做过什么"，后者是"我是什么"。
- 维度 3（被信仰拯救 vs 被信仰锁死）是他作为同侧对照（same-axis foil）的本体维度——它把维度 1 和维度 2 钉在借来信仰这条主轴上。**去掉维度 3，前两个维度就只是"师兄反派"的常见配置**——维度 3 让他成为一个被信仰锁死但被信仰救过的具体人，这是反陈词滥调的核保险。
- 三个维度在 Seq 3.3 + Climax 拍 1 同帧合上：维度 1（公开持典 vs 私下回避）→ 第八转他看见师傅没让戒尺落下、第九转他低头退半步；维度 2（持戒尺 vs 无丹）→ 他放弃持戒尺姿态但无退路；维度 3（被信仰拯救 vs 被锁死）→ 师傅破宗门律传位、他借的信仰塌方但他还在牢里。**三维度同帧合上**——这是 [[character-revelation]] 的标本案例。

### 维度多样性检查

不全集中在"宗门律 / 现任宗主"轴：维度 3（被信仰拯救 vs 被锁死）是个人伦理学，维度 1 是历史伦理（他做过什么），维度 2 是身份本体（他是什么）——三轴分布。

### 删除测试（自检）

> 如果删除维度 3，维度 1 / 2 还能产生 Crisis 吗？

**不能。** 维度 1 / 2 让司徒明璋成为一个"现任宗主+ 当年执戒尺者+ 没有独立创立丹的丹师"的具体人；但只有维度 3 让他承担**借来信仰的人**这一同侧对照的本体——没有维度 3，他就是一个"师兄反派"的常见配置（嫉妒主角天赋 / 维护宗门体制 / 被打脸觉醒），无法承担反陈词滥调的核保险。**[[characters/master]] §3 维度 2 师傅的"赌一辈子+三百条人命的避险者"需要司徒明璋作为活的对照**——一个借了师傅一辈子赌注的人，一个没有自己的赌注的人。维度 3 是承重墙，不是装饰。

## 4. Object of Desire（欲望对象）

司徒明璋作为 same-axis foil（同侧对照——同行另一条路的人），他的欲望与主角是**同侧但分叉**的关系：主角追"被师傅再认一次"，司徒明璋追"不被师傅看见自己借的全部"。两人都在追**不被看穿**这件事，但姿势相反——主角希望被看见（被父亲式的承认看见），司徒明璋希望不被看见（不被父亲式的判定看见自己借的）。

### 4.1 自觉欲望（Conscious）

> **守住宗门律法的全部完整性，让"师傅当年废徒一案"在宗门正典里成为不可被翻案的判例。**

他对外的所有公开行为（28 年宗主令任期、签发追杀令、主持联盟年会）都是这条欲望的形态。他真心相信宗门律的完整性等于宗门的延续——这一条信念不是错的，但它的根是借来的（师傅一辈子立的赌注），不是他自己的。

### 4.2 潜意识欲望（Unconscious）

> **不让师傅看见自己一辈子借了师傅一辈子的代价，自己却没有付出过任何同等的代价。**

他至今没有意识到这一点；他自己以为他在守宗。但他每一次走过祠堂正殿地砖第七块时不停步、不绕开、不躲避（[[characters/master]] §1 微动作 3 的精确反差）——他**不知道**那块砖下藏着师傅一辈子最大的秘密；他每次踩过去都在物质上证明**他从未付出过和师傅同等的代价**（如果他付出过，他应该知道地宫入口在哪里）。**他每天踩过那块砖二十年都不知道——这件事本身就是他借来信仰的物理形态**：他借的、他用的、他持的全部都建立在**他不知道的代价**上。

### 4.3 自觉与潜意识的不可调和

> 守宗（自觉）的形态是宗门律的完整性；不让师傅看见自己借的全部（潜意识）的形态是**永远不知道地宫第七层的秘密**。两件事不能同时为真——一旦师傅藏女儿的事被司徒明璋知道（如 Seq 4.2 师妹现身那一帧），司徒明璋就明白师傅一辈子立的"正"对他完全保留了一个秘密——他借的赌注不是他以为的赌注；他的潜意识欲望（不被看见自己借的）在被知道的那一秒立刻塌方。**师妹现身那一帧是他潜意识欲望的强制曝光**——他在那一帧第一次（强制地）看见自己借了一辈子的本人对自己保留了 23 年的真相。

### 4.4 这条 gap 与主角对位的同侧分叉形态

- **主角的 gap**：自觉（让师傅认我对）vs 潜意识（被师傅再看一次）——他追的是父亲那一字。
- **师傅的 gap**：自觉（不让反丹成功）vs 潜意识（看见反丹真的成立一次以解放自己）——他追的是替自己一辈子的赌注解套。
- **师妹的 gap**：自觉（不让父亲与师兄相撞）vs 潜意识（让师兄被父亲完整承认）——她追的是父亲那一字以求被永久悬空。
- **司徒明璋的 gap**：自觉（守宗门律法的完整性）vs 潜意识（不让师傅看见自己借的全部）——他追的是**师傅一辈子的赌注里那个不属于他自己的部分**。
- **四人的 gap 在终极反转 Climax 拍 1-7 同帧合上**：主角准备起炉自死换师傅认；师妹自愿入炉换父亲不必白活；师傅自愿入炉做反核衬体；**司徒明璋站着、看着、没动**——他作为现任宗主在这一帧能做的全部是**不动**。**他是四人中唯一在 Climax 没有做选择的人**——这是同侧对照的最深一种形态：他借了一辈子，所以在最大的选择面前他没有任何东西可以付出。**他的"不动"本身就是他借来信仰的最终物质形态**。

### 何以这五项是弹药而不是背景填充

每一项都直接改变下一场戏的力学：第 1 项揭司徒明璋拜师前是金丹瓶颈户——揭他作为现任宗主的合法性根基是恩（不是天赋）；第 2 项揭废徒那夜两人各自记忆中对方手抖的反向真实——同侧对照最深的一刀；第 3 项揭司徒明璋踩过祠堂第七块砖二十年不知道地宫秘密——他借来信仰的物理形态；第 4 项揭司徒明璋父亲临终笔记从未被打开——他选择保持借的状态的本体维度；第 5 项揭那截断戒尺另一截在主角手里、他从未问过——戒尺所有权的物理交接。**任一项被砍，下一场戏失去燃料**。

## 6. Biography Spine（生平脊椎，分三段——按用户指示三段：师弟死后被师傅收为二弟子 / 废徒大典执戒尺那一夜 / 现任宗主到大典反炼第八转低头）

### 6.1 师弟死后被师傅收为二弟子（0 岁～30 岁）

1. **0–8 岁**，中州正派世家司徒家出生；父亲是清辉宗旁支长老（清辉宗是师傅 22 岁前修业的小宗——这是司徒明璋家族与师傅师承的旁支关联）。父亲在他 8 岁那年给他改名"明璋"（之前叫"司徒承业"——意承父业）。
2. **12–18 岁**，与父亲同入清辉宗旁支修业；炼气、筑基。这一时期他以"正派世家子弟"身份成长——**他从小就被告诉"反丹道是错的"**，这是他作为正派世家子弟的家学基础。
3. **18–24 岁**，过金丹境（24 岁）；进入金丹圆满。
4. **24–30 岁**，金丹瓶颈户 6 年——他自知天赋天花板的问题。这一时期他对自己作为丹师的尊严产生了深刻的怀疑；父亲晚年对他这件事十分焦虑。
5. **30 岁那年**，正式拜师玄琢真人（师傅 60 岁那年）——师傅当时已是丹道宗门联盟立宗者（55 岁立宗）+ 联盟首座（任期 5 年）。师傅收他入门是**师傅在自己一辈子学说立宗后接收的第一个"已成型的正派人物"**——师傅看中他的不是天赋（师傅看一眼就知道他天赋天花板），是他**不会成为下一个沈砚**：他没有反的天赋，没有反的本钱，没有反的内心冲动——他是一个可以"守宗"的人。

### 6.2 废徒大典执戒尺那一夜（30 岁～25 岁拜师 + 23 年前那一夜）

6. **30–43 岁**，师傅用 13 年让他突破元婴初期（这是他这辈子最大的恩）。这一时期他对师傅的宗法依赖完全形成——他真心相信"反丹必反噬"是宗门的真理。
7. **43 岁那年**，炼成"乾元持守丹"（联盟续命丹）——他这辈子炼成的唯一一炉二品丹；方子是师傅"乾元守正丹"的细微变奏。
8. **25 岁拜师 + 23 年前那一夜**（即主角 23 岁被废那夜——司徒明璋当时 45 岁）：师傅当众判主角"走火入魔"，下令废徒。**司徒明璋作为大弟子（按宗门律应由大弟子执行废徒戒律——但师傅彼时尚未正式收沈砺为大弟子，名分上司徒明璋仍是大弟子，主角是首徒）执戒尺**——师傅亲手把宗门正典戒尺递给他，他动手敲断主角丹田。戒尺先点胸口左侧锁骨下三寸（留烫疤），再下击丹田位（戒尺断为两截）。**他那一刻手稳——他这辈子从未让自己回想自己是否真的稳**。断戒尺的两截留在大典广场，按宗门律应由现任宗主收回归档；主角偷偷拾回了一截，司徒明璋从未问过另一截去哪了。
9. **23 年前那夜后第二天**——师傅当众宣告"司徒明璋继任首徒"；司徒明璋的现任宗主合法性根基从这一夜立起。

### 6.3 现任宗主到大典反炼第八转低头（45 岁～48 岁，故事开始时及之后）

10. **45 岁那年**（即主角被废那夜+ 第二天），司徒明璋继任首徒——这一年是他作为现任宗主合法性根基的奠基年。
11. **5 年前**（司徒明璋 43 岁、师傅 67 岁），师傅退到联盟首席，把宗主令交给司徒明璋——他正式成为丹道宗门联盟现任首座。
12. **5 年来**，他作为现任宗主签发追杀令（包括循环一末"魔路追杀名录"第一人——主角）、主持联盟年会、维持宗门律的全部公开形态。**他从未独立创立过任何丹方**——他炼丹的天赋远不及师傅，更远不及主角。他的合法性来自师傅给他的位置。
13. **故事开始第六个月**（[[premise-card]] §3 激励事件——破观第一炉，循环一开端）：主角第一次反丹成功——司徒明璋作为现任宗主签发"魔路余孽追杀令"。他这一刻还不知道自己签发的是什么——他签的是宗门律的标准动作，他不知道这一动作会让自己 9 个月后在 Seq 3.1 同门反目场面对自己的合法性裂缝。
14. **故事开始第 25 个月**（循环三初，Seq 3.1 同门反目场——C7 / 独立 3K）：主角与司徒明璋在宗门地宫入口的拦截戏。主角一句话刺穿司徒明璋的合法性："师兄，你这一辈子的位置，是师傅废我那一夜替我空出来的。"司徒明璋回一句让主角愣神的话："师傅那夜下令的时候，手是抖的。"——**两人各自承认了对方那一夜的脆弱但都没承认自己的脆弱**（[[spine]] §13 #1 锁定）。送主角离开时主角问："师兄，你这一辈子有没有问过自己——为什么师傅每次走到这里都绕一下？"司徒明璋愣神。
15. **故事开始第 26 个月**（Seq 3.3 大典反炼第七至第九转，金场 1）：主角公开反炼师傅一生立宗之丹"九转还魂"。司徒明璋作为联盟首座站在主坛持典位。**第八转那一帧**——他第一次（在物质上）看见主角持着的断戒尺**就是 23 年前他亲手敲断的那把戒尺的另一截**；同时他看见师傅持完整戒尺举起未落。**他借了一辈子的赌注的本人在动摇**。**第九转丹成那一帧**——他低头退后半步。**这是他作为现任宗主对反丹道的物理承认，但是不公开、不解释、不让任何人（包括自己）说出口的承认**。
16. **故事开始第 27 个月**（终极反转 Seq 4.2 师妹现身那一帧 + Climax 拍 1）：师妹由师傅扶出现身——司徒明璋第一次（在他眼前的物质形态上）看见师傅一辈子立的"正"对他完全保留了一个秘密（藏女儿三年）；他借了 18 年的信仰原来不是他以为的全部。Climax 拍 1 师傅解下宗主令牌放在主角脚边——师傅这一动作是把宗主令牌从司徒明璋（现任持有者）那里**未通过任何仪式地直接传给主角**。**司徒明璋在这一帧没有动作**——他作为现任宗主在最大压力下能做的全部是**站着，看着师傅入炉**。
17. **Resolution（Climax 拍 7 之后）**：主角举断戒尺转身离开祭天台。**司徒明璋仍站在主坛持典位**——他没有动。**他这辈子借的全部都塌了，但他没有任何东西支撑他作为不持戒尺的人活下去**。读者最后看见他的物理动作是**他抬起腰间的完整戒尺——但他没有举起来，他只是握着**。这是他与主角举断戒尺转身离开的最深一种对位：主角举断戒尺未落（一种"未完成的姿势"），司徒明璋握完整戒尺不举（一种"无可为而为之"的姿势）。**两人都没有让戒尺落下**——但两人没让戒尺落下的理由完全相反：主角是被结构性夺走（师傅先一步入炉），司徒明璋是结构性失去任何举起的合法性（他借的全部都塌了）。**他余生的形态留白**——故事不交代他活了多久、做了什么、去了哪里。

## 7. Pressure Points（压力点 / 压到极限露出真实性格的具体场景）

至少 3 个具体场景。每场写明：表面层翻到深层的物理动作 + 翻面后露出的真实面。

### 压力点 1｜23 年前废徒大典执戒尺那一夜（闪回切片，故事开始前——位置由 scene-architect 决定，建议 Seq 3.1 同门反目场之前的某一帧瞬时闪回）

- **表面层**：他是大弟子（按宗门律应由大弟子执行废徒戒律）；师傅亲手把宗门正典戒尺递给他，他动手——**这是宗门律的标准执行**。
- **压力源**：主角是师傅 14 岁从城外破庙拣来养大的孩子——12 年的师徒情；司徒明璋当时 45 岁，知道主角是师傅最看重的徒弟。**他动手敲断的不只是师弟的丹田，是师傅 12 年的心血**。
- **翻面动作**：他执戒尺，先点主角胸口左侧锁骨下三寸（留烫疤），再下击丹田位（戒尺断为两截）。**他那一刻手稳——他这辈子从未让自己回想自己是否真的稳**。
- **露出的真实面**：他不是"为正方殉道的执法者"。他是**借了师傅一辈子的赌注，在那一夜亲手把师傅 12 年心血敲碎**的人——他用动手这件事换得师傅当众宣告"司徒明璋继任首徒"的位置。**他作为现任宗主合法性的真正物质根**就是这一夜——他不是被任命的，是**用动手买的**。这一笔他自己从未让自己看清——他用宗门律的语言"分类"为"执行师门律"，但每次想起来的瞬间他都快速避开。
- 这一帧通过闪回出现一次（约 200 字）；时机由 scene-architect 决定，建议放在 Seq 3.1 同门反目场之前的瞬时闪回（主角持断戒尺站在地宫入口前那一秒——司徒明璋看见断尺，闪回他自己 23 年前敲断戒尺的那个瞬间）。

### 压力点 2｜Seq 3.1 同门反目场地宫前对话（C7 / 独立 3K 字）

- **表面层**：他是丹道宗门联盟现任首座，主角是被联盟"魔路追杀令"列入第一人的"魔路余孽"——按宗门律他应当当场拿下主角。这是 Seq 3.1 表面上的设定。
- **压力源**：主角持断戒尺站在地宫入口前——这是司徒明璋这辈子第一次（物理上）看见那截断尺的另一段。23 年前他执的那一柄戒尺断成两截后——其中一截留在了主角手里，三年来一直被养着。
- **翻面动作**：他没有当场拿下主角（违反宗门律——他作为现任宗主应当出手）。他只是站在地宫入口对面，对主角说话。主角对他说："师兄，你这一辈子的位置，是师傅废我那一夜替我空出来的。"——他停了三秒，然后回："师傅那夜下令的时候，手是抖的。"——这一句让主角愣神。**他不否认自己执了戒尺**（这件事无法否认），但他用师傅那夜手抖这件事**第一次（对外人）承认了那一夜不是简单的"师门律法分明"**。
- **露出的真实面**：维度 1 的物理引爆——他**公开持典 vs 私下从未让自己回看那一夜**之间撕开一道缝。但他**没有承认主角对了**（[[controlling-idea]] §7 违规列表 #9 严守——他不是替代承认者）；他承认的是**师傅那夜手抖**——这是他借来信仰的本人那一夜的脆弱。**他用承认师傅的脆弱来回避承认自己的脆弱**——这是同侧对照的最深一种姿势。
- 主角的反应（关系函数的一帧）：主角愣神，然后说："师兄，你那夜手是抖的。"——**反向引爆**：司徒明璋记忆中"我那夜手是稳的"，但主角说他手是抖的——主角说的是**主角作为承担者**的记忆（主角的丹田被敲那一刻他听到的是抖的戒尺）。**两人各自承认了一个对方手抖的事实，但都没承认自己手抖**——这是同侧对照的最深一种相互不承认。
- 司徒明璋送主角离开时主角问："师兄，你这一辈子有没有问过自己——为什么师傅每次走到这里都绕一下？"——司徒明璋愣神。**这是 §5 弹药 #3 的引爆**：他第一次（在身体上）感觉到自己有些东西从未付出过；他踩过那块砖二十年都不知道这件事本身就是他借来信仰的物理形态。**主角不告诉他砖下是什么**——他让司徒明璋自己带着这个问题走完后面的戏。

### 压力点 3｜Seq 3.3 大典反炼第八至第九转，司徒明璋低头退半步（C 折叠后半 / 金场 1 内 0.3K）

- **表面层**：他是丹道宗门联盟现任首座，主角是当众反炼师傅一生立宗之丹"九转还魂"的"魔路余孽"。按宗门律他作为联盟首座应当宣告"魔路余孽以反方夺正方之炉"——这是宗门律的标准动作。
- **压力源**：第八转那一帧——他第一次（在物质上）看见主角持着的断戒尺**就是 23 年前他亲手敲断的那把戒尺的另一截**；同时他看见师傅持完整戒尺举起未落（[[characters/master]] §7 压力点 2 锁定）。**他借了一辈子的赌注的本人在动摇**——师傅没让戒尺落下=师傅自己也在动摇=他借的全部都开始失重。
- **翻面动作**：第九转丹成那一帧——他**低头退后半步**。腰间的完整戒尺没有举起；他放弃了持典姿态。**他没有宣告**——按宗门律他应当宣告但他没有；这一沉默本身就是宗门律在他身上的塌方。
- **露出的真实面**：维度 2（持戒尺 vs 无丹）+ 维度 3（被信仰拯救 vs 被锁死）+ 维度 1（公开持典 vs 私下回避）三维度同帧合上——他作为现任宗主对反丹道的物理承认，但是**不公开、不解释、不让任何人（包括自己）说出口的承认**。**他低头不是认主角对，是他借的东西不再撑得住他**。但他**没有公开承认**（联盟律），所以他低头同时退后半步——**这是他作为现任宗主对反丹道的物理承认，但是不公开、不解释、不让任何人（包括自己）说出口的承认**。
- **戒尺所有权的物理交接**（[[key-image]] §1 锁定）：第九转那一帧——司徒明璋作为现任宗主放弃持典姿态、退后半步；主角作为反丹道执行者持断戒尺敲炉沿"叮"。**这一帧承担戒尺所有权从司徒明璋（现任持典者）向主角（戒尺承担者）的物理交接**——他放手让主角接管这把戒尺的承担。**他不是把戒尺交给主角的人**（戒尺的物理交接发生在 23 年前那一夜断成两截那一刻），他是**放手让主角合法持有那一截断戒尺的人**——这是法统层面的承认。

### 压力点 4｜Climax 拍 1 + Resolution，司徒明璋站着没动（character revelation 的最终落点）

- **表面层**：他是丹道宗门联盟现任首座；天空变色、雷云倒卷、苍生之炉降临——按宗门律他作为联盟首座应当主持苍生之炉的开炉仪式。
- **压力源**：Climax 拍 1 师傅从主角身后走过、解下宗主令牌放在主角脚边、走向炉口——师傅这一动作是把宗主令牌从司徒明璋（现任持有者）那里**未通过任何仪式地直接传给主角**（按宗门律应通过联盟年会传位+ 七宗联议批准+ 三月公示，师傅这一帧全部跳过）。**师傅破宗门律传位这件事在司徒明璋眼前发生**——他借了 18 年的信仰原来不是他以为的全部；师傅愿意为反丹道破宗门律。
- **翻面动作**：他**没有动**——他作为现任宗主在最大压力下能做的全部是**站着，看着师傅入炉**。Climax 拍 2-7 全程他**没有任何动作**——他不阻止、不出声、不举戒尺、不签宣告、不退场、不参与。**他是 Climax 七拍唯一在场但完全无动作的角色**。
- **露出的真实面**：维度 3 的最深引爆——他被信仰拯救过的事实（30-43 岁 13 年师傅的恩）和被信仰锁死的事实（5 年现任宗主任期+ 23 年前那一夜）在同一帧合上：**他被救活的恩在这一秒被师傅自己亲手撤销了**（师傅破宗门律=他借的信仰塌方），**他被锁死的牢在这一秒变成无主之牢**（信仰塌了但他还在牢里）。他没有动作不是因为他不能动——是因为**他没有任何东西支撑他动**。他借的全部都塌了，但他没有自己创立的丹方、没有付出过同等代价的合法性、没有打开过父亲临终笔记。**他余生的形态留白**——故事不交代他活了多久、做了什么、去了哪里。
- 他真实性格的最终命名在这一帧合上：**「借了别人一辈子的信仰，借到他自己的腿都软了的人——但他至死不肯承认那是借的」**。

## 8. 关系函数（这个角色对周围人施加什么独特压力 / 别人对他施加什么独特压力）

### 8.1 司徒明璋 → 主角（与 [[characters/protagonist]] §8.4 主角→现任宗主司徒明璋 函数互补不重复）

- **压力**：他是**「如果当年沈砺没被废、没反、安稳走正方路线」的镜像**——一个没走那条路的沈砺。主角看他的时候在某种程度上看见的是**反例的自己**——这是师傅看徒弟时看见沈砚的鬼魂的对位（师傅看徒弟看见已死的师弟；主角看二师兄看见未活的自己）。**他是主角『没走那条路』的物理承担者**——他每出现一次都让主角面对一个反事实假设：如果当年没反，主角现在会是什么样？答案是司徒明璋的样子：被信仰救过又锁死的人。
- **独特性**：他是**唯一可能让主角『当年没反』的反事实假设具象化的人**——任何其他人（师傅、师妹、沈砚）都是主角"反了之后"路上遇到的人；只有司徒明璋是"如果没反"的物理形态。这件事比任何台词都重——主角在 Seq 3.1 同门反目场对司徒明璋的姿态（持断戒尺站立、不让自己愤怒）不是仇视，是**对自己另一种可能性的物理凝视**。

### 8.2 司徒明璋 → 师傅（与 [[characters/master]] §8 师傅关系函数互补不重复）

- **压力**：他是**师傅一辈子立宗的法统延续者，也是师傅一辈子立宗的法统空壳**——他维持宗门律的全部公开形态，但他没有师傅一辈子立学说的本金（屠村避险费的肉身记忆）。他对师傅的施压是双向的：(a) **正向**——他维持师傅一辈子立的"正方"信仰的全部公开形态；让师傅一辈子的赌注在公共层级上不塌方；(b) **反向**——他借了师傅一辈子的代价但从未付出过同等代价；他每天踩过祠堂第七块砖二十年都不知道地宫秘密，**这件事本身就是师傅对他不信任的物理形态**。师傅每一次入井换药都在物质上证明：自己最大的秘密不能让司徒明璋知道。
- **独特性**：他是**唯一一个能在师傅退到联盟首席之后让师傅一辈子立宗的位置不塌方的人**——师傅 67 岁退位时把宗主令交给他这件事本身是师傅一辈子立宗的延续机制。但他承担这一延续的方式是**借**——这件事是他和师傅之间最深的不对称：师傅给他的是位置，但师傅从未让他知道位置背后的秘密。

### 8.3 司徒明璋 → 师妹（玄漪）

- **压力**：他这辈子都不知道师妹还活着——他以为她 23 岁那年死于沈砺手下。在他对师傅的宗法依赖里，"小师叔玄漪"是一个被供奉的过往（[[characters/sister]] §1.2 锁定）。**Seq 4.2 师妹现身那一帧是他一辈子合法性的第二次裂缝**——他第一次明白师傅一辈子立的"正"对他完全保留了一个秘密（藏女儿三年）。但他在 Seq 3.3 大典反炼第八转那一帧已经预演过这种崩——他第一次看见师傅没让戒尺落下时，合法性根开始动摇（[[characters/sister]] §8.4 锁定）。
- **独特性**：他是**师妹一辈子被师傅保留的最大秘密的局外人**——他越对师傅恭敬，他越是这个秘密的局外人；他的合法性越来自师傅的"正"，他对这个秘密的不在场就越深。**他与师妹之间是最深的隔阂**：他不知道她活着，她不知道他存在（师妹在地宫石棺三年里，意识半睡半醒，她从未想过谁是现任宗主——这一名词对她不存在）。

### 8.4 司徒明璋 → 师弟（已故沈砚）（与 [[characters/junior-ghost]] §8.4 沈砚→司徒明璋 函数互补不重复）

- **压力**：他从未见过沈砚（沈砚 26 岁屠村自焚时他还未出生 / 出生后沈砚已死 18 年）。他对沈砚的认识来自宗门档案的简略记载（"反丹屠村案首犯，自焚于丹炉"）+ 师傅 18 年师徒任期里偶尔提及的"我那位走偏的师弟"。**他对沈砚的情感是完全的隔阂**——他作为现任宗主的合法性建立在"反丹必反噬"上，沈砚是这条信念的物质本金；他不能让自己对沈砚有任何同情，否则他对师傅的宗法依赖就裂（[[characters/junior-ghost]] §8.4 锁定）。
- **独特性**：他是**沈砚作为师傅一辈子赌注的物质本金的不可处理者**——任何让司徒明璋开始同情沈砚（或开始怀疑师傅一辈子押的赌注）的处理都让司徒明璋的合法性裂开。**Seq 3.1 同门反目场司徒明璋说「师傅那夜下令的时候，手是抖的」**——这一句让主角第一次怀疑师傅废他不只是惩罚；同时也让司徒明璋自己第一次（在身体上）面对师傅一辈子的赌注的真正本金可能是怕（不是对反丹道的反对）——**这是司徒明璋合法性的第一次自己撕开**。

### 8.5 关系函数去掉司徒明璋的塌方测试

- **去掉司徒明璋**：
  - **C 折叠点（[[spine]] §7 同门反目场）失去承担者**——Seq 3.1 独立 3K 字 + Seq 3.3 第九转低头退半步 0.3K 字两段戏完全消失；
  - 主角"没走那条路的镜像"（[[characters/protagonist]] §8.4 锁定）消失——主角失去一个反事实假设的物理形态，他作为反道执行者的"另一种可能性"无对照；
  - 23 年前废徒大典上"师傅下令、谁动手"的执行者无承担者——师傅自己动手则把师傅作为父亲式角色的伦理重量加到不可承受的地步（[[characters/master]] §3 维度 1"反丹必反噬最坚定鼓吹者 vs 二十年私下研究反丹道最深执行者"无法承担更多伦理重量）；
  - 戒尺所有权从现任宗主向主角的物理交接（[[key-image]] §1 锁定）失去执行者——Seq 3.3 第九转的法统层面承认无承担者；
  - 类型契约的"师徒情/复仇剧"次类型的"二师兄对位场"（[[genre-contract]] §6 反陈词滥调 #14 锁定）消失——反陈词滥调保险塌一层；
  - **借来信仰的肉身**这一同侧对照（same-axis foil）的本体维度无承担者——师傅有屠村避险费的本金（信仰的代价）、主角有自封丹田反向的代价（反丹道的代价）、师妹有反向毒养透三年的代价（反核的代价）、沈砚有 50 年前自焚于丹炉的代价（开创者的代价）——只有司徒明璋**没有付出任何同等代价**，他是这条对照轴的唯一承担者；去掉他则故事中所有有代价付出的角色都失去对照（每个人都在付代价，没有人在借——故事失去"借"这条本体维度）。
- **结论**：司徒明璋是同侧对照（same-axis foil）的唯一承担者；去掉就立不住——主角失去"另一种可能性"的物理形态、师傅一辈子立宗的法统延续无承担者、戒尺所有权交接无执行者、借来信仰的肉身无人承载。**他是 spine 的承重墙之一**。

### 8.6 司徒明璋作为 same-axis foil 的特殊位置（与师傅 antagonist-as-mirror、师妹 foil-of-recognition、沈砚 ghost-engine 的精确分工）

> **他不是反派、不是次主角、不是侧人物——他是『同侧但没走到那一炉的人』。**

- **同侧**：他和主角都是师傅亲传弟子；他们都从同一个师傅的"正方"信仰出发；他们都接受过师傅作为父亲式角色的塑造。
- **没走到那一炉**：主角走到了反炼九转还魂的那一炉（Seq 3.3）+ 天道之炉（Climax）；司徒明璋没走任何一炉——他这辈子炼成的二品丹只有一炉（"乾元持守丹"）且方子是师傅炉子的临摹。**他在所有关键的反炼戏里只是在场、不出手、不参与、不创立**。
- **独特性（same-axis foil 与 antagonist-as-mirror 的对比）**：师傅是 antagonist-as-mirror（主角十年后镜中的脸——主角追的"被认"师傅追的"被三百口眼睛放过一次"互为镜像，且各自付出同等代价）；司徒明璋是 same-axis foil（主角"如果没反"的物理形态——他追的"不被师傅看见自己借的全部"与主角追的"被师傅再认一次"是分叉的两条路）。**两种对照方式是不同的物理学**：师傅在场对抗主角（反讽镜像），司徒明璋在场但不对抗（同侧分叉）。
- **独特性（same-axis foil 与 foil-of-recognition 的对比）**：师妹是 foil-of-recognition（同侧反向对位——追同一字、姿势相反；她保承认形态的不可被给予）；司徒明璋是 same-axis foil（同侧分叉——追的不是同一字，他追的是不被看见自己借的全部）。**两种"同侧"的物理学不同**：师妹是同侧反向（同方向但相反姿态）；司徒明璋是同侧分叉（曾共起点但走出不同方向）。
- **独特性（same-axis foil 与 ghost-engine 的对比）**：沈砚是 ghost-engine（不在场的施压者——他通过物理同形让主角的主体唯一性被解构）；司徒明璋是 same-axis foil（在场的反事实假设——他通过物理在场让主角面对"如果没反"的另一种可能性）。**两种"对主角施压方式"不同**：沈砚不在场施压主体唯一性（你不是你自己），司徒明璋在场施压可能性边界（你可能不是反道执行者）。

**四种角色函数（antagonist-as-mirror / foil-of-recognition / ghost-engine / same-axis-foil）合起来构成了承认轴在四个不同物理位置上的承担**：师傅承担承认轴的对面（对抗者），师妹承担承认轴的同侧反向（关闭者），沈砚承担承认轴的源头（前体），司徒明璋承担承认轴的旁支（同侧分叉者）。**四人合起来让承认轴在 negation of the negation 处合上时同时从四个方向被钉死**——这是 [[principle-of-antagonism]] 的极限合奏。

## 10. 承认轴 image system 对位

参 [[controlling-idea]] §3 锁定的三个意象——司徒明璋与三个意象都有关，但他的位置是**戒尺意象的物理交接者 + 玄色斗笠意象的同位被忽视者 + 那一字意象的不在场观众**。

| 意象 | 司徒明璋与之的关系 | 他是源头 / 执行者 / 交接者 / 不在场观众 |
|---|---|---|
| **戒尺（断半 + 完整）** | **他是戒尺所有权交接的执行者**——23 年前他执的那柄戒尺断成两截后，他从未问过另一截去哪了（§5 弹药 #5）；Seq 3.3 大典反炼第八转他第一次（在物质上）看见主角持着断尺的另一截；第九转他低头退后半步——**这是戒尺所有权从司徒明璋（现任持典者）向主角（戒尺承担者）的物理交接**（[[key-image]] §1 锁定）。**他放手让主角合法持有那一截断戒尺**——这是法统层面的承认。同时他腰间挂的完整戒尺（宗门正典戒尺的"备份"——见 §1.1 衣着）在 Climax 之后他**握着但没有举起**——这是戒尺意象的另一面（他持的完整戒尺成为不举起的姿势，与主角持的断戒尺举着未落形成精确对位） | **物理交接者 + 完整戒尺的不举起者**——这是他在戒尺意象上的双重位置：他既是断戒尺所有权的放手者，又是完整戒尺的不举起者 |
| **玄色斗笠** | 师傅出门唯一一种装束。**他与主角共享"被师傅忽视"这一状态**——师傅 60-72 岁 12 年师徒任期里从未在司徒明璋公开持典（27 次大典持典）的任何一次现场出现过；同样地，师傅从未在主角大胜的现场出现。**两人都被忽视，但姿势不同**：主角扫人群最外圈寻找玄色斗笠（找不到、还在等），司徒明璋从不扫（他不知道有什么可找的）。**他是这一意象的同位承担者，但他的承担形态是不知道自己在承担**——这是同侧分叉的最深一种形态 | **同位被忽视者**（不知道自己被忽视的那一面）——这一双重身份是他对玄色斗笠意象的特殊位置 |
| **那一字（雷音盖过的字）** | 师傅在 Climax 拍 3 张嘴说一字、雷音盖过、永不可解——**司徒明璋是这一帧的不在场观众**（他在场但不参与；师傅说的那一字不是对他说的）。口型四读"对/错/砺/漪"中没有"璋"——师傅最后想的不是司徒明璋。**这件事本身就是他作为同侧分叉的最深一种确认**：他借了师傅一辈子的赌注，但师傅最后想的不是他。**他是承认轴最终态的不在场观众**——他在物理上在场（他没有离开祭天台），但他在承认轴的层级上完全不在场（师傅最后那一字与他无关） | **不在场观众**（物理在场但承认层级不在场）——这是他在那一字意象上的特殊位置 |

**司徒明璋的特殊位置**：他是承认轴 image system 三个意象的**侧位承担者**——他在戒尺意象上是物理交接者+完整戒尺的不举起者；在玄色斗笠意象上是同位被忽视者（但不知道自己被忽视）；在那一字意象上是不在场观众（物理在场但承认层级不在场）。**三个意象在他身上的形态都是『侧』而非『正』**——他是 image system 旁支的承担者；他不在台前，但他在每一个意象的旁边以"借的、不知道的、不在场的"形态存在。这是 same-axis foil 在意象系统中的最深位置。

## 12. 给作者 / 下游的开放问题（≤5）

1. **司徒明璋父亲临终前那本笔记的内容**（§5 弹药 #4）——读者永远不知道里面写了什么（这是开放的弹药）。**主角是怎么知道父亲临终前让他看那本笔记这件事的**？建议：**通过 §5 弹药 #1 的旁支长老的另一句话**——旁支长老作为父亲的老朋友，知道父亲临终的话；他在 Seq 2.5 之后某场偶遇主角时附带提及。但主角**不重复**这件事的来源（他在 Seq 3.1 对司徒明璋说这件事时不解释怎么知道的——这一不解释让司徒明璋无法逃避面对这件事）。
2. **23 年前废徒大典闪回切片**（§7 压力点 1）——闪回字数约 200 字、位置建议放在 Seq 3.1 同门反目场之前的瞬时闪回（主角持断戒尺站在地宫入口前那一秒，司徒明璋看见断尺、闪回他自己 23 年前敲断戒尺的那个瞬间）。**这个闪回是否需要以司徒明璋的视角呈现，还是以主角的视角呈现**？建议**司徒明璋的视角**——因为主角的被废那夜已在 [[characters/protagonist]] §6.2 + [[premise-card]] §3 锁定为主角视角；这一闪回应是司徒明璋视角的对位（他执戒尺那一刻的视角——"我那一刻手稳"）。这个对位让 Seq 3.1 主角说"师兄，你那夜手是抖的"的反向引爆有具体的物质对照。
3. **Seq 3.1 同门反目场司徒明璋对主角说的"师傅那夜下令的时候，手是抖的"**（[[spine]] §13 #1 锁定的开放问题之一）——这一句必须满足两条：(a) 让读者第一次怀疑师傅废徒不只是惩罚；(b) 让司徒明璋作为现任宗主的合法性裂缝在这一句中被自己撕开。**还可以加一条**：让司徒明璋说出这句话的姿势是**他没有看主角，他看着地宫入口那块青砖**（虽然他还不知道那块砖下是什么，但他下意识地朝那里看）——这一姿势让读者读出他潜意识里**已经感觉到师傅有秘密**，但他自己还不能命名。
4. **Seq 3.3 大典反炼第九转司徒明璋低头退后半步那一帧**（C 折叠后半 0.3K 字）——他低头时**面部表情**是什么？建议**没有可读情绪**——同主角 Climax 拍 7 的"面无可读情绪"对位（[[characters/protagonist]] §7 压力点 3 锁定）：两人都在最大压力下展现出"没有可读情绪"的姿态，但成因相反——主角是承认轴在他身上落于 negation of the negation 后的余韵（他余生不可知）；司徒明璋是借来信仰在他身上塌方时的中空（他不知道自己感觉到了什么）。**两人面无可读情绪的同帧对位是 spine 反讽极性的最深一种合奏**。
5. **Resolution 司徒明璋握完整戒尺不举起的那一帧**（§6.3 #17）——这一帧**是否需要在 Climax 拍 7 主角举断戒尺转身离开时让读者直接看见**？建议**让读者间接看见**——主角离开祭天台的镜头跟着主角走入晨雾；司徒明璋仍在主坛持典位，**只有读者通过镜头切换的瞬间瞥见司徒明璋的腰间——完整戒尺仍挂在腰间，他没有举起**。**他余生的形态留白**——故事不交代他活了多久、做了什么、去了哪里。这一处理让 司徒明璋的最后一帧不与主角的最后一帧争夺重量（[[controlling-idea]] §7 违规列表 #11 锁定的"主角的最后表情必须是断戒尺举着未落、面无可读情绪"硬约束严守）。
