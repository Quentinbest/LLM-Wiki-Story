# Character Arc ——玄漪

> 上游契约（全部 locked）：[[characters/sister]] / [[characters/protagonist-arc]] / [[characters/master-arc]] / [[spine]] / [[controlling-idea]] / [[key-image]]
> 弧形：**反向核 / 部分反损弧（reverse-core arc with partial-erosion resolution）** —— 不是主动 arc，不是"完成"或"成长"；是"被反向核养三年的活物在 Climax 前以三句话兑现自己的存在 + 在炉中主动让自己意识反损 + 在 Resolution 以张嘴叫不出『师兄』二字的形态合上"
> 极性对位：反讽（ironic, negative）· [[negation-of-the-negation|否定之否定]] 在承认轴二审通道层面的物理化
> 关系函数：**foil-of-recognition / 同侧反向对位** —— 与主角追同一字（"父亲那一字"），但姿势相反（主角求承认被验证 / 师妹保承认形态的不可被给予）

## 0. 一段定调

玄漪不是主动 arc 的角色。她的弧光不是"由 A 变 B 的成长"，也不是"被牺牲的受害者从存活到消解"——她是 **反向核 / 部分反损弧（reverse-core arc with partial-erosion resolution）**：被父亲以反丹原理藏在地宫第七层逆经井三年、肉身境界冻结于 23 岁、意识半睡半醒、每七日睁眼一刻钟的活物——她的"活着"本身就是反丹道成立的物理证据。

她在 spine 三十四个月的大部分时间不在场——她在地宫第七层的汉白玉石棺里。她的 arc **大部分通过缺席被读出**——她是承认轴 image system 之外的一个**对偶系统**：师傅的"玄色斗笠不出现"对位主角扫人群最外圈寻找承认者的不在场；师妹的"在石棺里看着师兄三年但不能动"对位主角"在反丹道一炉接一炉中等师傅再点一次头"的潜意识仪式。**两条缺席系统在同一物理空间反向运作**——一条在世界的最外圈（斗笠），一条在世界的最深处（井底）；两条都是"承认者的不在场"的不同形态。

但她不是被动的反向核。她在地宫三年里**部分意识清醒**——她拼出了三个人（父亲、师兄、自己）的全部因果链；她在石棺壁上以反向丹气写"砚"字 156 次；她在 Climax 拍 5 反向核被双面丹中和时**主动**让自己的部分意识反损——她记得师兄但叫不出"师兄"二字、她记得父亲但说不出他临终那一字。**这是她对承认轴负向极性的最深一种贡献**：她让承认的二审通道在自己身上被永久关闭。

**她不是完成弧光的角色，她是承担反讽极性物理形态的角色**。她的 arc 不是"变化"，是**部分反损**——她的肉身活下来、记忆部分恢复、但语言能力对应"师兄"和"父亲那一字"两个核词被她自己主动从身体上抹去。这是 [[controlling-idea]] §1 锁定句下半句"承认这件事被永远逐出可被验证的世界"在她身上的物理兑现——她是承认轴负向极性的最后一颗钉子。

## 1. 玄漪 arc 形态

### 1.1 反向核 / 部分反损弧（reverse-core arc with partial-erosion resolution）

**核心定义**：玄漪 arc 的形态不是常规意义上的"主角弧光"——她不是 spine 主体，她不追求 Want、她不在 Crisis 处做选择、她不在 Climax 处采取动作。**她是 spine 主体（主角 + 师傅）追求的物理基础与对偶系统**——她活着这件事是反丹道成立的物理证据；她在场=反丹道有解；她不在场=师傅的避险费白付。

她的 arc 由三段组成：

| 段 | 时间 | 形态 | 物理状态 |
|---|---|---|---|
| **段 1（不可见 arc）** | 故事开场前三年 → Seq 3.4 | **被反向核养三年的部分意识清醒**——她在地宫石棺里大部分时间半睡半醒、每七日睁眼一刻钟、部分意识清醒；她的"活着"是反丹道在父亲身上的活体证据 | 肉身境界冻结于 23 岁；皮肤近乎透明；唇色青白；每七日睁眼时瞳孔反向变化（黑→灰→金→灰→黑） |
| **段 2（可见 arc 的三句话）** | Seq 3.4 / Seq 4.2 / Climax 拍 4-5 | **三句话兑现自己的存在**——这是她在 spine 中唯一可见的"动作 arc"。每一句话都同时承担"对一个人的最终定位"+"对自己存在的物质化"+"对承认轴的对应推进" | 三句话依次发出，对应三种递降语态：陈述沉默（Seq 3.4）→ 陈述父亲一辈子的裂缝（Seq 4.2 上半）→ 重复 23 岁那夜对师兄的话（Seq 4.2 下半） |
| **段 3（部分反损 arc）** | Climax 拍 5 → Resolution | **在炉中主动让意识部分反损**——她在反向核被双面丹中和的瞬间主动让自己关闭承认轴的二审通道；她的语言能力在"师兄"和"父亲那一字"两个核词上被自己抹去 | Climax 拍 5：肉身重新顺长、意识恢复但反损；Resolution：站在反向核位置张嘴想叫"师兄"——叫不出 |

### 1.2 这条 arc 不是别的——四种被禁止的形态

按 [[characters/sister]] §2 锁定 + [[controlling-idea]] §7 违规列表 + [[genre-contract]] §6 反陈词滥调，玄漪 arc 必须**主动排除**以下四种常见师妹戏的写法：

| 被禁止的 arc 形态 | 形态描述 | 为什么违规 |
|---|---|---|
| **(A) 爱情对象 arc** | 师妹是主角的恋人；她活下来后与师兄余生厮守 | [[genre-contract]] §6 反陈词滥调 #1 严禁。她对主角的爱是"在被承认之外被看见"的爱，不是恋人；爱情对象 arc 会把师妹的关系函数（[[characters/sister]] §8.1）降级为情感配角 |
| **(B) 被牺牲的受害者 arc** | 师妹被父亲推入炉中作衬体；她无自主意愿；她的死是"工具被消耗"形态 | [[genre-contract]] §6 反陈词滥调 #3 严禁。她**自愿**入炉——这是她作为有内心活动的活人的最后主动动作；被牺牲的受害者 arc 会把她降级为反核工具 |
| **(C) 觉醒了的反向核 arc** | 师妹在地宫三年里"突然觉醒过来"——意识恢复 + 看清真相 + 入炉是"觉醒后的选择" | [[genre-contract]] §6 反陈词滥调 #8 严禁 + 觉醒叙事违反反讽极性。她在地宫三年是**逐步部分清醒**（不是一夜觉醒），且她的清醒不是"突然明白"——她是**把明白过来的东西用进自己的死法里**的人 |
| **(D) 承认替代者 arc** | 师妹活下来 → 完全恢复 → 告诉主角"父亲那一字是什么" → 主角余生有承认的二审通道 | [[controlling-idea]] §7 违规列表 #9 严禁（"替代承认者"）+ §7 #4 严禁（"师妹在结尾醒来对主角说『师傅其实一直认你对』")。这一选直接撕掉 controlling idea 锁定句下半句 |

### 1.3 推荐 arc 形态：反向核 / 部分反损弧

**核心物理**：师妹三年里 **被反向核养透 + 部分意识清醒**（段 1）；在 spine 末段以**三句话**兑现自己的存在（段 2）；在 Climax 拍 5 主动让意识部分反损（段 3 关键转折）；在 Resolution 以**张嘴叫不出"师兄"二字**的形态合上。

**关键**：段 3 的"部分反损"不是反向核被中和的物理副作用——是她**主动选择**的形而上动作。她在反核中和的瞬间**自主决定**让"师兄"两个字从她口中被抹去、让"父亲那一字"从她记忆中被抹去。这一动作的物理上不可被读者直接看见（炉中的瞬时），但读者在 Resolution 那一帧（她张嘴叫不出"师兄"）回头明白这是她主动选的。

**这条 arc 与主角 reverse arc + 师傅 reverse-bet arc 形成三足鼎立的镜像合上**：
- 主角："选 + 被夺走"（主体性的最大行动被另一个角色的主体性吞掉）
- 师傅："翻 + 不解释"（一辈子的赌注在自己手中被翻面但被自己取消可被验证性）
- 师妹："活 + 部分反损"（活下来但语言能力的承认轴核词被自己主动从身体上抹去）

**三个动作在 Climax 拍 3-7 同帧合上——这是反讽极性下三足 arc 的最高密度形态**。

## 2. Want vs Need（自觉欲望 vs 潜意识欲望）

### 2.1 自觉欲望（Conscious Want）

> **不让父亲与师兄在炉前真正面对面那一天到来——延后、化解、用自己的活着换那一天不来。**

- **形态**：以**保护性沉默**为姿势的延后机制。她 19 岁那年说过的"我怕父亲与师兄真正面对面那一天"是她自觉欲望的初次外化；她 23 岁那年向师兄求救并赌他的反向直觉是她**算定**"父亲会因为这一炉再也不能回到对师兄的简单认同"——所以她要让师兄反炼救她、让父亲被迫面对自己一辈子立宗的裂缝。
- **要点**：她要的不是"父亲赢"也不是"师兄赢"——她要的是**让那一天不发生**。她**算错的不是反向直觉**，是师兄那一夜对"被父亲承认"的需要会压过对她的爱——师兄选了正方丹方，她死在他炉前。她在石棺里三年看清楚了这一点。
- **物质载体**：她 23 岁那夜对师兄说的最后一句话"师兄，你不必照着方子来"——这一句不是哀求、不是控诉，是**留给师兄一辈子追悔的引子**（[[characters/sister]] §7 压力点 1）；她在地宫三年中每七日睁眼一刻钟时与主角的反丹气同频共振（弹药 #5）——她在石棺里看见主角三年里每一炉、每一次敲断戒尺、每一次扫人群最外圈寻找父亲——**但她不能动、不能说话、只能看**。
- **在文本里如何被读出**：她的自觉欲望在 spine 大部分时间通过**缺席**被表达——她不在场、不说话、不动；她的"不让那一天到来"以**自己被藏在地宫三年**的形态被父亲执行（父亲藏她的物理动作=她自觉欲望的物质化继续）；直到 Seq 4.2 她主动走出地宫现身——她**自己**取消了自己自觉欲望的物质形态。

### 2.2 潜意识欲望（Unconscious Need）

> **让师兄被父亲完整承认——以父亲口误命名时本想叫的"砚"字的形态被承认。**

- **形态**：不需要见证人、不需要"对错"裁决——一次**父亲心里的真相被验证**。师兄不是"砺"，他是"砚"——父亲给师兄命名时本想叫的字（[[characters/sister]] §1.1 微动作 2 + §4.2 + [[characters/protagonist]] §1 名字栏）。她要让师兄被父亲在心里真正叫的那个名字承认。
- **要点**：她要的不是"父亲承认师兄做对了反丹道"——她要的是"父亲承认师兄是他心里真正在叫的那个人"。这一秒承担的是父亲一辈子最不肯让任何人写出的那个字——师弟沈砚的"砚"——在主角身上被父亲完整给出。**她追的也是『父亲那一字』，但她追的方式是让那一字以师弟的名字、以父亲口误时的真实在主角身上被承认**。
- **物质载体**：她每七日睁眼那一刻钟里用右手在石棺壁上以反向丹气写"砚"字——三年下来棺壁内侧密密麻麻刻着同一个字，刻了 156 次（每七日一次 × 三年）。这个动作出卖她——她在父亲一辈子的禁忌里**用师兄被父亲口误命名时的那个字给师兄留位置**。
- **在文本里如何被读出**：她**至今没有意识到这一点**；她自己以为她在为师兄出力。读者要在 Seq 3.4 主角入井那一帧第一次看见棺壁内侧的"砚"字——读者**在主角之前**先看见"师妹三年里都在写父亲口误的那个字"。主角随后看见——他第一次明白自己活着的形态本身就是沈砚未尽的炉的延续，而师妹早就知道。

### 2.3 Want 与 Need 的不可调和

**这两件事不能同时为真**。这是 [[characters/sister]] §4.3 锁定的硬力学：

- **不让父亲与师兄相撞（Want）的形态** = 延后那一天 = 她用自己的活着换那一天不发生；
- **让师兄被父亲完整承认（Need）的形态** = 那一字以"砚"的形态被父亲在主角身上给出 = **那一天必须发生**（因为只有那一天父亲才会面对主角=沈砚的形似回响）。

她最终选择的是：**让那一天发生，但让那一天的内容永远不可解**——师傅说一字、雷音盖过、她意识反损叫不出。**这是她对两层欲望的同时拒绝**——她不让任一层完全兑现。

### 2.4 与主角 + 师傅 Want vs Need 的对位

| 角色 | Want | Need | 不可调和的形态 |
|---|---|---|---|
| **主角（沈砺）** | 让师傅当众承认他对（反丹道立宗） | 被师傅再认一次（私密点头） | 师傅一旦点头=真理被亲情软化（Want 失）；师傅永不点头=赢的姿势空对不在场的位置（Need 失）|
| **师傅（玄琢真人）** | 守住三百口避险费（不让反丹成功）| 被三百口眼睛放过一次（亲眼看见反丹真的成立）| 不让反丹成功=Need 永不兑现；亲眼看见反丹成立=避险费白付（Want 失）|
| **师妹（玄漪）** | 不让父亲与师兄在炉前相撞 | 让师兄被父亲完整承认（以"砚"的形态）| 不让相撞=Need 永不兑现（"砚"的承认需要那一天发生）；让相撞=父亲与师兄都将面对一辈子立的"正"的裂缝（Want 失）|

**三人的 Want vs Need 在终极反转炉前同时合上**——但师妹比主角和师傅都早三年看清这一点。**这是 foil-of-recognition 与 antagonist-as-mirror 的精确分工**：师傅是主角的反面镜像（他十年后镜中的脸），师妹是主角的**同侧反向**（追同一字、姿势相反）——他求承认的形态的，她保承认形态的不可被给予。

### 2.5 Want-to-Need migration —— 反向核 arc 形态确认

按 McKee Ch. 5 / Ch. 13 / Ch. 17：常规 character arc 是主角从执着 want 到真正面对 need 的转折——通常在 Crisis 处主角在最大压力下选择 need 而非 want。

**玄漪的 arc 不是这个形态**——她不在 Crisis 处做选择（Crisis 是主角的）；她也不"选 need 而非 want"。**她在 Climax 拍 5 同时拒绝 Want 和 Need 的完全兑现**——她让那一天发生（Want 失败）但让那一天的内容永远不可解（Need 也不兑现）。

这是反讽极性下 foil-of-recognition arc 的精确形态：**她不在追求 want 或选择 need 的过程中失败——她抵达了一个二者都不能完全兑现的第三态**（让那一天发生 + 让那一天的内容不可解）。她的 arc 不是"want 到 need 的转折"，是"want 与 need 同帧悬置"——这是反讽极性在 foil-of-recognition 身上的精确投影。

## 3. 反向核 arc 的特殊性 —— 不主动，但在场即对抗赌注的物理证据

### 3.1 她不是 spine 主体——她是 spine 主体追求的物理基础

按 McKee Ch. 7 [[spine]]：spine 是主人公追逐欲望对象的生命线，由激励事件点燃 → 高潮处被授予/拒绝/翻面。**玄漪不是 spine 主体**——她不是主人公（[[characters/protagonist]]）、不是次主体配额（[[characters/master]] 占次主体配额=antagonist-as-mirror）。她不竞争主人公地位。

但她**是 spine 物理学的承重墙之一**——按 [[setting-survey]] §3.4 + §6 硬规则 5 锁定：**天道之炉双面丹的物理钥匙是反向核**——反向核必须是**已经被反向气养满三年的活物**；**故事内符合此条件的活物只有一个：师妹玄漪**。

**这是反向核 arc 的特殊性**：

- **她不主动**——她不追求 Want（她的"自觉欲望"是延后那一天，不是去争取什么）；
- **她不选择**（直到 Seq 4.2 之前）——她在地宫三年里只能看、不能动、不能说话；
- **但她在场即对抗赌注的物理证据**——她活着这件事=反丹道有解=师傅一辈子押的避险费的"另一种可能性"的物理证据；她不在场=师傅的避险费白付。

**这是 spine 的隐性承重墙**——她不在 spine 的台前（不是主体），但她在 spine 的肉身底层（她活着这件事让 Climax 物理上可能）。

### 3.2 她对师傅一辈子赌注的对抗

按 [[characters/master-arc]] §1.2 锁定：师傅的 Need = "被五十年前那场屠村的三百口眼睛放过一次——亲眼看见反丹道真的成立一次以解放自己"。

**师妹活着这件事是师傅 Need 的私密形态兑现 + 同时是师傅 Want 的彻底失败的物理证据**：

| 师傅赌注的形态 | 师妹活着对它的对抗 |
|---|---|

### 3.3 她对主角"我害死了她"自我叙事的对抗

按 [[characters/protagonist-arc]] §0 锁定：主角三年来用"我用正方杀了她"自我叙事撑住自己活下去；激励事件后这条自我叙事开始动摇，但直到 Seq 3.4 师妹活着曝光时才被釜底抽薪。

- **Seq 3.4 师妹活着被披露**：自我叙事被釜底抽薪。主角失去三年来活下去的支撑信念。

**这是反向核 arc 对主角的对抗**：她不主动 arc，但她活着这件事**作为物质反例**在主角"我害死了她"自我叙事的每一炉中潜伏存在——她**用自己被藏的三年支撑主角三年来反丹道每一炉的反讽密度**。

## 6. "未落"的对偶 ——师妹"未叫出口"的对位

按 [[key-image]] §1.3 锁定：戒尺三态合一（"断的、举着的、未落的"）中"未落"承担 controlling idea 锁定句下半句"承认这件事被永远逐出可被验证的世界"——承认仪式的物理沉默化。

### 6.1 师妹身上的"未落"对偶

**师妹身上有一个戒尺"未落"的对偶——她"未叫出口的师兄二字"**。

按 [[characters/sister]] §3 维度 4 + §10 image system 对位 + spine §10 Resolution 锁定 (b) 部分反损形态：

| 戒尺"未落"（主角侧）| 师妹"未叫出口"（师妹侧）|
|---|---|
| 主角左手按炉沿、右手举起断戒尺准备最后一击（敲炉沿那声"叮"= 承认仪式）；戒尺还未落下；师傅先一步走入炉中夺走主角的选择 | 师妹站在祭天台的反向核位置上、张嘴想叫"师兄"二字（叫"师兄"= 承认师兄是她的师兄的言语仪式）；"师兄"二字未发出口；她在炉中主动让自己关闭这条言语能力 |
| 承认仪式的"叮"永远不再响 | 承认师兄的言语"师兄"永远不再出口 |
| 戒尺的不落由主角的选择被夺走承担 | "师兄"二字的不出口由师妹自己主动选择承担 |
| 这是承认轴对立级 → 否定之否定的过渡帧 | 这是承认轴二审通道层面的关闭帧 |
| 物件载体的物理沉默化 | 言语载体的物理沉默化 |

**两个"未"在同一物理空间（祭天台）同时承担承认轴否定之否定的两个维度**：
- **戒尺未落** = **主角侧的承认轴沉默化**（承认仪式的"叮"永远不响）—— 主角的主体性最大行动被对方先一步选走；
- **师兄未叫** = **师妹侧的承认轴二审通道关闭**（言语对师兄的承认永远不出口）—— 师妹的唯一可能成为承认二审通道的人主动放弃证人位。

**两个"未"是对偶系统**：戒尺三态合一（断+举+未落）中"未落"承担 controlling idea 下半句；师妹身上"叫不出师兄"承担 controlling idea 下半句的另一个维度（二审通道层面）。**两个"未"在 Resolution 同帧合上 —— 这是反讽极性下双"未"系统的精确语法**。

### 6.2 这个对偶如何在 Resolution 收束

按 spine §10 + [[characters/sister]] §3 维度 4 引爆场景锁定：

**Resolution 那一帧的物理图像**（写给 scene-architect）：

> 主角举着断戒尺转身离开祭天台、走入晨雾。雾里没有任何东西在等他。
>
> 与此同时，师妹站在祭天台的反向核位置上、看着师兄背影远去。她张嘴想叫"师兄"——叫不出。
>
> 主角没有回头。

**这一帧的双"未"对偶**：
- **戒尺未落（主角侧）**：主角的右手举着断戒尺，从此再没有敲过任何炉——承认仪式的"叮"永远不再响；
- **师兄未叫（师妹侧）**：师妹张嘴想叫"师兄"——叫不出；她对这件事的接受让读者看出她是同谋而不是受害者；
- **两个"未"在同一物理空间同时发生** —— 主角的"戒尺未落"与师妹的"师兄未叫"在同一镜头时间内分别承担承认轴否定之否定的两个维度；
- **主角没有回头** —— 这是两条对偶系统的最终隔断：主角的"未"和师妹的"未"在同一空间但永不交汇——双"未"系统在 Resolution 以**互不知道对方"未"的姿势**合上。

**这是反讽极性下双"未"系统的精确兑现**：两人都做出了能做的最大选择（主角举戒尺、师妹叫师兄），但两人的选择都在落下之前被结构性地取消——主角的选择被师傅夺走，师妹的选择被自己主动关闭。**两条承认轴在同一帧上以不同方式沉默化**。

### 6.3 这个对偶不可被妥协的硬约束

按 [[controlling-idea]] §7 违规列表 + [[key-image]] §1.3 锁定，双"未"对偶必须满足以下硬约束：

1. **师妹"叫不出师兄"必须由师妹自己主动选择承担** —— 任何让师妹"想叫但意识反损所以叫不出"的写法都把师妹降级为反核中和的副产品；师妹必须**明确意识到自己叫不出**，且接受这是她自己主动选的——这是 [[characters/sister]] §3 维度 4 引爆点的物理兑现。
2. **师妹"叫不出"的姿势不能挣扎、不能重试、不能哭** —— 任何"她尝试三次都叫不出师兄"或"她叫不出师兄然后哭泣"的写法都让师妹的接受变成"她在为这件事难过"——这违反维度 4 的"她是同谋而不是受害者"硬要求。
3. **主角没有回头这件事不能被任何后置物件追溯** —— 任何"多年后主角回到祭天台师妹仍在等"或"主角余生某夜梦见师妹叫师兄"的写法都让师妹的"叫不出"在余生中以某种形态被恢复——这违反 [[controlling-idea]] §7 #5 + #11 的反讽极性硬要求。
4. **戒尺未落 + 师兄未叫必须在同一镜头时间内分别承担** —— 不是先后；读者必须看见两个"未"在同一物理空间（祭天台）同时存在——一个在主角的右手上（举着断戒尺），一个在师妹的口中（张嘴叫不出）。这是双"未"系统的同帧合上硬要求。
5. **师妹不能在 Resolution 后"恢复"** —— 任何"师妹余生逐渐恢复说『师兄』二字"或"师妹某天突然能叫『师兄』"的写法都是 idealist 极性，直接撕掉 controlling idea。师妹的部分反损是**永久**的——这是 [[controlling-idea]] §7 #5 + spine §10 锁定 (b) 的硬要求。

2. **棺壁内侧的"砚"字必须密密麻麻 156 次** —— 物理细节：每七日一次 × 三年 = 156 次；每个"砚"字的位置稍稍偏移（不重叠）；最近一次写的"砚"字比往常更重（对应 Seq 3.3 大典反炼第八转那一刻师妹同步认出师兄反炼九转还魂的情绪重量）。
3. **不能让师妹解释她为什么写"砚"字** —— 她的目光在师兄腰间断戒尺上停一瞬即可；任何解释让师妹的潜意识欲望（让师兄被父亲完整承认）变成自觉欲望，撕掉 [[characters/sister]] §4.2 锁定的反讽密度。
4. **主角第一次完整意识到自己活着的形态本身就是沈砚未尽的炉的延续，而师妹早就知道** —— 这一意识必须通过主角的物理动作或表情承担（如"他直起身，看见井底石棺缝里师妹睁着眼"那一帧主角的眼神变化），不能通过任何内心独白承担。

### 7.4 Seq 4.2 师妹三句话的 scene-architect 硬约束

按 [[characters/sister]] §5 弹药 #1 + 弹药 #3 + §7 压力点 3 + [[characters/sister]] §12 #3 锁定，Seq 4.2 师妹三句话的 scene 必须满足：

1. **第一句话（对父亲）必须是陈述句结尾，不能加问号或试探语气** —— "父亲，您当年炼凝魂丹也是反着炼了一半的。" —— 她不是在问父亲是不是反着炼了一半，她是在告诉父亲她知道。这一句的份量来自她不给父亲否认的余地。
2. **第二句话（对师兄）必须是重复 23 岁那夜对师兄的最后一句话** —— "师兄，这次你别照方子来。" —— 这一句重复弹药 #3；师兄第一次（在听见之外）听见师妹三年前那句话其实是在赌他。
3. **两句话之间不能有任何其他对话或动作描写** —— 师妹对父亲那一句话、然后转向师兄那一句话，两句话之间只能有师妹转身的物理动作（约 30 字以内）。
4. **师妹不告诉师兄她当年在赌他** —— 师妹让师兄余生可以选择"以为师妹三年前那句话是哀求"或"以为是试题"——她让这条因果链也保持不可验证（弹药 #3）。
5. **师妹的语气必须是陈述不解释** —— 不哭、不哀求、不嘱托；她用两句话同时完成对父亲一辈子和对师兄一辈子的精确定位——但她不解释。

### 7.5 这条 arc 不能交付的事

按 [[controlling-idea]] §7 违规列表 + [[genre-contract]] §6 反陈词滥调 + 本文档 §1.2 四种被禁止 arc 形态：

1. **不能把师妹写成爱情对象** —— 任何让师妹与主角的关系被读为恋人/爱情的写法都违反 [[genre-contract]] §6 反陈词滥调 #1。
2. **不能让师妹在 Climax 之前出场说话** —— 师妹在 spine 中第一次开口说话是 Seq 3.4 那一帧（"师兄，原来你回来了"）；任何让师妹在 Seq 3.4 之前出场说话的写法都破坏她"沉默二十六年的份量"硬要求。
3. **不能让师妹成为"承认替代者"** —— 任何让师妹告诉主角"父亲那一字是什么"或"父亲其实早就认你对了"或类似的写法都直接违反 [[controlling-idea]] §7 违规列表 #9 + #4，撕掉 controlling idea。
4. **不能让师妹在 Resolution 后"恢复"** —— 任何"师妹余生逐渐恢复说『师兄』二字"或"师妹某天突然能叫『师兄』"的写法都是 idealist 极性，直接撕掉 controlling idea。师妹的部分反损是**永久**的。
5. **不能让师妹的自愿入炉读起来是"为爱牺牲"** —— 她自愿入炉的物理动机是她在地宫石棺三年里看穿了承认这件事的可被给予在自己身上是结构性悬空的、并主动让自己的存活成为这个悬空的物质形态；不是任何形式的爱的兑现。
6. **不能让师妹在地宫三年里"觉醒过来"** —— 她是逐步部分清醒（不是一夜觉醒）；她不是"突然明白"，她是把明白过来的东西用进自己的死法里。
7. **不能让师妹解释她为什么写"砚"字 156 次** —— 她自己未必能完全说清；任何解释让她的潜意识欲望变成自觉欲望，撕掉反讽密度。

### 8.2 Pin 11（Climax 拍 3-5）是 recognition / 还是 action

按 arc-tracer 系统提示 §5：Crisis 必须是 recognition（决定）/ Climax 必须是 action（动作）。

**师妹的 Crisis（Pin 10 处她由父亲扶出现身、对父亲说一句话、对师兄说一句话）**：师妹在被父亲扶出地宫之前已经做出**她自己的 Crisis 决定**：自愿入炉作反向核 + 在炉中主动让自己的部分意识反损。这是 recognition——她在内在认识到"这一刻是我做出我这辈子能做的最大选择的最后机会"。

**师妹的 Climax（Pin 11 / Climax 拍 3-5）**：她在炉中的最后一个完整意识动作是主动让自己的部分意识被反损——这是 action，不是 recognition。这一物理动作（炉中瞬时）兑现 Crisis 决定的物理形态。

**两者明确分离**：
- 师妹的 Crisis（Seq 4.2 现身那一帧）= 决定自愿入炉 + 决定在炉中关闭承认轴二审通道——这是 recognition；
- 师妹的 Climax（Climax 拍 3-5）= 在炉中主动让自己的部分意识被反损——这是 action。

**但师妹的 recognition 和 action 之间的物理时间差很小**（Seq 4.2 现身 → Climax 拍 3-5，约 7 日内）；与主角和师傅的 recognition-action 时间差相比，师妹的时间差最短。**这是反向核 arc 的特殊性**：她的最大选择和最大动作几乎同帧合上，但仍然在 recognition / action 上分离。

### 8.3 删掉中段任一节点后 Climax recognition / action 还成立吗

### 8.4 frontmatter 的 closing_state 与 opening_state 是否真正不同

- **opening_state**：23 岁那夜炉前断气前最后一句话"师兄你不必照着方子来"——已预判师兄会败的师妹，在赌师兄的反向直觉，主动以一句不带控诉的话留给师兄余生追悔的引子。
- **closing_state**：记得师兄但叫不出"师兄"二字——在炉中主动让自己意识部分反损，用自己半残废的活着让师兄余生的承认问题永远悬空。

**两者明确不同**：
- opening 是**主动用一句话留给师兄余生追悔的引子**（她预判会败，但她用一句话把选择交给师兄，赌他的反向直觉）；
- closing 是**主动让自己的存活成为承认结构性悬空的物质形态**（她活着，但活着的方式让她不能告诉师兄父亲那一字是什么，也叫不出"师兄"二字）。

**这不是 flat arc**——它是反向核 / 部分反损弧（reverse-core arc with partial-erosion resolution）。师妹的"主动用一句话"在 opening 与 closing 之间发生了形态变化：从"用语言对师兄说话"变为"用沉默对师兄说话"——她的语言能力在承认轴的核词上被自己永久关闭。这是承认轴在她身上从可言喻到不可言喻的递降。

### 8.5 arc 终态 value charge 是否对位 controlling idea value pole

- Controlling idea value pole = **反讽（ironic, negative） · 否定之否定**
- Arc 终态 = **反向核 / 部分反损 = 师妹活着 + 师妹关闭承认轴二审通道 = 承认本身被翻面在她身上以"叫不出"的形态永远不可解（Pin 11 + Pin 12 + 双"未"对偶）**

**完全对位**。

承认轴在师妹身上完成了一个特殊的递降——她不是 spine 主体，但她承担承认轴最后一颗钉子：
- 她是**唯一可能成为承认二审通道的人**（除主角外，她是炉中唯一在场的他者，理论上她若意识完整，她就是承认这件事的二审证人）；
- 她**主动放弃证人位**（在炉中主动让自己的部分意识被反损）；
- 她的"叫不出师兄"与主角的"戒尺未落"构成双"未"对偶系统，在 Resolution 同帧合上 —— 主角侧的承认轴沉默化（物件载体）+ 师妹侧的承认轴二审通道关闭（言语载体）= 承认轴在物件和言语两个维度同时被沉默化。

**这是反讽极性下 foil-of-recognition arc 的终极兑现**——师妹作为同侧反向对位（与主角追同一字但姿势相反）的最深含义在 Pin 11 + Pin 12 同帧合上。
