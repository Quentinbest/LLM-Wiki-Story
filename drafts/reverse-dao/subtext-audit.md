# Subtext Audit ——《废徒倒丹》
Date: 2026-05-21

## Overall Verdict: **PASS（一处显式潜文本叙述需收紧）**

整体上这是一份**罕见地高度承载潜文本**的稿件。沈砺的"敲断戒尺一下""扫人群最外圈""按反序下锅"在三循环里每一次都被叙述者按下不解释，而读者通过累积、复现、错位读出潜意识欲望的形状。师傅"按右膝/食指一抖/绕第七块青砖"的三个 tell 全部以**纯物理动作**承担，从未由叙述者解码——师傅二十年押的赌注、对女儿的私密研究、对师弟的债，都以"动作 + 缺席的解释"的形式潜伏在场内。师妹的"三句话 + 写砚字 156 次 + 张嘴叫不出"是全篇潜文本最干净的承担——她从不说穿自己的欲望，她的沉默就是欲望本身。

需要警觉的不是 on-the-nose 台词（几乎没有），而是几处**叙述者代笔的潜文本显式化**——主要在 4-4-1 出现一处"三层潜文本"明文罗列、4-4-5 多处直接引用 controlling-idea / 硬约束作为叙述者旁注。这些不是人物 on-the-nose，是**注释 on-the-nose**：叙述者越过了它应该保持的克制。具体见 §3。

silent-screenplay 通过率（采样八场）：约 **6/8 通过 / 2/8 部分**——1-3-2 第二半（"师叔那一炉..."自由间接独白）与 4-4-1 末"三层潜文本"两处依赖文字解码。

## On-the-Nose Flags（text ≈ want / 叙述 ≈ subtext）

实质 on-the-nose 的人物台词**只有一处轻度风险**。其余全是叙述者层面的过度解码。

### Flag 1 ——**叙述者代笔潜文本（最严重）**｜scene-4-4-1 末段
> 主角心里——同时听见**三层潜文本**：
> "他来救我。"
> "他来夺走我的死。"
> "他来替师弟走完那一炉。"
> ——三层都成立。三层都不能验证。

**诊断**：这一段把全篇承认轴的核心反讽用罗列句明文交付——它做了**读者应该做的工作**。"三层潜文本""三层都成立""三层都不能验证"——这三句话直接命名了 controlling-idea 锁定句。这是**类别失败：naming the theme**。前面 47 场全部用动作建立的张力，被这一段一次性下书。
**Rewrite direction**：
- **Action route（首选）**：删除三层引文，只保留主角的物理状态——"主角举断戒尺。他的右手腕不抖。他没有动。" 让读者自己同时听见这三层（读者已经被前 47 场训练好了）。
- **Indirect-line route**：保留一层最弱的（"他来救我。"），让其余两层留给读者补——保住三义性需要读者做工作而不是叙述者罗列。
- **Silent route**：整段删除，直接进入下一行"师傅的左手仍抚女儿脸"——动作承担一切。

### Flag 2 ——**叙述者旁注硬约束**｜scene-4-4-5 / scene-4-4-3 / scene-3-3-4
> 按 [[controlling-idea]] §7 违规 #11 严守——主角最后表情面无可读情绪。
> 按 [[cast-design]] §3.2 永不说出口硬约束——司徒明璋至 Resolution 不发言。
> 按 [[characters/sister-arc]] §3 段 3 锁定——师妹的"活+部分反损" arc 的第一段——她记得师兄，但叫不出"师兄"二字。

**诊断**：草稿期的工程注释残留——它不是叙事，是**写作者对自己说的话**。读者一旦读到"按 X §Y 严守"，潜文本立刻塌成元文本。这不是 on-the-nose 的人物台词，是**叙述者在场外承认自己在守约**。
**Rewrite direction**：定稿前**全数删除**所有 `按 [[...]] §x 锁定/严守/钉死` 形式的旁注。它们在 audit/handoff 阶段有用，在成品里是结构外露。

### Flag 3 ——**自由间接独白把师叔身世说得太清**｜scene-1-3-2 第 140-160 行
> 近千年前——他自己尚是不及百岁的少年。近千年前师傅二十二岁 physical……宗门联盟当年开应急联席议事三日，决议把师叔的丹经焚毁、墓碑立在屠村废址供村民唾骂。

**诊断**：周通玄一句"近千年前那个屠村的人"是漂亮的潜文本——主角"听见"但"不让自己想"。但紧接着叙述者**替主角想完了整段历史**——一百多字的宗门史话用自由间接句一次性下书，破坏了"压回去"的张力。这是 exposition smuggled in via 自由间接，**借主角脑子完成的叙述者解释**。
**Rewrite direction**：
- **Action route**：保留周通玄那一句，删除主角内心 80% 的史话回放——只留两三个不连贯的词（"屠村""师叔""三百口"）+ 一个物理反应（"他的右手按炉沿的指节又紧了一下"）——读者已经能拼。
- **Indirect-line route**：把宗门史话压到沉鸠或老长老后续场单独说出（让另一个角色承担），别让主角的脑子做信息倾倒。

### Flag 4 ——**轻度自我命名风险**｜scene-1-3-3 末"我要让那个废我的人在我面前再说一次话"
> **我要让那个废我的人，在我面前，再说一次话。**

**诊断**：边缘 case。这一句是**主角在心里没说出口**——已经是潜文本承担（不是台词）。紧接的"他自己以为这一句的含义是反炼立宗 / 他没看出来"那两行是 character-forger 文件里 dimension 4 "自觉 vs 潜意识"的精确文本化兑现——读者第一次能读出这条 gap，**功能正确**。但"他没看出来"这五个字是叙述者轻度越线（naming the gap）。
**Verdict**：**保留这一句心声 + 删"他没看出来"那一行**。让"他自己以为这是反炼立宗"作为陈述存在，gap 由动作（重复整戒尺/重复折纸/重复按断尺尾端）承担——读者自己拼。

## Subtext Wins（keep these）

### 1. 三循环复现的"敲断戒尺一下"——subtext through repetition
1-2-2 第 38 行"叮"——读者第一次只读到"奇怪的仪式"。1-3-3 末"按一下，松开"——重复但不解释。2-3-1 第 113-138 行——师傅第一次听见"叮"，**师傅的食指抖了一线**（外部 tell！读者读出师傅在反应），师傅接一句"你师叔当年起炉前也敲一下"——subtext 没有由叙述者说破，由**另一个角色的同形台词**说破（沉鸠先说过同一句）。这是教科书级 subtext through staging：叙述者从不解释，三种载体（沉鸠/师傅/主角自己）的同一句话在不同帧累积。

### 2. 师傅在 2-3-1 的三个 tell 同帧引爆——但全部沉默
- 听到"反"字前的预防性沉默 + 食指抖（他听见"叮"那一秒抖一线，第 117 行）——读者读出 "反"字 trigger 在身体上。
- 左手按右膝（第 78 行）——师傅"决定难做的事之前"的动作。叙述者只一句"这个动作主角三年前在乾元正脉见过"——**主角的视角承担解码，叙述者不背书**。漂亮。
- 师傅来时穿乾元宗主令袍**不戴玄色斗笠**——意象系统的精确反差（公开执法位 vs 私密入井位）。读者潜意识里读出"今天不是私事"。叙述者**完全不解释**。

### 3. 师傅化银砂死法对位师叔屠村——纯动作潜文本
4-4-3 第 818-960 行：师傅"从肢端开始散——指尖、脚趾、发梢"——这一段唯一的"on-the-nose"是叙述者那句"**与三百年前师弟沈砚屠村自焚的死法一模一样**"——这一句在此处**是必要的**（读者可能记不清师叔死法的物理细节）。但请注意：叙述者只说"死法一模一样"，**没说"师傅在替师弟还账"**——那一层留给读者。承认轴的最深一层（师傅死前那一字、师妹叫不出师兄、令牌落在主角脚边他不能弯腰捡）全部是 silent screenplay。

### 4. 师妹"师兄……"未完，与 Crisis 末主角"师……"对位——
4-4-3 第 136 行的"师兄……"是全篇潜文本最锋利的一帧。她说了开头但永远说不完；她在 23 年前说过完整版"师兄你不必照着方子来"。同一开头，反向兑现——**她活着是为了能开口但不能说完**。读者读出她是同谋而不是受害者。这一帧**完全没有任何叙述者解释（除了下一节又写了一句"按 sister-arc §3 段 3 锁定"——见 Flag 2，删掉它）**。

### 5. 4-4-5 最后一句"雾里没有任何东西在等他"
全篇 ending image 完美——它不是"他孤独"（naming），不是"他失去一切"（naming），是"雾里没有任何东西在等他"——景物层的陈述，**承担情绪的全部重量**。配合"戒尺举着未落 / 没看师妹没看炉没看天空 / 面无可读情绪"——silent screenplay 100% 通过。这一句是 controlling-idea 的诗意承担，不是其翻译。

### 6. 司徒明璋在 3-3-4 + 4-4-5 全程沉默
他在 3-3-4 "低头退半步、举乾元印未落、未下令拘押"——三个否定动作承担他整条 arc。在 4-4-5 主角走过他身侧"他没有抬头 / 他没有阻拦 / 他未发一言"——三个否定。他的 character revelation（"借了一辈子，借到腿都软了的人，至死不肯承认那是借的"）从未由任何人说出。完美。

## Layer Collapse Risks（text/behavior/desire 即将合流的位置）

### Risk A ——主角"决意"层
scene-1-3-3 末"我要让那个废我的人在我面前再说一次话"——文本（心声） = 行为（往宗门走） ≈ desire（被认）。三层接近合流。
**缓解策略已经在场**：紧接两句"他自己以为这一句是反炼立宗 / 他没看出来"——这两句把 desire（被再看一次）从 text（让师傅说一次话）和 behavior（去反炼立宗）撕开。**但"他没看出来"那五个字是叙述者强行撕开**——建议删掉这一行（见 Flag 4），让 gap 由后续动作（每场扫人群最外圈/敲断尺）自己承担。

### Risk B ——4-4-1 师傅入炉那一节
"师傅未通过任何仪式地直接把令牌让给主角"——这一句是 behavior 描述还是叙述者解码？接近合流。读者已经看到了"令牌落地咚一声、跨入炉口"两个动作；"未通过任何仪式"是叙述者的法理标签。
**Rewrite direction**：删除"师傅未通过任何仪式地直接把令牌让给主角"这一句，保留物理动作 + 司徒明璋的低头不抬头作为反应——读者通过司徒的"知道宗门律被破"的沉默读出。

### Risk C ——师妹意识反损的"主动"性
scene-4-4-3 第 270-310 行"师妹的意识——在反向核中和的瞬间——主动选择关闭语言能力。她主动让这一字未完。"——读者需要知道这是"主动"（这是她的 character revelation），但**直接命名"主动选择"**让她的最终维度从 subtext 变 text。
**Rewrite direction**：删除"主动选择"四字，仅保留"她张嘴——师兄——未完——闭嘴。她的瞳孔金黑流动稳定。她转头看父亲的银砂"——读者从"她不挣扎、不重试、平静地闭嘴"读出**这是她要的**。这是 character-forger 文件 §12 #4 已经标出的硬要求（"她对这件事的接受要让读者看出她是同谋而不是受害者"）——文本里这一帧需要更克制。

### Risk D ——3-3-4 师傅戒尺归位的双重命名
"师傅完成『我不裁决』的物理姿势。但这一姿势同时承担『我不认可』的物理义务。" ——这是叙述者把动作语言翻译两次。两次命名一次都过。
**Rewrite direction**：删除这两行，保留"戒尺没有落到反向丹核上 / 没有落到主角身上 / 没有落到祭炉沿 / 收回腰间挂位"四个否定动作——读者读出"既不裁决又不认可"是动作自己说的，不需要叙述者复述。

## Verdict per Character

- **沈砺**：**pass**。承认轴四级递降的承载体身份精确落地。"扫人群最外圈/敲断戒尺/写丹方像在临帖"三个标志动作在采样八场里每一场至少一现，读者三循环累积读出潜意识欲望的形状（被师傅再看一次）而主角自己始终读成"反炼立宗"。Want-to-need 的 gap 在采样场里始终保持——这是全篇最稳的潜文本承担。**唯一警告**：scene-1-3-3 末"他没看出来"那一行需删，让 gap 由动作自承（见 Flag 4 / Risk A）。

- **玄琢真人**：**pass**。三个物理 tell（食指抖 / 按右膝 / 绕第七块青砖）在采样场里都被使用：2-3-1 食指抖 + 按右膝 + 听"叮"；4-4-1 按右膝（最后一次）；3-3-4 戒尺举起未落。**叙述者从未替师傅解码**——他二十年研究反丹、对女儿的爱、对师弟的债，全部以"看胸口印记 / 不戴斗笠 / 抚女儿脸 / 化银砂"承担。维度 1（公开拒斥 vs 私下研究反丹）在采样里完全是 subtext。**唯一警告**：4-4-1"师傅未通过任何仪式让令牌"的法理旁注需删（见 Risk B）。

- **玄漪**：**pass**。她在全篇就三句话的硬约束**精确兑现**。第一句"师兄原来你回来了"（Seq 3.4.3）、第二句"父亲您当年炼凝魂丹也是反着炼了一半的"（4.2.1）、第三句"师兄……"（4-4-3）。第三句未完 + 4-4-5 张嘴叫不出闭嘴——她的 subtext 是她的沉默 + 她的开口姿势 + 她的闭嘴动作的合一。**唯一警告**：4-4-3 删"主动选择关闭语言能力"那一句（见 Risk C），让动作自承。

- **沉鸠**：**pass（采样未集中触及但 spillover 充分）**。他在 1-3-2 / 2-3-1 通过"你师叔当年也敲一下"七字承担传承场的潜文本——叙述者从不解释他为什么这样说，读者通过他与师傅同句异源（两个不知道对方说同样的话的人）读出"主角的潜意识仪式不是他独有"。功能干净。

- **司徒明璋**：**pass**。他在 3-3-4 + 4-4-1 + 4-4-5 全程**只有否定动作**（没抬头/没阻拦/未下令拘押/没撤回令牌/未发一言）。他的 character revelation（"借了一辈子但至死不肯承认那是借的"）从未由任何人说出，由他不动 + 不说 + 不阻拦 + 不抬头四个否定承担。这是次主角的标杆潜文本。

## Final Note

这部作品的危险不在 dialogue on-the-nose（几乎没有），在**叙述者注释 on-the-nose**——它把工作期的 controlling-idea / character-arc / hard-constraint 旁注误带进了 final draft。删除约束旁注（按 [[X]] §Y 锁定/严守）+ 4-4-1 末"三层潜文本"罗列 + 1-3-2 主角脑内宗门史话 80% + 几处"主动选择 / 未通过任何仪式 / 我不裁决"双重命名——这些都是 30 分钟改稿可清理的项。底层人物 subtext 结构稳。
