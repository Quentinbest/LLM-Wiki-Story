# §0 调性 anchor — 凡人修仙传

本世界遵循"凡人调性"硬约束：

1. **物质感优先**：每一颗丹要药材 + 火门 + 时间 + 灵石成本；每一道法宝要血脉 / 灵气 / 维护费用；没有"凭空一道剑光"
2. **计算先于豪情**：主角在每场起炉前都会计算（药材是否够 / 账本余额 / 时间 / 风险）— 不写"他心中一横起炉"
3. **升级有过程感**：境界突破不是 instant / 反丹道立宗不是一夜成名 / 周通玄不是一招败
4. **反派也务实**：周通玄 36 年首席不是傲慢二流货 / 联盟有体制逻辑 / 师傅有避险费的具体重量
5. **爽点克制**：让读者满足但不侮辱智商 — 主角不必"一掌灭七大宗门" / 但每一场胜要付清账本

# §1 修真境界体系（六阶 / 内丹凝聚为锚）

## 1.1 六阶基本结构

```
炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 合体期
```

寿元对应：凡人 60-80 → 炼气 100-200 → 筑基 300-400 → 金丹 600 → 元婴 1200 → 化神 2400 → 合体 4800-5000

出处：参《抱朴子·内篇》"炼形—炼气—炼神"三段说 + 《周易参同契》"还丹"之论 + 现代修真小说沿用萧潜《缥缈之旅》六阶基准。

**典型突破年龄区间（按资质 / calendar 历法年龄）**：

| 境界 | 一般资质 | 稍优资质 | 极优资质 | 千年一遇天才 |
|---|---|---|---|---|
| 筑基 | 100–180 | 60–100 | 30–60 | < 30 |
| 金丹 | 200–350 | 260–400（中位）| 100–260 | 60–100 |
| 元婴 | （多数无望）| 400–580（顶级）| 300–500 | 150–250 |
| 化神 | —— | —— | 凤毛麟角（800+）| 400–700 |
| 合体 | —— | —— | —— | （万年一遇）|

**容颜规律**：筑基后容颜停留（修为越高越接近停留时的形态）；化神后容颜可由修为本人主动调整（多数化神选定一副"中年长须"形态以示威仪）。**寿元接近上限者长须自青转白**。

> 由此推算：化神前期者最低 calendar age 也在 **700-800 岁** 以上；千年级化神为本世界**当世活跃者中之最强**。本书化神 3-5 位皆已闭关百年（calendar 当下 1500-2000+ 岁），师傅玄琢真人 **calendar 998 岁** / 化神前期 / 未闭关——是当世唯一活跃在外的化神。

## 1.2 各阶详解

### 炼气期（一至十三层）
通过吐纳引天地灵气入体，淬炼经脉。十三层为大圆满。突破到筑基需"凝丹海"——以丹田作池，凝液态灵气。

**突破必备**：服用筑基丹（七品级别 / 由六品丹师以上炼制）。

### 筑基期（初/中/后/大圆满）
丹海固定为筑基海，经脉变质能容更浓灵气。寿元三百载。突破金丹需在体内凝出固态金丹。**凡人修仙者的第一道大坎**——筑基死亡率约六成。

**突破必备**：服用**结丹真丹**（五品级 / 由五品丹师以上炼制 / 配合一颗下品灵石）。

### 金丹期（一品至九品 / 初/中/后/大圆满）
体内灵气凝为一颗金丹，可日运吐纳。寿元六百载。**金丹品级**对应丹师能炼制的丹品阶——这是炼丹与境界之间不可分割的关联（详 §2.2）。

主角被废前：**99 岁**（calendar）成金丹三品圆满 = 三品丹师 / 乾元六脉首席。**这是宗门联盟近百年最年轻的金丹三品**（同期最年轻者多在 150-200 岁圆满）/ 千年一遇天才级速度。容颜表观停留在 早三十。

**突破必备**：服用元婴丹（二品级 / 仅二品以上丹师能炼 / 配合三颗中品灵石）。

### 元婴期
金丹破开，结出元婴。元婴可离体百丈。仙俗界限。寿元一千二百载。当世七大宗门各一至数位元婴期。

**突破必备**：服用化神丹（一品级 / 当世无人能炼新丹 / 仅遗留古丹）。

### 化神期
元婴长成形神，可游天地之间。寿元二千四百载。本作品中尚有 3-5 位化神期长老（皆已闭关百年）+ **师傅玄琢真人化神前期但未闭关**（详 §3.1.1）。

**突破必备**：合体丹（仙器级 / 万年不出一颗）。

### 合体期
元婴与肉身合一，无形遁。寿元四千八百至五千载。本作品中无人在场（皆已飞升或入沉睡）。

## 1.3 主角的特殊状态：**破丹反炼态**

被废丹田 = 金丹散尽 + 经脉点断 → 但反丹气可逆向重凝。

**破丹反炼态**是反方丹道独有的"伪境界"：金丹散尽之后用反向丹气在丹田原位凝起一颗**反向丹核**——形态上是金丹，物理是反的。

- 主角现在的反丹核约金丹初期容量（远低于被废前三品）
- 不能再炼正方丹（自封丹田反向之后**永远不能**）— 这是 Act 4 Crisis 的物理基础
- 但反方丹术上限高于他被废前的三品（因为反向丹道宗师无人达过）
- **反向丹核也可作内丹反哺反向外丹**（详 §2.4）

# §2 丹道体系 — 内丹 / 外丹 reconciliation

> 核心 design 来源：《云笈七签》传统区分内丹与外丹。

## 2.1 内丹 vs 外丹（**核心法则**）

```
┌─────────────────────────────┐
│ 内丹 — 修真者自身金丹         │
│ 修为层的物理载体 / 不可外赠   │
├─────────────────────────────┤
│ 外丹 — 炼丹师炉中产出         │
│ 服用品 / 流通 / 经济          │
└─────────────────────────────┘
```

**宇宙法则**（不可逾越）：

> 修真者的内丹**必须用外丹辅助**才能突破到下一阶。

具体物理：
- 服用外丹后体内灵气暴增 / 需用自身内丹催化
- 内丹纯化 + 外丹力量 → 突破成功
- 没有外丹 / 无法突破 / 永困当前境界

**这就是炼丹师在修真界的政治根基**——他们直接操纵修真法则的"突破环节"。

## 2.2 金丹品级 = 丹师品级（不可分割）

修真者自身的金丹品级 = 他作为丹师能炼制的外丹品阶。

这是因为：**炼丹时，丹师以自身内丹之力反哺炉中外丹**——内丹的品级决定外丹的上限。

| 自身金丹品级 | 能炼外丹品阶 | 当世人数 | 代表 |
|---|---|---|---|
| 一品（无人达到）| 苍生大丹 / 化神丹 / 合体丹 | 0 | —— |
| 二品 | 元婴丹 / 凝魂丹 / 续命真丹 | 3 人 | 师傅玄琢真人 / 太乙青琅前任 / 南斗御药老祖 |
| 三品 | 结丹真丹 / 九转还魂小成 / 反方三品 | 12 人 | 主角（被废前）/ 周通玄昔年 |
| 四品 | 通脉丹 / 明心丹 / 易筋丹 | 30+ 人 | 司徒明璋 |
| 五品 | 起死回生丹 / 大补丹 / 筑基丹 | 100+ 人 | 周通玄今年 / 联盟首席丹师等级 |
| 六品-九品 | 基础丹 / 辟谷丹 / 战斗丹 | 数千 | 四座师兄 / 联盟应急队 |

**关键 reconciliation**：周通玄作为五品丹师为何位高？因为五品丹师炼制**绝大多数修真者每日所需**的丹药（疗伤 / 战斗 / 辟谷 / 基础突破丹）/ 他控制大宗流通。一品 / 二品丹师极少 / 他们炼的是高阶突破丹（元婴丹 / 化神丹）/ 数量极少但每颗价值连城。

## 2.3 炼丹的多维度功能（远超经济）

| 功能 | 丹药类型 | 谁需要 | 备注 |
|---|---|---|---|
| **境界突破** | 筑基丹 / 结丹真丹 / 元婴丹 / 化神丹 | 所有突破修真者 | **universal law / 不可绕过** |
| **战力 enhancement** | 战斗丹 / 暴血丹 / 怒火丹 | 战时 | 短时增加战力 / 多服伤身 |
| **法宝供能 / 开光** | 开光丹 / 法宝养护丹 | 法宝持有者 | 法宝长期使用需丹药供能 |
| **阵法启动 / 阵眼** | 阵眼丹 / 大阵守护丹 | 阵法师 | 高级阵法需丹药做阵眼 |
| **功法突破辅助** | 各功法配套丹 | 功法修炼者 | 突破功法各层需特定丹药 |
| **疗伤恢复** | 疗伤丹 / 续命真丹 | 战后 | 高品续命丹只有元婴+ 服用有效 |
| **改变体质 / 洗髓易筋** | 洗髓丹 / 易筋丹 | 突破前 | 体质不够无法承载更高境界 |
| **续命** | 续命丹 / 长生丹 | 元婴+ 寿元有限者 | 老一辈修真者的命脉 |
| **神魂修炼** | 凝神丹 / 锁魂丹 | 神魂修真者 | 神魂之路的辅助 |
| **悟道辅助** | 悟道丹 / 通玄丹 | 临突破者 | 助进入悟道状态 |
| **辟谷** | 辟谷丹 | 闭关者 | 修真者不食人间烟火的替代 |

→ 炼丹覆盖修真者**所有维度**。但每一维度都还需要其他 specialty（法宝铸造 / 阵法 / 功法 / 等）的协作。这就是为什么七大宗门**互相依赖**（详 §3）。

## 2.4 顶级炼丹师特别——内丹反哺外丹

**一品 / 二品丹师**可以**用自身金丹之力反哺炉中外丹**——这是为什么一品 / 二品丹师极少：

- 一颗顶级外丹（元婴丹 / 化神丹）需要消耗**自身金丹力的 1/100 至 1/10**
- 顶级丹师炼一颗顶级丹后需**闭关数十年**恢复 + 成功率不高
- 但炼成的外丹可让一位元婴期突破化神 — 价值连城
- 这就是政治权力：**一品 / 二品丹师是修真界寿元延续的根**

### 各品级炼丹的闭关时间 / 成功率（解释当世稀有性）

| 丹师品级 | 当世人数 | 炼对应品级丹 闭关时间 | 成功率 | 一生总产 |
|---|---|---|---|---|
| **一品** | 0（百年无人）| 数百年 / 无新丹 | — | — |
| **二品** | 3 人 | **30-50 年** | 30-50% | 一生 1-3 颗 |
| **三品** | 12 人 | **5-10 年** | 50-70% | 一生 5-10 颗 |
| **四品** | 30+ 人 | **1-3 年** | 60-80% | 一生数十颗 |
| **五品** | 100+ 人 | **数月** | 70-90% | 一生数百颗 |
| **六品-九品** | 数千 | 数日 - 数月 | 80-95% | 一生数千颗 |

**关键解释**：
- 元婴丹 / 化神丹的稀有性 = 炼制周期长（30-50 年）+ 成功率低（30-50%）+ 二品丹师极少（3 人）
- 一位元婴期若想突破化神，可能要等十几年才能买到一颗化神丹（市价以上品灵石计 + 排队）
- **这就是为什么修真界元婴期及以上的高阶修士非常少**（七大宗门每宗仅一两位元婴老祖 + 化神 3-5 位皆闭关）
- 而五品丹师炼基础丹只需数月 + 成功率高 = 修真界**普通修真者每日所需丹药供应**充足

### 主角的反方丹道作为颠覆性

- 主角的反向丹核 → 反哺反向外丹
- 反向外丹由反向内丹催化 / 正方丹师炼不出来 / 必须反方丹师
- 这就是为什么反方丹道几乎失传——没有反方内丹（反向丹核）就炼不出反方外丹
- **主角作为反方派的"二品级反方丹师"潜力**：一旦立宗后可独立成派 / 不依赖乾元
- 反方派如能炼出反方版的元婴丹 / 化神丹（即使周期同 / 成功率不同）— **将打破乾元的元婴 / 化神突破垄断**

## 3.1.1 乾元主导的修为基础

乾元正脉之所以能主导联盟，物质基础是**修为差距**：

- **玄琢真人化神前期 / 二品丹师** — 当世唯一活跃在外的化神期高阶（其他 3-5 位化神长老已闭关百年）
- **乾元正脉自身阵容**：化神 1 位（师傅 / 998 岁）+ 元婴中期 2 位（司徒明璋 712 岁 / 周通玄 623 岁）+ 长老团元婴初期数位（400-500 岁）
- **对比其他宗门**：太乙 / 玄武 / 南斗都是元婴中-大圆满老祖 + 长老团元婴初期 / 整体战力**弱于乾元约半个境界**
- **政治权力的物质来源**：乾元一宗能 1v2 任何两宗的合力 / 这就是为什么联盟存在而其他宗门不轻易公开挑战
- **师傅"出关"主导联盟而不闭关化神** = 他的避险费（师弟屠村 + 女儿玄漪 + 反丹道 lifelong battle）让他不允许自己闭关 / 这是 character 维度的物质根据。**他化神后已 200 余年未闭关进阶 / 寿元仍在 2000+ 余里（化神寿元 2400）** / 但本就该闭关推到化神中期的 200 年全用在守正与赎罪上。

## 3.2 三派分立的物质基础

```
┌──────────────────────────────────────┐
│ 正统三宗（乾元 + 太乙 + 南斗）           │
│ 守"丹道为核心"教义 / 控制顶级丹药供应    │
```

|---|---|---|---|

## 6.4 反方派的**法则唯一性威胁**

> 反方派颠覆的不只是教义、不只是经济，**是修真法则的唯一性**。

具体来说：
- 三百年来乾元正脉立宗 = 修真法则的"丹道为核心"+"乾元为法则锚"
- 反方派如果能炼出反方版的**突破丹 / 续命丹**（即反方对应的元婴丹 / 化神丹）→ 反方派可以**独立于乾元正脉生存** / 不再依赖乾元的外丹

- 这就是为什么联盟反对反方派 / **不只是教义之争 / 是法则唯一性之争**

→ 主角的反丹道立宗 = **建立第二个修真法则锚** / 这是为什么这件事的政治重量这么重

# §8 修真者整体战力公式

修真者整体战力**不只是炼丹 / 不只是法宝 / 不只是阵法**——是综合：

```
整体战力 = 自身境界（修为）
       × 法宝品级 + 法宝养护程度
       × 阵法掌握 + 阵眼丹品级
       × 功法层级 + 功法配套丹
       × 实战经验
       × 灵石/丹药储备
```

但所有 dimension 都受**炼丹这一根本法则**约束：
- 自身境界突破必须服外丹（universal law）
- 法宝长期使用需开光丹（炼丹师炼）
- 阵法启动需阵眼丹（炼丹师炼）
- 功法突破需配套丹（炼丹师炼）
- 寿元延续需续命丹（炼丹师炼）

→ 这就是为什么炼丹是**法则锚**而不仅是 specialty。

七大宗门各自掌握一关键 dimension 的优势 / 但都需要乾元的炼丹支持 / 这是联盟存在的物质基础。
