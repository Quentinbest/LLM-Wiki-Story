## §1 审计标准（exposition as ammunition 三要素）

McKee Ch. 15 硬规则：每一条铺陈信息必须在一场冲突中以**弹药**形态发射——

- **谁说**（who fights to reveal）：有动机揭开 / 抛出此信息的角色，且他在该场景里**正在战斗**（不是讲解）。
- **谁听**（who fights to hear / extract）：有动机听到 / 抗拒听到此信息的角色，他听到此信息后**场景价值电荷必须发生跃迁**。
- **谁阻挡**（who fights to conceal / weaponize / withhold）：有动机隐瞒 / 武器化此信息的角色，他不让信息 leak 出来或他把信息当刀使。

三要素缺一，信息退化为 lecture（旁白讲解 / 角色独白 / 倒叙专章 / 转场字幕 / 脚注）。

弹药评级：
- ✅ **smuggled**：三要素齐备 + 场景价值电荷因发射而跃迁
- ⚠ **partial**：信息被场景包装但战斗对位弱（谁说 ✓ 谁听 ✓ 谁阻挡 ✗，或反之）
- ❌ **lecture**：以旁白 / 内心独白 / 倒叙专章 / 转场字幕 / 脚注形态出现
