# Scene 3.3 — 天台

## 1. Frame
- **Where/when**: 停车场七层屋顶。下午 17:50。风。城市边缘看得见。日光是斜的，过一会儿橙起来。屋顶有几个通风口、一个生锈的旗杆、一道矮墙——人可以靠。
- **Who's in it**: 陈默；苏寒。苏寒比他先到，没站在矮墙边——她站在屋顶中央，手里只有一个牛皮纸文件夹。她没拎包。
- **Spine position**: **PC-5 的兑现场景**——脊椎最内点。

## 2. Objectives & antagonism
- **POV scene objective**：来，听，回去——**不被说服**。
- **Forces of antagonism in this scene**：
  - 苏寒本人——但她**不施压**。这是她的危险性。
  - 内部：他已经爬上来了。爬上来这件事本身就是 70% 的承诺。
  - 文件夹——一份被改写过的他的过去。物证级别的诱惑。
- **Other characters' secret objectives**：苏寒——让他看见**他自己**已经做过的事；不要求他承认；中段停止说话，把沉默还给他。

## 3. Value
- **Named value**：自我命名 / 被命名（self-naming / being-named）。
- **Opening charge**: 内部 −2 / 外部 +1
- **Closing charge**: **内部 −3（最深点）/ 外部 +2**——他没拒绝。

## 4. Beats（7 节拍——本场是 PC-5 兑现，必须充分）

1. **Action**：陈默走到苏寒对面，矮墙在两人之间不远。**Expected reaction**：苏寒会先开口。**Actual reaction**：苏寒先把文件夹放在他们之间的通风口顶上，然后退一步。*(gap: 她不递给他——她**放下**。这是她的策略：让物件自己说话。)*

2. **Action**：陈默打开文件夹。**Expected reaction**：是合同、是招募书。**Actual reaction**：是 72 小时内他做过的事的清单。每一条用魔尊视角写：
   - "**3 日凌晨 02:14**：'墨渊'拒绝在小区门口对一只受伤的流浪猫施暴，反而把自己的雨衣盖在猫身上——**伪善·二级**。"
   - "**4 日 19:30**：'墨渊'将一份送错地址的外卖自费补送至正确住户，并未追讨费用——**布局·三级**。"
   - "**5 日 13:22**：'墨渊'在便利店主动让一位老人先结账——**收买人心·一级**。"
   清单有十几条。每一条都是**他真的做过的事**。每一条都被翻译成了反派编码。*(gap: 他来这里是要被招募——他得到的是**他自己的善行被认领**。)*

3. **Action**：陈默抬头。**Expected reaction**：苏寒在等他反应。**Actual reaction**：苏寒在看城市另一边——夕阳那边。她**没在看他**。*(gap: 她不施压。她给他空间。这比施压可怕得多。)*

4. **Action**：陈默掏烟。他平时不抽——这是他的应急烟，一包能放半年。他点了一根。**Expected reaction**：苏寒会用这个空隙说话。**Actual reaction**：**苏寒不说话**。她也没看他。她只是站着。*(gap: 沉默被还给了他。**这是全书第一次有人允许他在自己的决定面前慢慢站着。**)*

5. **【一根烟的时长】**——本节拍不写他在想什么。**只写时长**。
   - 烟在他指间燃过第一段灰。
   - 风把灰刮走了一半。
   - 远处有救护车的声音，过了一会儿没了。
   - 苏寒动了一下，挪了脚的位置。没有说话。
   - 烟燃到一半。
   - 一只鸽子飞过屋顶矮墙。
   - 烟燃到三分之二。
   - 陈默把烟掐了——还有最后三分之一。**他掐早了**。
   
   *(gap: 一根烟的时长不在他的内心里——在读者的耐心里。读者会读出"他在挣扎"。回读时：他没在挣扎，他在**第一次站着等自己**。)*

6. **【苏寒的诊断台词】**——苏寒不转过来看他。她看着夕阳那边。她说：
   > **"你做的事，世界终于看见了。只是它叫你魔尊。"**
   
   说完她**没看他**——她**看了别的**。一只信号塔上的红灯。她没等他回应。
   
   *(gap: 表面：反派的诱惑说辞。真实：**主控思想被一个人当面说出**。他听见的，是诱惑；他听不见的，是诊断。)*

7. **Action**：陈默开口。**Expected reaction**：他会说"我是陈默"或"我不是"。**Actual reaction**：他说：
   > **"你们什么时候要我？"**
   
   苏寒终于转过来。她没笑，没点头。她只是说："**不急。**"——和电话里那句一模一样。然后她拿起文件夹，没合上，**留在通风口顶上**，转身走向屋顶门。
   
   *(gap: 文本上这不是"是"。这是个问句。但读者会读作"是"——**这是误导峰值**。回读：他问的是 timing；他没接受 identity。但**他也没拒绝**。)*

## 5. Turning Point
> 第 7 节拍：陈默说"你们什么时候要我？"
>
> **不可逆性**：这是他第一次在被命名为魔尊的情境里**没有否认**。从此他不能假装那次否认还可以延后。他停止说不——这是 [[negation-of-the-negation]] 走上舞台的瞬间，正是脊椎的最内点。
>
> **misdirection peak**：文本（一个询问 timing 的问句）vs 读者期待（一个 yes 的伪装）之间的裂口，就是误导的最高承载点。

## 6. Subtext
**陈默的潜文本**：他这一辈子没有人问过他"什么时候有空"。他每天被派单系统告知什么时候去哪里。他问"你们什么时候要我？"——这句话表面是接受魔尊招募；潜文本是**"我想知道有没有人在某个具体时间需要我作为一个具体的人**。"他借了魔尊的名义问了陈默想问的问题。这就是为什么这句不是 yes——但它也不是 no，因为它**真的在问**。

**苏寒的潜文本**：她整场的动作都在做一件事——**给他空间**。她不递文件夹、她不催、她不看他、她诊断完不等回应。她在表演"如果你来，我们这边的世界是这个节奏"——这个节奏本身就是诱惑的内容。她不是在卖魔尊；她在**演示陈默从未拥有过的尊重**。

裴鸣笔记：苏寒的台词必须没有"坏人腔"。她说"你做的事，世界终于看见了"——语气接近**老师在批改一份试卷时随口说出的判断**。没有戏剧色彩。这是关键。
