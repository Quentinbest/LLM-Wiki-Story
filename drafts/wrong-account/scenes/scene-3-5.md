# Scene 3.5 — 花坛与 Moleskine

## 1. Frame
- **Where/when**: 小区中庭。一个种着月季的水泥花坛——边沿可以坐人。清晨 07:00。阳光开始变实。两个早餐袋之间放着——一个用塑料袋折成的小垫子（**林晟做的**——他怕花坛边脏）。
- **Who's in it**: 陈默、林晟。
- **Spine position**: **PC-6 兑现**。本场的核心动作是 Moleskine 的翻页和"豆浆 vs 咸豆浆"。

## 2. Objectives & antagonism
- **POV scene objective**：吃完，离开，**不让这件事看起来像一次会面**。
- **Forces of antagonism in this scene**：
  - 林晟的**完整方案**——他真的准备了一份。这是预谋的温柔。
  - Moleskine 本身——一个物件级别的对手。它存在的意义就是"被打开来给陈默看"。
  - 内部：陈默在天台之后已经下沉到 −3；本场不能再让他下沉，**但能让他理解他下沉的位置**——这是更冷的一种不可逆。
- **Other characters' secret objectives**：林晟——执行"人道主义方案"。他真心认为这是体面的折中。他自己甚至觉得有点超前——"我们这边的高层不愿意这么做，是我说服的"。这是少年的虚荣，无害的虚荣，**因此更难处理**。

## 3. Value
- **Named value**：被理解 / 被处理（understanding / processing）。
- **Opening charge**: 内部 −3 / 外部 +2
- **Closing charge**: 内部 **−3 但维度改变**——他没下沉到 −4，因为下沉空间已用完；他**理解了自己在哪里**：温柔的版本仍然是同等的抹除。外部维持 +2（林晟仍然把他当主角）。

> 注：本场的翻转是**质性的**——价值数值不动，但价值的**性质**变了。Chen Mo 之前以为"软招"vs"硬招"是两种东西；本场结束他理解它们是**同一件事的两种界面**。这是 McKee Ch.10 允许的"质性翻转"——一个值的名称未变，但其**意义在场景内被重新定义**。

## 4. Beats（7 节拍）

1. **Action**：林晟吃完，把油纸叠成小方块。**Expected reaction**：进入正题。**Actual reaction**：林晟先**整理两个人的垃圾**——他把陈默的油纸也收过去叠了。*(gap: 关爱。无懈可击的关爱。)*

2. **Action**：林晟从书包里掏出 Moleskine——**深棕色，皮面，他用的是一根橡皮筋束着**。**Expected reaction**：他要审问。**Actual reaction**：林晟**先翻开给陈默看**——主动展示。他翻到中段一页，递过来。*(gap: 反派给被审问者看自己的笔记——这是一个**示好动作**，但也是一个**支配动作**——是他在决定要给陈默看什么。)*

3. **Action**：陈默看那一页。**Expected reaction**：是他被审问时说过的话。**Actual reaction**：是一个**重新评估表**。表头写着——
   > **"对象 #M-7（魔尊·墨渊）行为模式再评估"**
   
   下面是栏目：
   - 攻击性：低
   - 配合度：中
   - 谎言征兆：低
   - **结论**：**"可能是退化型魔尊——其残存人格可能值得保留。建议：人道主义收编。"**
   
   *(gap: 表层：林晟在替他争取生路。底层：**整张表里没有"陈默"两个字**——只有 #M-7。林晟的进步是**给一个编号加上了人道关怀**，而不是**问那个编号叫什么**。读者第一遍读：林晟变好了。回读：**他从未变好**。)*

4. **Action**：林晟翻到下一页——是他自己写的"人道主义投降方案"。**Expected reaction**：陈默会反对。**Actual reaction**：林晟一条一条解释——他的语气是**一个高中生在年级展示自己写的小论文**。他说："这一条是关于您退场之后的住所安排——"
   - "我们这边的高层不愿意安排这么细。是我说服的。"
   - "您退场后还是可以送外卖的——如果您还想送的话。"
   - "我们不会公开您之前的事——给您留一个普通生活的可能性。"
   
   *(gap: 林晟在说"普通生活"——但他没意识到陈默**本来**就在过普通生活。林晟以为他在恢复陈默的"普通"，实际上他在**赦免**一个本来没罪的人去过本来就在过的生活。)*

5. **Action**：陈默打断不了。他想打断，但他不知道在哪一句打断。他低头看自己手里的豆浆杯——**甜豆浆**。**Expected reaction**：他喝一口。**Actual reaction**：他**没喝**。他只是看着那个杯子。*(gap: 杯子是种植——下一节拍要兑现。)*

6. **【豆浆台词】**——林晟翻到下一页之前，突然抬头，**带着真诚的关心**说：
   > **"对了——墨渊先生，您愿意吃豆浆还是咸豆浆？我下次记得。我们这种会面可能还会有几次。如果您喜欢咸的，我提前去买。"**
   
   陈默看着杯子。**他没回答**。停了三秒。林晟以为他没听见，**又问了一遍**：
   > "墨渊先生？"
   
   *(gap: **这是 PC-6 的兑现瞬间**。林晟把"我会记得您的早餐偏好"作为善意承诺；陈默听见的是**"墨渊先生"被叫了第二遍**——林晟从未问过他叫什么。**最温柔的版本仍然是同等的抹除**。)*

7. **Action**：陈默抬头。**Expected reaction**：他会说"我叫陈默"。**Actual reaction**：他**说**：
   > **"咸的。"**
   
   就一个词。然后他站起来。他没把豆浆喝完。他把杯子放在花坛边——杯子在那儿，**像一个小小的纪念碑**。林晟点头记下来：他**真的**在 Moleskine 上写：**"#M-7 偏好：咸豆浆。"** 不抬头。
   
   *(gap: 陈默说"咸的"——文本上是回答口味问题；潜文本是**他放弃了说"我叫陈默"这件事**。他选择**接受被叫"墨渊先生"**作为这场对话的代价——因为他理解了：在林晟的世界里，"陈默"这个词进不去。回到 Moleskine 里的，永远是 #M-7。读者第一遍读：他妥协了或在配合。回读：**他理解了——并且离开了**。)*

## 5. Turning Point
> 第 6–7 节拍连结：林晟问"墨渊先生，您愿意吃豆浆还是咸豆浆？"——陈默看着豆浆杯——停三秒——林晟又问一遍"墨渊先生？"——陈默回答"咸的"。
>
> **不可逆性**：陈默在本场结束时**接受了"墨渊先生"这个称呼作为社交单位**。他没纠正林晟。**他不再为'陈默'这个名字辩护**。从此他不能假装林晟会自己想起来问——他**已经知道**林晟不会，并接受了这个事实。
>
> 这是 [[negation-of-the-negation]] 反方向的镜像——苏寒在天台让他**走向**接受魔尊；林晟在这里让他**理解**陈默之名在两个世界里都没有位置。

## 6. Subtext
裴鸣笔记：
- 林晟的 Moleskine 必须被物理展示。读者要看见那一页——但读者**不会注意没有名字**。这是关键：让读者看页面，但用 #M-7 取代名字。读者读"M-7"读过去，不停顿。回读时才发现。
- 陈默看豆浆杯那三秒不写他在想什么。**只写他的脸的方向，他的手的位置**。让读者填那三秒。
- "咸的"这一个词必须是全场最短的台词——和林晟那段长长的方案介绍形成节奏对照。短句收刀。

陈默的潜文本：他理解了苏寒和林晟在做同一件事——给他一个名字，没人问那个名字对不对。苏寒给的是"魔尊"，林晟给的也是"墨渊"。**两边都是 #M-7**。他终于知道为什么"陈默"那么累——因为这个名字没有人方说。

- [x] Earned through conflict（温柔本身就是冲突——它让他无法反击）
- [x] Turning Point is locatable（第 6 节拍 / "墨渊先生？"的第二次叫）
- [x] Scene objective is playable（吃完离开——他做到了，但代价是放弃了名字辩护）
- [x] Gap is present（每节拍 / 第 3 节拍是最大 gap——表层是慷慨，底层是无名）
