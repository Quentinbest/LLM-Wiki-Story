# Scene 3.8 — 救护车

## 1. Frame
- **Where/when**: 陈默公寓楼前空地。14:10。救护车（北京急救 120）停在单元门口。日头偏西一点，但还是亮的。地砖上有一小滩水（救护车下来时踩湿的），有一只钥匙——李奶奶上楼时掉在 4 楼的——陈默捡了下来攥在手里。围观的邻居只有两个人——一个二楼的中年男人（穿拖鞋出来），一个一楼小卖部的店主。其他户**没有人下来**。
- **Who's in it**: 
  - 陈默——外卖服还没换，肩膀有点尘
  - 李奶奶——担架上，左肩绑着急救纱布，意识清醒，神情茫然多于疼痛
  - 急救医生（女，40 出头，姓什么不重要——她是流水线的一环）
  - 急救员（男，年轻，抬担架）
  - 二楼男人 + 小卖部店主——背景
  - **远处一名玄机阁观察员**——在单元门对面的电动车棚旁，**抱着手机**——他在向苏寒回报
- **Spine position**: **Act III 出幕**。脊椎转折点 PC-7 的最终一刻——救护车关门、老太太不认得、陈默握门把。

## 2. Objectives & antagonism
- **POV scene objective**：**让她平安上车**。次级：被她认出来。（他不会承认第二个目标——但他的眼睛会承认。）
- **Forces of antagonism in this scene**：
  - 急救流程——医生在按表问话，**没有人为陈默留时间**
  - 李奶奶本人——她**清醒**，她**看着他**，她**没认出来**
  - 远处的玄机阁观察员——他不动，但他的存在让陈默理解"我仍然在被记账"
  - 内部：陈默的眼睛——他不能演"无所谓"，但他也不能崩溃；他必须在一个普通邻居的身份里完成这一刻
- **Other characters' secret objectives**：
  - 李奶奶——她想知道这是哪个邻居在帮她（她**真的不记得**陈默是谁——她记得"那个送饺子的"，但她从未把"送饺子的"与一张具体的脸对齐）
  - 急救医生——填表，问亲属电话
  - 玄机阁观察员——报告"墨渊"是否还在场（他不需要走近——力场已经被记录）

## 3. Value
- **Named value**：**被看见 / 不被看见**——脊椎主价值在此场以最纯净形式锚定。
- **Opening charge**: 内部 −3 / 外部 +2
- **Closing charge**: 内部 −3（**钉死，不可悬置**）/ 外部 +2（**钉死，由观察员的存在确认**）

> 注：本场作为 Act III 出幕，价值数值在场内**不动**——它的功能是**锚定**前几场积累的 −3 / +2，让这个状态变得**不可再含糊**。这是 McKee Ch.9 [[act-rhythm]] 允许的"幕终静像"——幕末场景以最低数值动作完成最高情境承重。

## 4. Beats（6 节拍——本场要慢，给读者呼吸，是 Act III 整个深潜的收束）

1. **Action**：急救员把担架从 4 楼抬下来——陈默在前面**扶着担架边缘**走。**Expected reaction**：他会说点什么安慰她。**Actual reaction**：他**没说话**。他只是扶着。**李奶奶看着天花板**——一节一节灯管经过她的视野。*(gap: 他想说"奶奶您慢点"——这是他送外卖时对老人说的话。但他没说。他不知道自己为什么没说。**他不想再被她认作'那个声音'**——如果她连脸都不认得，那么连声音都不要让她有机会再次不认得。)*

2. **Action**：到楼下，担架上车前。急救医生问李奶奶："家里人电话？"。**Expected reaction**：李奶奶报电话。**Actual reaction**：李奶奶**报不全**——她记得儿子号码的前八位。她皱眉。**她转头——看陈默**——像在问"你帮我想想"。*(gap: 她**向他求助**——但**不是把他当熟人**——她只是因为"这个人在我身边"。读者第一遍读：感人。回读：**她身边的人本来一直就是他——但她到现在才把他当作'身边的人'**。)*

3. **Action**：陈默看她。**Expected reaction**：他会说"我帮您联系"。**Actual reaction**：他**抬手——拿出自己的手机**——他**有**李奶奶儿子的电话。**他送过几十次餐，留过一次儿子的电话作为应急联系人**——那是李奶奶有一次出门把门反锁了，托他打的。**他直接拨了**。"喂，您母亲在 121 路救护车上。她没大事，肩膀擦伤。地址 [略]——医院 [略]——您过去。"挂了。**李奶奶看着他打电话的整个过程**——她的眼神是：**这小伙子怎么知道我儿子的电话？** *(gap: 这是本场最锋利的 gap。**他对她的世界了如指掌——她对他的存在一无所知**。)*

4. **Action**：李奶奶问——**这是她在本场的唯一问句**——
   > **"……你是楼里的？"**
   
   她不是问"你是谁"——她是问"我是不是该认识你"。她**在试图分类**他。**Expected reaction**：陈默会说"我是送饺子的"或"我是您邻居"。**Actual reaction**：陈默**停了一拍**。他说：
   > **"嗯。楼里的。"**
   
   *(gap: 他**没纠正**。和早上"咸豆浆"那一刻是同一个动作——**他选择不解释自己是谁**。但这一次的代价更重——他选择不告诉一个**他认识三年的人**他是谁。)*

5. **Action**：担架被推上救护车。**Expected reaction**：关门。**Actual reaction**：急救员把担架卡好；急救医生上车；陈默**握住后门门把**——金属，热的（救护车晒了一会儿）——**他帮他们关**。**李奶奶在车里——担架的角度让她仍然能看见他**——她又看了他一眼——
   - 这一眼**没有疑惑，也没有认出**
   - 她只是**看了**——像看一个**陌生但帮过忙的人**——一种**淡淡的感激**——
   - 然后她**移开了视线**——看车顶
   
   *(gap: 这是 PC-7 的最终瞬间。**她没有认出来——并且不会再认出来了**。她对他的存在的最高承认是"陌生但帮过忙"——这正是陈默三年来在所有客户眼里的位置。**PC-7 的伤口不在她肩膀上，在他和她对视的那不到一秒里**。)*

6. **Action**：门关上。**Expected reaction**：救护车开走。**Actual reaction**：救护车没立刻开——医生还在车里填表——**陈默的手仍在门把上**。他**没松**。他攥着钥匙的另一只手——指节白。**远处玄机阁观察员在打电话**——很轻——"……刚才那阵气场，是被动的。他没主动用。重复，没主动用。" 救护车终于发动。陈默的手**自然松开**——不是他松的——是车开走了。**他还站在那里**。他看了看自己手里**李奶奶的钥匙**——他得**还给她家**——但他不知道她家门牌——他**只知道她楼层和门的颜色**。三年送了几十次餐，**他从来没记住过她的门牌号**——他靠的是直觉和门的样子。*(gap: 他和她的关系，全部建立在**他记得她的门，她不记得他的脸**。这是脊椎的最终图像。Act III 在这一拍出幕。)*

## 5. Turning Point
> 第 5 节拍：李奶奶在救护车里再看他一眼——"没有疑惑，也没有认出"——然后移开视线。
>
> **不可逆性**：(1) Act III 的"两派争夺战 + 旁观者受伤 + 力场展示"三件事全部完成；(2) **陈默被一个他熟悉三年的人当面错过认出**，且这次错过**不可补救**——李奶奶离开了，进了医院，回家之后她仍然不会认出他；(3) Act III 的内部曲线在 −3 处被血和不被认出**钉死**——他不能再悬置。
>
> Act IV 的危机不可避免：02:14 楼道，苏寒在一个手机，林晟方案在另一个手机——这两通电话此刻已经在产生（玄机阁观察员正在向苏寒回报；林晟也会在几小时内得知 4 楼的事）。

## 6. Subtext
裴鸣笔记：本场必须**慢**。这是 Act III 的呼吸。前面三场（3.6 / 3.7 / 3.8）是 PC-7 的弧线，但 3.6 和 3.7 是上升的强度，3.8 是**完全相反的姿态**——动作减到最少，强度全部转入内部凝视。这是裴鸣最擅长的写法——**爆发后那张假装没事的脸**。陈默在本场的脸必须是**没事的**——一个邻居在帮另一个邻居处理急救。但他的手——他攥钥匙的手、他握门把的手——是他全部内心戏的载体。

李奶奶的潜文本：她**真的**不记得他。这不是冷漠，不是势利——这是普通老年人对"放门口就行"那个外卖员的**正常程度的不记忆**。她的"楼里的？"是真心想分类，**她的不认出是真心的不认出**。这是本场的核心力量——它**没有人为强调**。

玄机阁观察员的潜文本：他的存在让陈默理解 Act III 没有结束——它只是从公开战变成了**记账状态**。他汇报"被动 / 没主动用"——意味着苏寒知道：陈默没接受力场，但也没拒绝它。**她的窗口仍然开着**。

陈默的潜文本：他想还钥匙——这是一个**他能完成的小动作**——但他**不知道李奶奶家的门牌号**——这意味着他连"我和她之间唯一的客观关系（送餐人/客户）"都没有完整建立过。他不是"被她遗忘"——他**从未在她的认知里成形**。这正是主控思想：**无名不是被剥夺的名字；无名是从未被建立的关系**。

## 7. Eight-Point Scene Audit
- [x] Turns（数值不动——但本场的功能是**锚定**：把 Act III 的 −3 / +2 状态钉死成不可悬置）
- [x] Earned through conflict（最深的"看见 vs 不被看见"冲突——以最低物理动作呈现）
- [x] Turning Point is locatable（第 5 节拍 / 李奶奶在救护车里的那一眼）
- [x] Scene objective is playable（让她平安上车——他做到了；被她认出来——他没做到，且不可能）
- [x] Gap is present（每节拍——第 3 节拍是全书最大 gap 之一：他知道她儿子电话 / 她不知道他叫什么）
- [x] Minimum, conservative action honored（陈默几乎没动作——他扶担架、拨电话、握门把、攥钥匙。没有一句多余台词）
- [x] Subtext is alive（钥匙 / 门把 / 没记住的门牌号——全部为底层动作）
- [x] Serves the spine（**Act III 出幕**——主控思想以字面伤口 + 字面无名 当面戏剧化；为 Act IV 危机的两通电话提供因果地基）
