# Scene 4.3 — 我是陈默。外卖员。工号 887423. / I am Chen Mo. Delivery rider. ID 887423.

## 戏剧问题 / Dramatic Question
*Will the system accept a declarative speech act from someone with no current credential, and drop the Demon Lord overlay?*
系统会不会接受一段没有凭证的宣告语言，让魔尊覆盖层脱落？

## 目标 / Objective
**宣告** —— 把一段四句话的自我陈述清晰说进对讲机麦克风。
（不是辩护，不是请求，不是质问。是宣告。）

## 冲突 / Conflict
- **外部 / 机制**：对讲机不是平台 App。他不知道这套楼宇广播是否真的会触发系统响应——他只是**记得** PC-3 拉面馆林晟复诵答句时系统抖过一下（Act II 2.2 的种植）。他在赌一个他从未验证过的规则。
- **社交**：楼下已经听得见正道联盟脚步重新逼近的方向感。他有大概 90 秒。
- **内部**：他说出工号时的声音必须是**他自己的**——不能像在车里念对讲机派单时那种职业平直腔，也不能像林晟那种把"墨渊先生"说得很认真的认真腔。他这辈子没有用"自己的"声音说过自己的名字。
- **层级**：[[extra-personal-conflict]]（与系统）+ [[inner-conflict]]（与自己声音的关系）。

## 转折点 / Turning Point
> 他说完最后一个字"尊"——"这个楼里没有魔尊"——那一秒的静默比他想象的长。然后他左手腕外侧那道淡灰色雾痕（3.3 力场被动激活后留下的）从皮肤上**脱落**，像晒干的一层胶，飘下来，落在他鞋面，消失。他没有低头。他**听见**对讲机里楼宇系统的回执音——平台系统的提示音紧跟而至——一种他在过去八天里第一次听见的"正常的工作声音"。这是不可逆的一拍：覆盖层脱落，姓名回到 App。

## 价值翻转 / Value Shift
- **命名价值**：−2（动作已发起） → **+3 外部**（系统纠错完成，姓名 = 陈默）。
- 这是本剧外部戏剧问题的答案：**yes**。陈默证明了自己不是魔尊。
- 内部曲线**尚未**翻转——本场结束时陈默尚未感受到这个胜利的空洞。空洞留给 4.4（False Ending）首次显形。
- 这是 [[story-climax]] 的机制段；意义段在 4.4。

## 种植项 / Planted Items
- **App 名字栏的第二次出现**：建议在本场结尾以一个 1–2 句的小动作呈现——他从口袋里取出**其中一个**手机（不是去看消息），点亮屏幕，看美团界面恢复："陈默 / 评级 0（恢复中） / 北区站点"。**写快**——按 misdirection-plan 的"第二次写快，强调胜利的仓促"。
- **腕上灰雾痕脱落**：3.3 中力场被动激活的可见后果，在本场被显形并被消除。这是"力场"作为物理事实的最后一次出现——它来过，它走了，它什么都没救下。
- **对讲机面板的蓝色待机灯** 在他按下通话键时变为白色——一个被广播系统接管的微小信号。

## 还款项 / Payoffs
- **Act II 2.2 系统抖动**（林晟复诵答句时）→ 此处兑现："系统对宣告性言语作出响应" 规则成立。
- **PC-1 美团账号冻结的画面**（"账号已冻结·身份冲突·关联账号「墨渊」·评级 0"）→ 此处镜像翻转为"陈默 / 评级 0（恢复中）"。
- **PC-3 林晟的 Moleskine 上"否认阶段。预期。"** → 此处被推翻：他不再是在"否认阶段"，他是在**宣告**。林晟的诊断模型本场失效——但林晟不在场看见这次失效（他在楼下，听见广播音）——这一缺席本身是讽刺的预备。

## 图像/母题 / Image & Motif
- 对讲机面板的金属下沿——他要踮一点脚才够到通话键。
- 腕上的灰雾——像电视雪花点定格成的一小片旧贴纸。
- 楼宇广播喇叭的回声延迟——他听见自己说的"陈默"两遍：一遍在耳朵里，一遍在楼道里。

## 裴鸣备注 / Persona Notes
- **决斗伪装成闲聊**：他说的话像一份送餐确认（姓名、岗位、工号、站点）——**装作平淡的报备**，但每一个字都是刀刃。裴鸣的"突然锋利"在最后一句："**这个楼里没有魔尊。**"——前面三句是绕路，第四句是收刀。
- **真相库 #2（面子和羞辱）**：他没有反驳过任何人。他只是把自己的简历念出来。这是裴鸣最爱的一种胜利——**不羞辱对方，直接绕过对方的语法**。
- **拒绝清单**：不写音乐起；不写他热泪盈眶；不写楼下的人露出错愕表情（他们不在视野里）。系统响应必须**机械、冷、准确**——这正是讽刺的开始。
- 句式：前三句陈述句（"我是 X。"），第四句换一个语法（"这个楼里没有魔尊。"）——不要解释他为什么这么说。
- **声音质感**：他这句话不是喊出来的。是用他派单时报地址的那种平直腔说的——把这种腔用在自己名字上，是本场最大的声音事件。

## 八点场景审计 / Eight-Point Scene Audit
- [x] 翻转：−2 → +3（外部），数值上是全书最大单场跃迁。
- [x] 冲突挣得：通过他**亲自说出**那四句话，而非通过任何第三方介入。
- [x] 转折点可定位：腕上雾痕脱落 + 对讲机回执音。
- [x] 场景目标可演：演员可以演"把四句话念进麦克风"。
- [x] 鸿沟在场：他期望对讲机可能不响应；结果系统**立刻**响应——这是预期与结果之间的鸿沟（"它本不该这么简单"）。这个"太简单"本身就是讽刺的预备。
- [x] 保守动作：他说话，没有动手。最少能量。
- [x] 潜文本：表面是"我在做身份纠错申报"；潜文本是"我这辈子第一次不需要任何人替我担保就说出我是谁"——但**这件事被系统接受得太轻松**，潜文本里第二层意思是"原来一切只需要这么一段广播"。这是空洞感的真正种子。
- [x] 服务脊椎：Climax 机制段。

**审计：通过。**

## Handoff
→ `subtext-whisperer`：本场的潜文本第二层（"原来这么轻松"）是 False Ending 空洞感的种子，**绝不能**在本场显形——否则 4.4 失重。需要审计：陈默在按下通话键到说完最后一字之间，**不能**有任何"反思"或"感慨"的句子。他只是在做申报。
