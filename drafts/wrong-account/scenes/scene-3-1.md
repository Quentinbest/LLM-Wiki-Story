# Scene 3.1 — 打电话

## 1. Frame
- **Where/when**: 出租屋外走廊尽头的窗台。下午 16:40。光斜斜进来，照见走廊地砖的瓷裂。陈默没在屋里打，他不愿意让那张菜单和电话同时在一个房间里。
- **Who's in it**: 陈默；苏寒（电话另一端，玄机阁某处的安静背景）。
- **Spine position**: PC-5 启动。第一个不可逆动作：拨号。

## 2. Objectives & antagonism
- **POV scene objective**：把电话打通，听她接起，然后挂掉之前——约一个地方见面。一个**他选的**地方。
- **Forces of antagonism in this scene**：
  - 内部：他不知道自己为什么要打。这是 [[inner-conflict]]。
  - 外部（极轻）：苏寒接电话太快——她在等。
- **Other characters' secret objectives**：苏寒——不让他觉得是被拉去面试。她接电话的语气是"哦，你来了"，不是"你终于来了"。

## 3. Value
- **Named value**：自决 / 被动（self-determination / being-acted-upon）。也可读作：饥渴的具体化。
- **Opening charge**: −2（饥渴存在但还在屋里）
- **Closing charge**: −2（饥渴已被拨号外推到地理上；电荷数值未变，但**矢量改变**——下一场不可避免）

> 注：本场是 [[scene-objective|意图场景]]，价值数值不动；它是 PC-5 三场弧线的**点火**。McKee Ch.10 允许序列起手场景以"激活"形式存在，只要序列整体翻转值。

## 4. Beats
1. **Action**：陈默把菜单从口袋里掏出来，背面朝上，平放在窗台。**Expected reaction**：他会立刻拨。**Actual reaction**：他先把菜单转了 180 度，让数字正对自己。*(gap: 他做了一个仪式动作，自己没注意。)*
2. **Action**：拨号。**Expected reaction**：响几声。**Actual reaction**：第二声就接。*(gap: 她在等。)*
3. **Action**：苏寒："喂。" **Expected reaction**：他报名字。**Actual reaction**：他没报。他说："停车场天台。哪个停车场？" *(gap: 他用一个反问代替了承认自己是谁。)*
4. **Action**：苏寒报地址——是他熟悉的一片。**Expected reaction**：他记下来。**Actual reaction**：他不用记。"我知道。" *(gap: 那是他送餐区。她又一次用他的地理。)*
5. **Action**：他要挂。**Expected reaction**：她道别。**Actual reaction**：她说"陈先生"——停一拍——"不用着急。" 然后她挂。*(gap: 她叫他陈先生。读者会注意。陈默自己——一时没听清。)*

## 5. Turning Point
> 第 5 节拍：苏寒在挂电话前叫他"陈先生"。这是全书第一次有人在电话里这样叫他——但他没听清，或装作没听清。**他没让自己听清**。这是不可逆的：他选择不把那声"陈先生"接住，因为接住就要承认自己来这里不是为了被叫陈先生。

## 6. Subtext
他在屋里来回了一下午。他打这通电话不是为了"赴约"——他是为了**让自己不能反悔**。一个人独自的犹豫太轻，需要外推到一个具体地点、一个具体时间。表层动作是约定；底层动作是**把自己钉在一个未来的地理上**，这样他就不必再做选择。裴鸣会让这一段全部在身体里——转菜单、拨号、不用记地址。不写他在想什么。

## 7. Eight-Point Scene Audit
- [x] Turns（矢量改变，charge 不动但情境不可逆）
- [x] Earned through conflict（内部冲突；苏寒一句"陈先生"是外部小冲突，他选择忽略）
- [x] Turning Point is locatable（"陈先生"那一拍）
- [x] Scene objective is playable（拨号、约地点、挂电话）
- [x] Gap is present（每个节拍都有）
- [x] Minimum, conservative action honored（最低限度：他只打了一通电话，没解释什么）
- [x] Subtext is alive（电话内容是约定，潜文本是钉自己）
- [x] Serves the spine（PC-5 的不可逆第一步）

## 8. Handoff
→ `scene-3-2`（爬楼）。本场结束即触发下一场的物理位移——他必须去。
