# Scene 3.2 — 爬到天台

## 1. Frame
- **Where/when**: 一座七层老式停车场。下午 17:20。电梯故障告示已经贴了至少一周（边角卷了）。陈默从一楼电梯口转身，走进安全楼梯间。日光在楼梯转角越来越多，越往上越亮。
- **Who's in it**: 陈默；一位坐在岗亭里的中年保安，只露半张脸；楼梯口的几个空可乐瓶——是上面有人下来过的痕迹。
- **Spine position**: PC-5 序列的物理跨越段。承担"他第一次站在自己城市的某个高处"这一脊椎承诺。

## 2. Objectives & antagonism
- **POV scene objective**：按时到达天台，**不让自己在路上想清楚**。
- **Forces of antagonism in this scene**：
  - 物理：电梯坏了——他没法躲在电梯里发呆；他必须用身体一层一层上升。
  - 内部：每一层都是一个可以停下来不上去的机会。
- **Other characters' secret objectives**：保安——只想知道这个穿外卖服的小伙子是不是上来偷东西。

## 3. Value
- **Named value**：高度 / 平面（height / ground）。地理变成心理。
- **Opening charge**: −2
- **Closing charge**: −2（数值未动，但**他到了。城市第一次在脚下。**——为下一场提供基线）

> 注：这场是过桥场景。McKee Ch.10 允许过桥，**前提是它本身有微转折**。本场的微转折是第六层的停顿——他第一次有"我可以下去"的清醒念头，并选择继续上。

## 4. Beats
1. **Action**：陈默推开楼梯间防火门。**Expected reaction**：里面是楼梯。**Actual reaction**：里面是楼梯**和一股潮湿水泥味**——他熟。送过太多老楼，老楼味都一样。*(gap: 他到了一个新地点，鼻子却觉得是旧地点。)*
2. **Action**：上到三层平台。**Expected reaction**：继续上。**Actual reaction**：停下来，掏出手机。**手机没消息**。他平时这个时间手机一定有派单铃。*(gap: 他第一次在工作时间不工作。)*
3. **Action**：四层。墙上有张"对讲机维修——拨 5188"的红字小广告，纸已发黄。**Expected reaction**：他不看。**Actual reaction**：他看了一眼，记了一下号码——**职业习惯**（他每天看楼内贴的小广告，因为送餐找门牌靠这些）。*(gap: 一个未来高潮的零件，被他用送餐人的眼睛收了起来。读者此刻不会注意。)*
4. **Action**：六层。他停下来。**Expected reaction**：继续上。**Actual reaction**：他靠在楼梯扶手上，**看下去**。他从来不知道自己住的这一片从六楼看是这样的。他听见了自己的呼吸。*(gap: 他来这里是为了见一个人；他不该看下面。他看了。)*
5. **Action**：他继续上。**Expected reaction**：屋顶门会锁。**Actual reaction**：屋顶门是开的。一道光，**斜的**。*(gap: 他到了。他没有给自己留犹豫的时间。)*

## 5. Turning Point
> 第 4 节拍：他在六层楼梯间记下"对讲机维修——拨 5188"。**他不知道这一笔重要**。读者也不知道。这是种植项——Act IV 序列 4.1 的对讲机机制由此触发可信度。本场作为过桥的微转折是第 4 节拍（看下去 = 第一次城市在脚下），但更深的不可逆动作是第 4 节拍：他在自己的职业反射里收下了未来救自己的工具。

## 6. Subtext
裴鸣写法：这一整场全用动作写。每个节拍只许动词不许形容词（"潮湿水泥味"是例外——气味是动词的近亲）。陈默以为他在爬楼；他实际上在**逐层放弃送外卖那个版本的自己**。每一层都是一次"我还可以下去"的拒绝。但他不知道——他以为自己只是在迟到边缘。

## 7. Eight-Point Scene Audit
- [x] Turns（数值不动，但情境不可逆——他到了；并埋下了 5188）
- [x] Earned through conflict（内部冲突 + 物理阻碍）
- [x] Turning Point is locatable（第 4 节拍 / 5188 广告 / 看下去）
- [x] Scene objective is playable（爬上去）
- [x] Gap is present（每节拍都有）
- [x] Minimum, conservative action honored（他甚至没有焦虑——他用了最少的内心戏）
- [x] Subtext is alive（爬楼 = 弃旧身份的位移）
- [x] Serves the spine（PC-5 的物理段；并落地"对讲机"种植）
