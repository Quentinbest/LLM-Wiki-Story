# Scene 4.4 — ……陈默先生？对不起。 / ……Mr. Chen Mo? I'm sorry.

## 戏剧问题 / Dramatic Question
*Will the name, finally spoken by the only listener who ever listened, be enough?*
那个一直认真听他的人终于叫出了他的名字——这够吗？

## 目标 / Objective
**回应** —— 接住林晟那句话，或者不接。
（陈默自己的场景目标其实是**被动等待**——他刚说完自己的名字，他没有下一个动词。林晟的目标是"完成礼貌的撤退"。两个目标都不是冲突性的——这正是空洞的引擎。）

## 冲突 / Conflict
- **外部 / 社交**：林晟是唯一从未对陈默施加暴力的"敌人"。他认真、温和、刚刚学到对方的名字。他现在的任务（在他自己的协议里）是"承认错误并撤离"。陈默面对的"对抗力量"不是恶意——是**礼貌**。礼貌是最难拒绝的撤离。
- **内部**：他刚刚赢了。系统纠错了，敌人在退。**他应该感到什么**——他没有感到。这个"没有感到"是本场的内核。
- **层级**：[[personal-conflict]]（与林晟）+ [[inner-conflict]] 第一次显形为**缺席感**（不是痛苦，是没有东西到来）。

## 转折点 / Turning Point
> 林晟从 7 楼平台抬头看他——隔着一层半楼梯——犹豫了大约一秒（不是太久，正好是认人需要的时间），说："……陈默先生？" 这是全书第一次有人主动说出他的名字。短暂停顿。林晟低头看自己的 Moleskine（**他在确认自己刚才说对了**），抬头，再说："对不起。" 然后转身上楼。脚步既不快也不慢。陈默站在那里，听着脚步声远去，**认出**那是他自己上下楼送餐的节奏——不疾不徐、节省体力、礼貌但完成任务的节奏。这一**听见自己**的瞬间是本场不可逆拍：他赢回了名字，并在同一秒理解了名字不解决任何事。

## 价值翻转 / Value Shift
- **外部命名价值**：+3 保持（林晟说出了名字，是外部胜利的语言级兑现）。
- **内部命名价值**：从未在 Act IV 前显形过的"内部空洞"在此**第一次有了具体形状**——林晟的脚步声 = 他自己的脚步声 = 那个隐形的、节省体力的、礼貌但被遗忘的人的步态。从未显形（中性？） → **−3 显形**。
- 这是 [[false-ending]] 的峰：外部曲线触顶 / 内部曲线触底，**同时**发生。McKee Ch.13 [[meaning-produces-emotion]]——意义在此被读者**感到**，但不被任何角色说出。

## 种植项 / Planted Items
- **林晟的脚步节奏**：在本场之前需要被建立——建议在 PC-3（拉面馆离开时）或 PC-6（早餐后离开时）有一笔陈默注意到林晟走路"像送过外卖的人"的暗示。如果未铺设，本场的"认出自己脚步"会失重。**给 `setup-scout` 一个回查点。**
- **林晟低头看 Moleskine 确认名字**：PC-3、PC-6 已两次出示这本本子。本场是**它第一次写下"陈默"**——但这件事观众看不见（本子是合上的）。这是 misdirection-plan 第二触发点之一。

## 还款项 / Payoffs
- **激励事件**："你好，请问你是魔尊·墨渊吗？" → 本场："……陈默先生？对不起。" 句式对位完成。第一句是疑问 + 错名 + 礼貌；末句是疑问 + 真名 + 道歉。两句话之间是整部小说。
- **林晟从未问名字（PC-3 / PC-6 二次铺设）** → 本场兑现：他没问，他**查到了**——而**查到**比问到更说明他不需要陈默亲自告诉他。这正是讽刺：连真名都不是被询问得到的。
- **Moleskine 本子（PC-3 物理植入 / PC-6 二次出示）** → 本场低头确认动作完成本子的"姓名页"被启用。
- **苏寒"你做的事，世界终于看见了。只是它叫你魔尊"（PC-5）** → 在本场以反讽形式被验证：世界改口了，但**离开**了——苏寒说的是看见与命名的悖论，本场把"被正确命名"也加入悖论。

## 图像/母题 / Image & Motif
- 楼梯感应灯第 N 次熄灭，林晟的脸**消失在感应灯熄掉的那一秒**——他说完"对不起"时，他的脸已经在暗里了。陈默听到声音不看到表情。这一处理避免林晟被"看见的脸"独占道德焦点。
- 林晟手里 Moleskine 的边角——红色橡皮筋，PC-3 已建立。
- 林晟的脚步声 → 陈默自己的脚步声——这是裴鸣的"一首歌重新定义一场戏的道德距离"的本剧版本：**声音的回响**而非真正的音乐。

## 裴鸣备注 / Persona Notes
- **决斗伪装成闲聊**：本场最危险的失败模式是把林晟写成"反派觉悟"。裴鸣的原则——**没有反派觉悟，只有任务完成**。林晟不是被陈默感动；他是在按协议完成"误识纠错"的最后一步。他的道歉**真诚**，但**不是关于陈默这个人**——而是关于"我们抓错了"。这才是讽刺。
- **真相库 #3（私人准则高于公共道德）**：林晟在遵守他自己的内心准则——他错了就道歉。**他遵守了**——所以他有尊严。裴鸣不评判他。
- **拒绝清单**：不写林晟"眼神变了"。不写林晟"留下了什么"。不写陈默"心里一空"。**所有这些都必须由脚步声和暗中的脸完成**。
- 林晟离开的节奏：**不能太快**（残忍）；**不能太慢**（戏剧化）。他完成了任务在离开——misdirection-plan 已明示。建议字数：本场总长 1,800–2,500 字，林晟出现到离开占其中 60%；林晟离开后到本场结束（楼道静下来）占 40%。**后半的静必须有重量**——按 misdirection-plan："不要急着进入结局，让它呼吸"。
- 林晟离开后陈默有大约 3–4 拍：他没有动；他听见楼宇广播说"楼内警报解除，请居民正常作息"；他听见远处一辆车（玄机阁）启动；他看见对讲机面板的蓝灯——已经回到待机状态。**这 3–4 拍不解释他在感受什么。**他可能掏出手机看了美团 App（App 名字栏第 2 次出现的正式时刻——按 misdirection-plan "第二次写快"），看见"陈默 / 评级 0（恢复中）"，然后把手机放回口袋。

## 八点场景审计 / Eight-Point Scene Audit
- [x] 翻转：外部 +3 保持 / 内部从中性首次落到 −3。**双线的剪刀差**就在本场张开。
- [x] 冲突挣得：林晟的道歉是 PC-7 后的逻辑必然——不是恩赐，不是顿悟。冲突的胜利由 4.3 的宣告挣得；本场是**胜利落地后的余震**——余震本身是冲突的：他面对"赢了什么"的问题。
- [x] 转折点可定位：林晟脚步声 = 陈默自己的派单脚步声——这个"听出来"的拍。
- [x] 场景目标可演：演员可以演"接住一句道歉，没有回话"。
- [x] 鸿沟在场：他期望被叫出名字时会感到什么——结果他**没有感到**那种被叫出来该有的东西。McKee 的鸿沟在此最锋利。
- [x] 保守动作：他没有走下去握林晟的手；他没有喊"等等"。他只是站着。保守到极致。
- [x] 潜文本：林晟的表面是"我道歉并撤退"；潜文本是"我完成了我的任务，包括知道你叫陈默——但**知道**不等于**认识**"。陈默的表面是"我接受这个道歉"；潜文本是"原来我赢回来的东西就是这样一个声音、一个脚步、一个上楼"。
- [x] 服务脊椎：False Ending 峰 + 控制思想首次显形（无台词地）。

**审计：通过。本场是全剧最难写的一场。**

## Handoff
→ `subtext-whisperer`：林晟的两句对白（"……陈默先生？" + "对不起。"）必须在不加任何修饰副词的状态下携带全部讽刺——本场的所有重量都压在这两句的**间距**与林晟离开的**步频**上。
→ `prose-stylist`：林晟离开后的 3–4 拍是 misdirection-plan 明示的"让它呼吸"——压制任何抒情冲动。建议这 3–4 拍只用动名词：广播声、引擎声、蓝灯、手机屏幕、口袋。
