# Scene 3.7 — 楼内

## 1. Frame
- **Where/when**: 陈默的公寓楼内。13:42。一楼电梯厅 → 安全楼梯 → 4 楼平台。光线递减——电梯厅亮，楼梯间惨白节能灯，4 楼平台只有自然光从走廊尽头窗户透进来。空气中渐渐有金属味（剑）、水泥粉末（破墙）、最后是**血**（4 楼）。
- **Who's in it**: 
  - 陈默（中间）
  - 赵铭 + 正道联盟 6 名修真者（赵铭本人不主动出剑，由骨干执行——他在观察）
  - 玄机阁 3 人（小队长**周牧**——苏寒派的副手，30 出头，少话）
  - 李奶奶（4 楼住户，刚出单元门去市场，回来时**正撞上 4 楼平台的交火**）
  - 楼内 2–3 户居民（**全部闭门——他们听见动静，但没人开门**。这是 PC-7 旁观者主题的另一层）
- **Spine position**: **PC-7 兑现**——脊椎最高物理强度场，魔尊力场首次主动展示。

## 2. Objectives & antagonism
- **POV scene objective**：**让楼里没有人因为他受伤**。
- **Forces of antagonism in this scene**：
  - 物理：两派交火——12 个修真者在一个普通居民楼内动手
  - 地形：陈默的楼是住宅，每一道剑光都可能切到一扇门
  - 内部：他**没有打架的能力**——他是外卖员；魔尊力场不在他控制下，是**被动的**
  - 时间：李奶奶从市场回来——她正在上楼
- **Other characters' secret objectives**：
  - 赵铭——结束。他不让骨干太用力（楼里有居民），但**不撤**。
  - 周牧（玄机阁）——保住"墨渊"。他奉命来"护主"——但他**不知道魔尊本人也不愿意被这样保护**。
  - 李奶奶——**回家**。她没看到楼下，她只听到楼上有动静以为是装修。

## 3. Value
- **Named value**：保护 / 无能（protection / impotence）。也可读作：力场 / 普通人。
- **Opening charge**: 内部 −3 / 外部 +2
- **Closing charge**: 内部 −3（数值不动，**但被血锚定**——他不能再悬置在"两边之间")/ 外部 +2（变质——他不再是被争夺的对象，他是**在场失败的人**）

> 注：本场的翻转是**质性**+**情境**——价值数值不动，但场景结束时陈默的"内部 −3"被**血**钉死，不可悬置。

## 4. Beats（8 节拍——本场是 PC-7 兑现，必须充分）

1. **Action**：玄机阁三人冲进单元门。**Expected reaction**：赵铭迎击。**Actual reaction**：赵铭让出位置——**他让骨干上**。他自己往后退到电梯门边，**观察**。两边的骨干在电梯厅碰头。第一道剑光擦过——**墙上瓷砖崩了一块**。*(gap: 这不是仙侠剧的飘逸——这是物理。墙会碎。)*

2. **Action**：陈默被周牧推到楼梯口——周牧用手按住陈默的肩膀，对赵铭骨干用左手出剑。**Expected reaction**：陈默被保护。**Actual reaction**：周牧手按得**太重**——陈默感到肩胛骨疼。**他被作为物件保护**。*(gap: 玄机阁不把他当人——把他当资产。和赵铭抓他没有性质差别，只是方向相反。)*

3. **Action**：陈默挣开周牧——往楼梯里跑。**Expected reaction**：他要躲。**Actual reaction**：他往**楼上**跑——他想到了李奶奶。他记得她每天市场回来的时间——下午两点前。**他要拦住她**。*(gap: 全书第一个"陈默主动救人"的动作——而救的是一个不认得他的人。读者第一遍读：英雄行为。回读：**这是陈默的私人准则——他熟悉这栋楼，他知道谁该几点回家**。)*

4. **Action**：他冲上二楼平台，三楼平台。**Expected reaction**：李奶奶在 4 楼。**Actual reaction**：他听见**4 楼有脚步声**——上楼的脚步——**慢，老人的步速**——还有钥匙在塑料袋里晃的声音。**李奶奶已经在 4 楼了**。*(gap: 他迟了一拍。)*

5. **Action**：陈默冲上 4 楼平台。**同时**——两边的骨干追上来，正道一人 + 玄机一人，从 3 楼追上 4 楼。**Expected reaction**：他能在剑落下之前拽住李奶奶。**Actual reaction**：**他来不及**。一道剑光（正道骨干劈向周牧——周牧已经追上来），周牧侧身避开，**剑光走偏，切向 4 楼走廊**——李奶奶站在自家门口正在掏钥匙。
   
   **这里需要慢镜头节奏**：
   - 陈默的喉咙里有一声"奶奶"——没出口
   - 李奶奶抬头——她**没看陈默**，她看的是剑光来的方向
   - 钥匙从塑料袋里掉出来
   - 剑光擦过她的左肩——**不深，但她在 70 岁的身体里**
   - 她坐了下去——不是倒，是坐——背靠在自己家门
   
   *(gap: 没有戏剧化的"挡剑"，没有英雄主义的"以身相替"。**他迟了**。这就是真相。)*

6. **【魔尊力场被动激活】**——这是契约 §9 第 1 条的核心兑现。
   
   陈默扑过去——三步——**距离李奶奶还有两米**。这时他身上**有什么开始溢出**：
   - 走廊的灯**全部熄灭一拍**——然后亮起，**颜色偏冷**
   - 他周围一米的空气**变重**——像下沉的水
   - 走廊瓷砖上有一道**黑色覆盖层**短暂闪过——魔尊视觉皮肤的痕迹
   - 正道骨干和玄机骨干**同时停手**——他们认出来了——"这就是力场"
   - **赵铭从一楼听到反应**——他短促地说："让开"
   
   **但是**——
   - 力场**不在陈默控制之下**——他不会用
   - 力场**不能治伤**——它只是一个**气压**
   - 力场**不能让李奶奶起来**——她还是坐着，肩上在出血
   - 力场**持续了不到十秒**——然后**消退**，像潮水退去
   
   *(gap: 类型读者期待"力场=主角觉醒/反杀"。**裴鸣的处理：力场出现了，被见证了，然后没用**。这是契约 §9 第 1 条的反类型兑现——力场存在性被证实，**力场救人能力被否定**。这是 Act IV 高潮"放弃力场"获得情感重量的唯一方式。)*

7. **Action**：力场消退。陈默扑到李奶奶身边——蹲下。**Expected reaction**：李奶奶会抓他的手。**Actual reaction**：李奶奶**抬头看他——她皱眉**——她不认识他。她的左肩在出血。她说——**带着痛**——
   > **"……谁啊？"**
   
   *(gap: 这是全书最锋利的一句台词。她每月让他放门口几十次。她甚至记得他敲门的节奏（应该是）——但**她不记得他的脸**。**力场也不能让她记得**。)*

8. **Action**：两派同时撤——正道骨干听到赵铭"让开"的命令；玄机骨干跟着撤。**Expected reaction**：解决谁的归属问题。**Actual reaction**：没人留下处理李奶奶。**陈默一个人**蹲在她身边。**赵铭从楼梯口看了一眼——他对周牧说："今天到这。"** 然后两边一起退下楼。
   
   走廊里——**陈默、李奶奶、血、关着的居民门、对讲机的红色 LED 在墙上闪**。
   
   *(gap: 派系都走了。**留下的是陈默和那个不认得他的老太太**。这就是 PC-7 的真正姿态——不是"两派打他"，而是"两派把代价留给一个普通人"。)*

## 5. Turning Point
> 第 7 节拍：李奶奶看他，说"……谁啊？"
>
> **不可逆性**：(1) 已经流血；(2) 他用过了力场——他知道力场是什么、做不到什么；(3) 老太太当面在伤痛中**问他是谁**——她不是没看他，她**看了**，她**问了**，她**不认得**。
>
> 这一拍把脊椎的 PC-7 钉死：陈默不能再假装这是关于"名字"的争执。**李奶奶的伤口和不认得，是同一件事的同一个证据**——主控思想"无名才是真正的伤口"在 PC-7 显形为字面意义的伤口 + 字面意义的无名。

## 6. Subtext
裴鸣笔记：
- 力场的描写必须**抑制美感**。这不是觉醒戏。颜色冷、空气重、灯闪、瓷砖闪——这些是**症状**，不是**胜利**。任何"觉醒"语气都会破坏 PC-7。
- "……谁啊？"必须是李奶奶的**唯一台词**。不能再有第二句。她不能解释——她只问。
- 两派撤退的节奏要快——读者会以为"派系还要打"，但派系**走了**。他们走得有一定职业冷静——他们知道这事大了（流血），他们要回去开会，他们不在走廊上谈判。**这种职业感本身**就是 PC-7 的姿态：陈默对他们不是个人，是事件。

陈默的潜文本：他第一次知道魔尊力场是什么。他也第一次知道——**他不要那个东西**。不是因为它是"邪恶的"，而是因为它**不够**。他试过了。他没救成。

## 7. Eight-Point Scene Audit
- [x] Turns（情境不可逆——血 / 力场用过 / 老太太当面不认 / 派系撤但都没解决）
- [x] Earned through conflict（最高强度物理 + 内部冲突共振）
- [x] Turning Point is locatable（第 7 节拍 / "……谁啊？"）
- [x] Scene objective is playable（救李奶奶——他失败了）
- [x] Gap is present（每节拍 / 力场出现 vs 力场无效 是最大 gap）
- [x] Minimum, conservative action honored（陈默没"反杀"。他只是跑、扑、蹲）
- [x] Subtext is alive（力场是症状不是胜利；老太太的一句是字面伤口=主控思想）
- [x] Serves the spine（PC-7 兑现 / 力场可见性兑现 / 派系对决类型奖赏 / 老太太不认得=结局 603 的最大预告）
