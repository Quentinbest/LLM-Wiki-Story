# Scene 4.1 — 两个手机放在地上 / Two phones on the floor

## 戏剧问题 / Dramatic Question
*Will Chen Mo, in the smoke-filled stairwell between floors, hold both options open long enough to see a third?*
他能不能在两个手机之间，撑到看见第三条路？

## 目标 / Objective
**接听** —— 同时接住两个电话，不让任何一个先挂断；他在动作上还不知道自己已经在拒绝。
（裴鸣的动词测试：他没有在思考；他在用拇指防止两个屏幕熄灭。）

## 冲突 / Conflict
- **外部 / 物理**：楼下走廊飘上来的塑料烧焦味、消防警报短促的预警（未拉响）、楼梯灯每 12 秒灭一次的感应灯节奏。
- **外部 / 社交**：左手手机里苏寒的呼吸；右手手机屏幕上林晟方的"人道主义投降协议 PDF"——一个待签字的灰色按钮。两条人声，都在等他。
- **内部**：他第一次同时握着两份"被看见"。两边都没有错。两边都不是他。
- **层级**：[[inner-conflict]] 终端，叠加 [[extra-personal-conflict]] 残余（系统警报、烟味）。

## 转折点 / Turning Point
> 他蹲下来，把两个手机面朝下扣在水泥地上——不是挂断，是**让屏幕的光不再照他的脸**。他抬头时，对讲机面板那盏蓝色待机灯正好亮在他左眼上方。这是本场不可逆的一拍：从"接住两个名字"翻为"放下两个名字"。

## 价值翻转 / Value Shift
- **命名价值**：被错名而被看见（−3，矛盾深） → 不被任何一个名字捕获（仍 −3，但**性质变了**：从"被两个名字拉扯"翻为"暂时无名"——本场不完成正极翻转，把翻转留给 4.2）。
- 这是一场**预备性场景**：电荷数字相同，但**矛盾结构**翻转。审计豁免理由记于第 7 节。

## 种植项 / Planted Items
- **楼内对讲机面板**：1.2 和 3.3 已建立物理存在。这里第一次被陈默"看见"——蓝色待机灯，墙体高度约 1.6 米，金属壳上有一道被钥匙划过的旧痕。
- **两个手机的位置**：左口袋（苏寒）/ 右口袋（林晟链接）。这个左右分布在 3.2 早餐场景被无意识建立（他用右手接林晟递过来的豆浆）。
- **烟味**：3.3 派系大战的物理残留——楼里仍未通风。
- **他蹲下**：与 1.1 序列开场"在车上吃别人的冷饭"那次蹲姿同形——回响[[image-systems]]。

## 还款项 / Payoffs
- **本场尚未兑现任何种植项**——这是 Act IV 全部兑现的"上膛动作"。

## 图像/母题 / Image & Motif
- 两块发光的长方形扣在水泥地上，光透不出来，但屏幕震动的嗡嗡声还在脚下传导。
- 对讲机面板的蓝灯——一个**不需要被对准的脸**就能说话的接口。
- 感应灯节奏：12 秒一次的暗与亮，给六分钟一个可数的肋骨。

## 裴鸣备注 / Persona Notes
- **沉默不是内心戏**。六分钟里裴鸣不写一句"他想"。他写：陈默的鞋底踩到一小块碎玻璃，没有移动；他的左小腿开始发麻，他换了一下重心；他注意到墙皮一道发霉的水痕从 9 楼方向往下渗。
- **真相库 #1**：不让陈默对自己说"我要选哪个"。让他注意小事。真相从不在思考里漏出，**在他不思考的时候漏出**。
- **决斗伪装成闲聊**：这场没有第二个在场的人，所以"闲聊"由身体承担——他的呼吸、烟味、感应灯——它们是这场决斗的对白。
- **拒绝清单**：不写"他想起了过去 8 天"。不写"他理解了什么是身份"。不写任何让读者觉得他正在抒情的句子。
- 句式：短句切入（"烟味。"），长句绕路（描述墙皮、玻璃、感应灯），然后短句收刀（"他蹲下来。"）。

## 八点场景审计 / Eight-Point Scene Audit
- [x] 翻转：价值数字未变，但矛盾结构从"被双重捕获"翻为"放下双重捕获"——**质性翻转**而非数值翻转。
- [x] 冲突挣得：他没有被告知第三条路；他通过把两个手机放下，物理上腾出了视线，**才**看见对讲机。
- [x] 转折点可定位：扣下手机 → 抬头 → 蓝灯入眼。
- [x] 场景目标可演：演员可以演"同时握住两个电话不让它们熄屏"。
- [x] 鸿沟在场：他期望选边；结果——既未选边、也未保持悬置——发现第三介质。
- [x] 保守动作：他没有大动作，没有喊叫；他只是蹲下。**最低能量动作**。
- [x] 潜文本：表面动作是"接电话"；潜文本是"我无法让任何一个屏幕熄灭，因为屏幕熄了就没人在等我了"。
- [x] 服务脊椎：本场是 Crisis 入口；不可省略。

**审计：通过。** 一点说明：opening/closing 电荷相同但**矛盾结构翻转**——这是 McKee Ch.11 [[scene-design]] 允许的"预备性 turning"（场景内的小转，承担更大序列转折的上膛）。如审计严格要求数值差，需把本场与 4.2 合并；但合并会破坏"六分钟"的物理时长——故保留为两场。
