# Scene 4.6 — 放门口就行 / Just leave it at the door

## 戏剧问题 / Dramatic Question
*Can being correctly named fill the wound of namelessness?*
被正确命名，能不能填上无名的伤口？
（这是全书的真实戏剧问题——不是脊椎宣告的"他能否洗清冤屈"，是 misdirection-plan 揭示的隐藏问题。）

## 目标 / Objective
**送达** —— 把麻辣烫和两杯柠檬茶交到 603 室。
（这是他在 1.1 序列开场时的同一种目标。本剧的 Resolution 的图像本质：**他赢回的，是再一次执行 1.1 序列动作的资格。**）

## 冲突 / Conflict
- **外部**：电梯故障——他要爬 6 层。这是 1.2 激励事件的物理回响（那次也是 6 层走读）。物理冲突极轻——他每天都在爬这种楼。
- **社交**：门后的女人是一个**他没有任何线索的陌生人**。她可能是新住户，可能是隔壁租客，可能她和老 602 一家从来不来往（事实如此）。她对陈默没有任何前史。
- **内部**：**他没有期待**。这是本场最锋利的处理——他不是去希望被认出来。他**已经知道**会发生什么，**他还是去送**。这种"已知的执行"是讽刺最终的形状。
- **层级**：[[inner-conflict]] 终结——但它的形式是**没有冲突感的执行**。

## 转折点 / Turning Point
> 他敲门三次（与 1.2 激励事件同一节奏——这是必须的对位）。门开。一个 40 多岁的女人，穿着家居服，手机在另一只手上亮着抖音的蓝光。她**不看他的脸**，眼睛只在外卖袋上停了半秒，伸手——
> "放门口就行。"
> 然后门关上了。
>
> 陈默站在那里。一口气。他的嘴唇**微微张开**——他差点对着关上的那扇门说"我是陈默"——**他没有说**。他低头看一眼手机：陈默。他转身。他下楼。
>
> 这个"嘴唇张开又没说"是本场不可逆的一拍——也是全书最后一拍。主控思想在此被图像化兑现：**被正确命名不能填上无名的伤口；他知道这一点；他不再尝试**。

## 价值翻转 / Value Shift
- **数值**：+3 外部 / −3 内部 — **不动**。
- **意义**：从"剪刀差的稳定常态"翻转为"主控思想的图像兑现"。读者在本场看到 misdirection-plan 的真实读法浮现——苏寒 PC-5 那句话回响（"你做的事，世界终于看见了。只是它叫你魔尊。"现在改名了，世界依然不看见）。
- 这是 [[meaning-produces-emotion]] 的最终落地——情感由意义触发，而非由数值变化触发。
- McKee Ch.13 允许 Resolution 的最后一拍**不**做数值翻转，只要意义翻转可定位——本场满足。

## 种植项 / Planted Items
- **App 名字栏第 3 次出现的尾声呈现**：建议在他**下楼时**再看一次手机（接下一单）——名字栏第 3 次出现的"最长停留"在 4.5 已完成；本场是它的呼应一瞥（短，约 20 字）。
- **柠檬茶 + 麻辣烫** = 激励事件"两杯奶茶 + 炸鸡"的镜像——同一栋楼，隔壁一户。
- **敲门三次**：与激励事件同节奏。建议字面对位（1.2 写"他敲了三下门"，本场写"他敲了三下门"——**完全同一句**）。
- **6 楼走读**：电梯故障的物理回响——但这次没有人在 8–9 楼之间的楼道里等他。
- **三个"陈默"**：(a) 他刚才在 App 接单时看到的名字；(b) 他差点说出口的；(c) 他下楼后再次看 App 看到的——三次在本场内出现，**没有一次被他人听见**。

## 还款项 / Payoffs
- **激励事件 "你好，请问你是魔尊·墨渊吗？"** → 本场 "放门口就行" —— 句式完全对位：第一句是疑问 + 看着他的脸 + 错名；最后一句是命令 + 不看他的脸 + 无名。两句话之间是整部小说。
- **苏寒 PC-5 诊断台词**："你做的事，世界终于看见了。只是它叫你魔尊。" → 本场最强回响：世界改口了，世界也不看了。苏寒说对了——只是她当时以为答案是"接受错名"，本场证明**没有答案**。
- **林晟从未问名字 / Climax 第一次叫名字 / 然后离开** → 本场延续：林晟离开后，世界回到这种状态——没有人会再叫他的名字，包括这个新客户。
- **PC-1 美团账号冻结画面 / 4.3 恢复 / 4.5 第 3 次出现 / 本场尾呼应** → 名字栏的四次出现完成完整闭环：冤屈 → 胜利 → 常态 → 常态延续。**名字一直在屏幕上，没有人来过门口。**

## 图像/母题 / Image & Motif
- 女人手机里的**抖音蓝光**——她的注意力已经在别处。这是 2026 年的"无人在等"的最准确视觉化。
- **门关上的声音**——必须写。一种廉价防盗门的金属响，回声在 6 楼楼道里短促一弹。这个声音是 Climax 对讲机蓝灯回执音的镜像（机械、冷、准确）——同一种声音质感，给出两种相反的判决。
- **他下楼**——电梯依然故障；他走下 6 层。1.2 时他向上爬向激励事件；本场他向下走向常态。**上 / 下**完成 [[image-systems]] 的对位。
- 他在下到 1 楼时听见手机震动——下一单——他没有马上看。**他先把外卖包侧带紧了一下**，然后看手机。这个"先调整外卖包"的动作是他的私人仪式——裴鸣的"私人准则高于公共道德"在最后一拍的具体物化。

## 裴鸣备注 / Persona Notes
- **基调极其关键**：**不是悲伤，不是凯旋，是基线**。misdirection-plan 明示："恐怖在基线里——他赢回的就是这个"。整场的语调必须像他过去 8000 单送过的任何一单。**他不痛苦，也不麻木。他在工作。**
- **决斗伪装成闲聊**：本场没有对白冲突——女人那句"放门口就行"是一句**完全无害的日常话**。整场的暴力在于：**这句话是结局的答辞**。裴鸣"前面绕路然后一刀"的极致版——绕路是一整部小说，刀是一句"放门口就行"。
- **真相库 #1（语言是武器也是泄露）**：女人**没有泄露任何东西**——她是"无名"的具象。她不在场，**她在抖音里**。这才是当代讽刺的精度——伤害他的不是恶人，是注意力的位置。
- **真相库 #3（私人准则高于公共道德）**：陈默差点说出名字，没说。这个"没说"是他的最大尊严。他没有向一个不在场的人索取在场。**遵守了准则**——裴鸣的最高评价。
- **拒绝清单**：
  - 不写"他笑了一下"或"他苦笑"——任何带情绪标签的反应都会毁掉基线。
  - 不写"他想起了林晟"或"他想起了苏寒"——任何回溯都把图像变成阐释。
  - 不写"他明白了什么"——主控思想必须被读者明白，不能被陈默明白。
  - 不写"他擦了擦眼"——任何泪痕都让本场变成情节剧。
- **句式**：本场是全书最短句式段——3–7 字短句堆叠为主。最长不超过两行半。**敲门三次 / 门开 / 她不看他 / "放门口就行" / 门关 / 一口气 / 嘴唇张开 / 没说 / 下楼 / 手机震动 / 下一单。**

## 八点场景审计 / Eight-Point Scene Audit
- [x] 翻转：数值持平；意义翻转可定位（"嘴唇张开又没说"那一拍——"被正确命名不能填伤口"在此被陈默自己**懂了**，读者跟着懂）。McKee Ch.13 [[meaning-produces-emotion]] 允许 Resolution 的翻转为意义级。
- [x] 冲突挣得：本场的"冲突"是他与"差点说出名字"这个冲动的对抗——他赢了——通过**不说**。
- [x] 转折点可定位：嘴唇张开又闭上的那一拍。
- [x] 场景目标可演：演员可以演"送达 + 不说一句话 + 下楼接下一单"。
- [x] 鸿沟在场：他没有期待——但他**仍然有一拍想说出自己的名字**。这一拍就是鸿沟（他知道没用，他还是想说；他没说）——McKee Ch.7 [[the-gap]] 在本场的最隐微形式：**期望与已知之间的鸿沟**。
- [x] 保守动作：极致——他全场只做"送 / 不说 / 走"。
- [x] 潜文本：表面是"一次普通的送餐";潜文本是"这就是我赢回的全部"。
- [x] 服务脊椎：主控思想图像兑现。**结局必须为完整场景而非尾声**——本场长度建议 2,500–3,500 字（虽然事件简单，须给楼道、敲门、门关、下楼以**完整呼吸时间**——脊椎 §9 第 5 条明示）。

**审计：通过。本场是全剧总落点。**
